# 微表情库

基于 Paul Ekman 面部动作编码系统（FACS），43 种微表情 + 眼神/嘴部/眉部动作路径。用于在 prompt 中精确描述角色面部情绪，增强 AI 生成的情感准确度。

---

## 三、在 Prompt 中使用

```
角色微表情：[主角面部表情描述]。眼神：[眼神路径/状态]。嘴部：[嘴部微动作]。眉部：[眉部状态]。
```

例：
```
角色微表情：主角面部 EX15. 心动（微眯眼+微扬嘴角+瞳孔微扩），随后转为 EX23. 隐忍（眉头压低+咬牙+嘴唇紧抿）。
眼神：EP1. 闪躲→锁定→移开→回望（暗恋路径）。
嘴部：EX41. 咬唇（紧张克制）。
眉部：EX14. 犹豫（眉毛微抬+眉头微皱）。
```
