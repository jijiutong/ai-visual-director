## 一、粒子系统美学

### PS1 自然粒子

| 粒子类型 | 物理行为 | 光影特征 | Prompt 要点 |
|---------|---------|---------|-----------|
| 雨滴 | 直线下落、速度均匀、遇面飞溅 | 逆光下发光、有运动模糊 | rain streaks, backlit, motion blur |

| 烟雾 | 湍流上升、渐扩散稀薄 | 体积感、半透明、遮挡/半遮 | volumetric smoke, turbulent diffusion, partial occlusion |
| 水花/飞溅 | 撞击后放射状飞散 | 逆光下晶莹、慢动作可见水滴形状 | water splash, radial scatter, backlit drops, slow motion |

## 二、实拍 vs 数字选择原则

### VF1 选择矩阵

| 场景 | 推荐 | 原因 |
|------|------|------|
| 真实物理交互(雨/风/火) | 实拍+后期增强 | 物理规律AI理解不了 |
| 角色的真实皮肤质感 | 实拍 | 微血管、汗毛、毛孔——CG还做不完美 |
| 超大规模(千军万马) | 数字 | 实拍成本/安全不可行 |
| 魔法/能量/非自然 | 数字 | 现实中不存在 |
| 大破坏/爆炸 | 实拍+数字增强 | 真实火光和碎片+数字扩规模 |
| 角色变老/变年轻 | 实拍+数字化妆 | 基础是真人表演+数字辅助 |
| 未来城市/异世界 | 数字+微缩模型 | 微缩模型的物理质感+数字扩展 |

## 三、合成原理

### CP1 合成层次（从后到前）

| 层序 | 内容 | 说明 |
|------|------|------|
| 1 天空/远景 | 天空、远山、星云、远景环境 | 最深，被前景遮挡 |
| 2 远景建筑/环境 | 远处的建筑、山脉、大气层 | 有大气透视(远=低对比+偏蓝) |
| 3 中景环境 | 场景的主体建筑/地形 | 主角的周围环境 |
| 4 中景角色/物体 | 次要角色、车辆、道具 | 半遮挡关系 |
| 5 前景角色 | 主角、关键动作 | 最前面、焦内或浅景深前 |
| 6 前景遮挡 | 树叶/栏杆/窗框 | 最前景，可能虚化 |
| 7 粒子/大气 | 尘/雨/雾/雪/烟 | 叠加在所有层之上 |
| 8 光效/后处理 | 光晕/lens flare/色散/暗角 | 最终画面滤镜 |

**Prompt 分镜要点**：明确每层的内容和遮挡关系。

---

### CP2 合成可信度核心法则

| 法则 | 说明 | 违反后果 |
|------|------|---------|
| 光线统一 | 所有元素被同一个光源照亮 | 光不对=一眼假 |
| 色彩统一 | 所有元素在同一色彩空间 | 色温不一致 |
| 透视统一 | 所有元素在同一焦段和消失点 | 空间感崩塌 |
| 分辨率统一 | 所有元素同清晰度/颗粒度 | CG太干净、实拍太糊 |
| 大气统一 | 远景=低对比偏蓝，近景=高对比 | 深度感丢失 |
| 黑电平统一 | 最暗点所有元素一致 | 暗部不融合 |
| 运动模糊统一 | 所有运动的模糊程度一致 | 运动不流畅 |

## 四、特效即叙事

| 叙事目的 | 特效处理 |
|---------|---------|
| 角色能力觉醒 | 视觉特效从无→有→可控→强大 |
| 世界崩坏 | 特效从有序→出现异常→崩解→混沌 |
| 时间操控 | 粒子冻结/倒流/加速=时间被操控 |
| 情感外化 | 角色的内心状态通过环境特效外化（愤怒=周围起火/悲伤=下雨） |
| 维度转换 | 从现实维度进入魔法/数字维度——视觉纹理全部切换 |
| 死亡/消散 | 角色死亡=物理瓦解为粒子→飘散 |
