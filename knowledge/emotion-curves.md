# 情绪曲线数据表

12种情绪曲线定义 + 各曲线镜数分配表。

## 一、12种情绪曲线定义

### EC1 标准四阶段（通用商业）

```
平静(25%) → 对峙(25%) → 爆发(25%) → 余韵(25%)
  铺垫         张力升级     高潮释放     收尾回响
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 4 |
| 适用类型 | 动作/爱情/科幻/喜剧/大多数商业类型 |
| 色彩倾向 | 冷→暖对比→高饱和→低对比统一 |

---

### EC2 悬疑五阶段

```
平静(20%) → 不安(20%) → 紧张(20%) → 爆发(20%) → 反转(20%)
  日常        异样出现     危机逼近     冲突揭示     真相颠覆
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 5 |
| 适用类型 | 悬疑/犯罪/推理/心理惊悚 |
| 特殊技法 | 反转阶段用 Rack Focus + Dutch Angle + 冷光突转 |

---

### EC3 喜剧四阶段

```
日常(25%) → 意外(25%) → 混乱(25%) → 解决(25%)
  正常生活    突发事件     连锁反应     和解回归
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 4 |
| 适用类型 | 喜剧/搞笑/轻松 |
| 特殊技法 | 混乱阶段用快速切换/表情夸张/明亮均匀光 |

---

### EC4 悲剧四阶段

```
美好(25%) → 裂痕(25%) → 崩溃(25%) → 余痛(25%)
  幸福起点    矛盾出现     彻底毁灭     空白收场
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 4 |
| 适用类型 | 悲剧/虐心/宿命 |
| 特殊技法 | 美好用温暖光/柔焦；崩溃用极端冷光/大面积黑色/面部特写；余痛用空镜/拉远/冷色调 |

---

### EC5 爱情五阶段

```
相遇(20%) → 试探(20%) → 心动(20%) → 确认(20%) → 余韵(20%)
  初次见面    互相了解     情感觉醒     关系确立     甜蜜延续
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 5 |
| 适用类型 | 爱情/甜宠/双向暗恋 |
| 特殊技法 | 相遇用聚光效果；试探用三分法+暖色暧昧光；心动用 Tight CU + 慢动作；确认用 Eye Level + 统一暖光 |

---

### EC6 成长六阶段

```
起点(15%) → 召唤(15%) → 尝试(20%) → 挫折(15%) → 突破(20%) → 蜕变(15%)
  平凡世界    机会出现     学习探索     失败低谷     觉醒反击     完成成长
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 6 |
| 适用类型 | 成长/冒险/热血/少年漫 |
| 色彩演变 | 起点暗淡→微光→多彩→暗淡→金光→明亮统一 |

---

### EC7 复仇五阶段

```
日常(15%) → 失去(15%) → 蓄力(25%) → 复仇(25%) → 虚无(20%)
  幸福        被摧毁      修炼准备     血腥复仇     复仇后空虚
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 5 |
| 适用类型 | 复仇/武侠/黑帮 |
| 特殊技法 | 蓄力阶段用暗调+局部光；复仇用极端对比+血红；虚无用冷灰+大面积留白 |

---

### EC8 压迫释放（限时任务）

```
倒计时开始(20%) → 压力递增(30%) → 临界点(20%) → 释放(20%) → 余波(10%)
  T-30min        T-20→T-5      最后5分钟     成功/失败     后果
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 5 |
| 适用类型 | 拆弹/救援/限时任务 |
| 特殊技法 | 每个时间节点景别逼近、运镜加速、色彩变极端 |

---

### EC9 压抑爆发

```
压抑(35%) → 裂缝(15%) → 压抑加深(20%) → 爆发(20%) → 消散(10%)
  忍耐        偶尔泄露     更大压力     彻底爆发     无力收尾
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 5 |
| 适用类型 | 心理/社会现实/家暴/霸凌 |
| 特殊技法 | 压抑阶段用冷灰/Static/低对比；爆发用极端暖红/Handheld/高对比 |

---

### EC10 双线情感

```
A线(现在): 相遇→发展→冲突→解决
B线(过去): 回忆→甜蜜→裂痕→揭示
最终汇合: A线解决 = B线揭示
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 双线交织 |
| 适用类型 | 破镜重圆/前世今生/失忆重逢 |
| 特殊技法 | A线和B线各有独立色彩和光影，汇合时统一 |

---

### EC11 反高潮

```
buildup(30%) → 假高潮(20%) → 低谷(25%) → 真正高潮(15%) → 余韵(10%)
 紧张积累        虚假释放       绝望        最后反击        平静
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 5 |
| 适用类型 | 智斗/悬疑/策略 |
| 特殊技法 | 假高潮后突然转入更低谷，再突然反转 |

---

### EC12 循环往复

```
平静→微澜→平静→波动→平静→高潮→平静
```

| 属性 | 值 |
|------|-----|
| 阶段数 | 螺旋式 |
| 适用类型 | 文艺/散文/生活流 |
| 特殊技法 | 每次"平静"都比上一次更低/更高，情绪整体螺旋 |

## 四、镜数与曲线联动规则
