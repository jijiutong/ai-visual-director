# 导演调教与表演指导

顶级导演如何通过 Blocking、走位、视线设计、表演方法把文字变成画面里的生命。

---

## 一、Blocking（走位调度）

### BK1 基本走位模式

| 编号 | 模式 | 描述 | 关系暗示 | 适用 |
|------|------|------|---------|------|
| BK1-A | 并肩同行 | 两人并排朝同一方向走 | 同盟、平等、共同目标 | 搭档、恋人 |
| BK1-B | 一前一后 | 一人领先一人跟随 | 权力差、引导/追随 | 师徒、上下级 |
| BK1-C | 面对面 | 两人正面相对静止 | 对峙、亲密、谈判 | 对决、分手、告白 |
| BK1-D | 背对背 | 各看各的方向 | 分歧、冷战、各自独立 | 夫妻裂痕、战友分离 |
| BK1-E | 圆形围绕 | 一人绕另一人转 | 审视、威胁、好奇 | 审讯、试探、吸引 |
| BK1-F | 交叉通过 | 两人从画面两侧走到对面交换位置 | 命运交错、错过 | 擦肩而过 |
| BK1-G | 追-逃 | 一人追逐一人避让 | 控制vs反抗、渴望vs恐惧 | 囚禁、暗恋、追杀 |

**Prompt 写法**：
```
走位：BK1-[编号]。[角色A]与[角色B][走位描述] → 暗示[关系/情绪]。
```

---

### BK2 深度走位

利用近/中/远景深的移动表达心理变化。

| 编号 | 模式 | 画面处理 | 心理含义 |
|------|------|---------|---------|
| BK2-A | 走向镜头 | 角色从画面深处走向镜头/观众 | 走向真相、直面恐惧、主动进攻 |
| BK2-B | 远离镜头 | 角色远离镜头走向画面深处 | 逃避、放弃、走向未知 |
| BK2-C | 深度分层 | 前景一人、中景一人、远景一人 | 关系层次——亲密度梯度 |
| BK2-D | 穿过门/窗/通道 | 角色穿过物理空间框架 | 心理转变、进入新阶段 |
| BK2-E | 停-走-停 | 角色停下思考→继续走→再停 | 犹豫、内心冲突 |
| BK2-F | 突然转向 | 角色一直朝一个方向走→突然90°转身 | 改变主意、发现新线索 |

---

### BK3 空间占领

| 模式 | 谁占得多 | 含义 |
|------|---------|------|
| 一人独大 | 角色A占据画面70% | 权力极不对等 |
| 平分画面 | A和B各占50% | 平等对峙/亲密 |
| 三人三角 | 三人呈三角站位 | 同盟-对立关系复杂 |
| 一人被挤压 | 主角被挤到角落 | 被边缘化、弱势 |
| 高度占领 | 站着的对坐着的 | 心理优势 |

## 二、视线设计（Eye Line Design）

### EL1 视线方向含义

| 方向 | 含义 |
|------|------|
| 看向左（往过去） | 回忆、过去、已知 |
| 看向右（往未来） | 期待、未来、未知 |
| 看向上方 | 希望、祈祷、理想、权威 |
| 看向下方 | 羞耻、屈服、思考、死亡 |
| 看镜头/观众 | 打破第四面墙、质问、告白 |
| 看画外 | 关注不在场的人/事 |
| 避开视线 | 逃避、说谎、羞耻、暗恋 |
| 视线空白 | 空洞望向前方=崩溃/麻木 |

---

### EL2 双人视线模式

| 模式 | 含义 |
|------|------|
| 互看双眼 | 亲密、信任、对峙 |
| A看B，B不看A | 单恋、被注视的压力、权力差 |
| 两人都不看对方 | 冷战、尴尬、无法面对 |
| A看B，B看C | 三角关系、注意力转移 |
| A透过B看画外 | 利用B、B不重要、A有心事 |

**Prompt 写法**：
```
视线设计：[角色A]视线[方向/模式]，[角色B]视线[方向/模式] → 暗示[关系动态]。
```

## 三、表演方法视觉转化

### PM1 方法派（Method Acting）

演员调用自身真实情感记忆——视觉上是"真的"。

| 视觉特征 | 处理方式 |
|---------|---------|
| 沉浸感 | 表情不是"表演"而是"发生" |
| 微表情连续 | 哭不是一下子哭，而是从眼眶微红→泪光→第一滴泪→崩溃 |
| 身体诚实 | 不是摆pose，是重力、惯性、肌肉真实作用 |
| 拍摄建议 | 长镜头、少打断、让演员沉浸 |

---

### PM2 梅斯纳（Meisner）

"把注意力放在对方身上，而不是自己"——视觉上是"真实的反应"。

| 视觉特征 | 处理方式 |
|---------|---------|
| 反应优先 | 听的人比说的人更有戏 |
| 视线锁定 | 两人的眼神交流不是演的，是真的看对方的反应 |
| 即兴感 | 动作自然，看起来不是排练过的 |
| 拍摄建议 | 双人中景、过肩镜头、捕捉反应镜头 |

---

### PM3 布莱希特（Brecht）

"让观众意识到这是戏，不是真的"——间离效果。

| 视觉特征 | 处理方式 |
|---------|---------|
| 看镜头/观众 | 直接打破第四面墙 |
| 不自然光 | 明显是舞台光/人工光 |
| 布景可见 | 不隐藏摄影棚/后台 |
| 标题卡 | 字幕/标题强行插入打断叙事 |
| 拍摄建议 | 刻意不流畅、强调人工感 |

## 四、群像调度

### GE1 群体走位模式

| 模式 | 站位 | 含义 |
|------|------|------|
| 金字塔 | 领袖居中高位，成员在两侧阶梯式 | 等级森严 |
| 圆形 | 所有人围成一圈 | 平等、共同体 |
| V字展开 | 领袖在前尖位，成员向两侧后方展开 | 冲锋队形 |
| 散点 | 各自散落在画面各处 | 分裂、各自为政 |
| 双阵营 | 画面左右分两团 | 派系对立 |

## 五、表演节奏设计

### PR1 情绪递进（大情绪不是突然来的）

| 层次 | 视觉 | 时长 |
|------|------|------|
| 种子 | 面部微僵、呼吸微变、手指微颤 | 2-3s |
| 萌芽 | 眼眶微红、嘴唇微抿、喉结滚动 | 2-3s |
| 生长 | 第一滴泪/第一声笑、肩膀轻微抖动 | 3-5s |
| 绽放 | 崩溃大哭/大笑/爆发 | 5-8s |
| 余韵 | 抽泣、沉默、茫然、虚脱 | 3-5s |

**Prompt 写法**：
```
表演节奏 PR1：情绪递进五层次。[情绪]从种子→萌芽→生长→绽放→余韵。
镜头跟随递进节奏 [景别/运镜变化]。
```

---

### PR2 表演停顿的力量

| 类型 | 时长 | 效果 |
|------|------|------|
| 呼吸停顿 | 0.5s | 一句话中的重要词前 |
| 打断停顿 | 1-2s | 被意外信息打断 |
| 决定停顿 | 2-3s | 做重大决定前的一刻 |
| 崩溃停顿 | 3-5s | 话说不下去/情绪堵住 |
| 沉默空镜 | 5s+ | 留给观众自己感受 |

## 六、动作表演设计

### AP1 行为即性格

| 行为 | 揭示的性格 |
|------|-----------|
| 反复整理衣领/袖口 | 强迫、焦虑、控制型 |
| 走路时手插口袋 | 缺乏安全感、内向 |
| 说话时比划很大 | 外向、自信、情绪化 |
| 总是避开身体接触 | 防御、创伤、洁癖 |
| 无意识地触碰特定部位（疤痕/戒指/项链） | 有隐藏秘密 |
| 吃东西的方式（狼吞虎咽/细嚼慢咽） | 出身/教养/当前状态 |
| 点烟/熄烟的动作 | 从容从容/烦躁 |

## 七、场景表演调度

### SP1 双人对峙

| 技法 | 描述 |
|------|------|
| 桌面阻隔 | 桌子/台面在两人之间=物理屏障=权力/关系屏障 |
| 距离变化 | 对峙从远到近=冲突升级，从近到远=缓和 |
| 高度差 | 一人坐一人站=权力/道德高低 |
| 镜头暗示 | 对峙阶段用Dutch Angle+手持→和解阶段用水平+稳定 |

---

### SP2 独白/内心戏

| 技法 | 描述 |
|------|------|
| 一人配空镜 | 不让角色占据画面，用环境代笔 |
| 长镜头不切 | 让观众和角色一起承受时间 |
| 身体传导 | 情绪从手→肩→脸，不直接从脸开始 |
| 镜子/窗子 | 独白时角色看镜子=与自我对话，看窗外=向往远方 |

## 八、大师导演表演技法速查

| 编号 | 导演 | 标志性表演技法 | 适用 |
|------|------|-------------|------|
| DPM1 | 是枝裕和 | 让演员长时间不说话只做日常动作——切菜、叠衣服、走路。在这些动作中流露真实情绪。| 家庭/日常/克制的情感 |
| DPM2 | 李沧东 | 长镜头跟随角色背部——观众看不到表情，但通过肩膀/脖子的紧张度感受情绪 | 压抑/社会/无力 |
| DPM3 | 滨口龙介 | 演员坐在车内/桌前长时间对话，固定机位不切——靠微表情变化推动节奏 | 对话/亲密关系 |
| DPM4 | 许鞍华 | 让角色在不经意间流露真情——不是在设计好的高潮点，而是日常瞬间 | 写实/女性/家庭 |
| DPM5 | 陈凯歌 | 仪式的慢动作表演——角色的一举一动像在举行仪式 | 史诗/东方美学 |
| DPM6 | 张艺谋 | 群体表演统一节奏——千军万马/宫女同时做同一动作，个人消失于集体 | 宏大/战争/仪式 |
| DPM7 | 姜文 | 角色边走边说长台词Bravura——表演与运镜同步，镜头跟住走位 | 快节奏/雄性/黑色幽默 |
| DPM8 | 贾樟柯 | 非职业演员的真反应——不演，只是活在画框里 | 纪实/社会/时代 |

## 在 Prompt 中使用

```
导演调度：[走位编号]。[角色]走位[描述] → [心理含义]。
视线设计：[模式]，[角色A]视线[方向]，[角色B]视线[方向]。
表演方式：[PM编号]，[视觉特征]，[拍摄建议]。
情绪递进：PR1 五层次，[景别变化路径]。
```
