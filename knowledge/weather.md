## 四、在 Prompt 中使用

```
天气与大气：[编号. 天气/大气效果]。[核心视觉描述]。光影[光影效果]。情绪[情绪词]。[环境互动：雨水打湿衣物/风吹动头发/雪花落在睫毛上等]。
```

例：
```
天气与大气：WT1. 暴雨。密集雨线如幕，地面水洼不断溅起水花，全身湿透衣物贴身，能见度低于50米。光影：雨面反射霓虹灯光，闪电瞬间照亮整个场景。情绪压抑悲壮。环境互动：雨水顺脸颊流下，头发湿透贴在额头，衣角不断滴水。
```
