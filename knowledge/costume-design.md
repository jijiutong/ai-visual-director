# 服装造型设计数据表

服装即叙事。好莱坞服装设计师的知识体系：服装叙事、面料语言、色彩编码、轮廓演化。

---

## 一、服装叙事原则

### CD1 服装即角色弧光

| 弧光类型 | 服装演变 | 例子 |
|---------|---------|------|
| 上升弧 | 暗→明、简→繁、粗→精 | 灰姑娘的蜕变 |
| 坠落弧 | 明→暗、整→破、精→简 | 贵族沦为乞丐 |
| 蜕变弧 | 一层层脱去旧自我→露出新自我 | 英雄觉醒 |
| 伪装弧 | 穿别人的身份→脱掉伪装→露出真我 | 卧底/间谍 |
| 成长弧 | 不合身的衣服→越来越合身=找到自我 | 青少年成长 |

---

### CD2 服装颜色编码

| 颜色 | 含义 | 变体 |
|------|------|------|
| 红 | 权力/激情/危险 | 暗红=压抑的权欲，正红=宣告主权 |
| 蓝 | 理性/冷静/忧郁 | 海军蓝=权威，浅蓝=天真 |
| 白 | 纯洁/死亡/新开始 | 婚纱白=纯真，苍白=病态 |
| 黑 | 权威/死亡/优雅 | 全黑=反派/力量，黑+金=最高权力 |
| 绿 | 自然/嫉妒/成长 | 翡翠绿=贵气，暗绿=阴谋 |
| 金 | 神圣/权力/贪婪 | 过多金=腐败，点缀金=高贵 |
| 灰 | 压抑/平庸/隐藏 | 深灰=压抑，银灰=科技感 |
| 紫 | 贵族/神秘/精神 | 帝王紫=最高权力，淡紫=浪漫 |

**角色服装颜色叙事**：
- 主角服装颜色渐变=角色弧光可视化
- 主角&反派服装颜色对立（如红vs蓝）
- 配角服装颜色靠近主角色系=同盟
- 角色服装颜色偏离原色系=背叛/转变

---

### CD3 服装演变编码

| 阶段 | 角色状态 | 服装特征 |
|------|---------|---------|
| 起点 | 初始状态 | 服装定义角色身份与阶级 |
| 转折1 | 离开舒适区 | 服装开始不适应环境（太正式/太简陋） |
| 中点 | 虚假胜利 | 穿"别人的衣服"（权/地位/身份的伪装） |
| 低谷 | 一切失去 | 服装被剥去/毁坏=身份崩塌 |
| 高潮 | 真正的自己 | 服装=角色内在的真正外化 |
| 终点 | 新的自我 | 服装融合了旅程的痕迹+新的身份 |

### FB2 面料动态

| 动态 | 效果 | 画面处理 |
|------|------|---------|
| 飘动 | 轻盈、自由、仙气 | 风+慢动作+逆光穿透 |
| 垂坠 | 沉稳、品质、静态力量 | 暗光下的面料重量感 |
| 褶皱 | 岁月、使用痕迹、真实 | 近景面料纹理 |
| 湿透 | 脆弱、暴露、危机 | 面料贴在身上=失去保护 |
| 撕裂 | 暴力、转折、蜕变 | 破口的设计位置=叙事信息 |
| 磨损 | 时间、贫困、旅途 | 膝盖/手肘/领口的磨损 |

## 三、轮廓与剪影

### SL1 服装轮廓

| 轮廓 | 形状 | 含义 |
|------|------|------|
| A型 | 上窄下宽 | 稳重、母性、大地感 |
| H型 | 上下同宽 | 现代、干练、中性 |
| V型 | 上宽下窄 | 权威、雄性、攻击性 |
| X型 | 收腰宽肩宽下摆 | 经典女性、华丽、年代感 |
| O型 | 圆润无棱 | 亲和、喜剧、福气 |

**轮廓叙事**：
- 轮廓变化=角色身体和身份的边界重新定义
- 从H型→X型=从压抑到释放的性别/自我认知
- 从A型→V型=从弱到强的权力增长

## 四、配饰语言

| 配饰 | 叙事功能 |
|------|---------|
| 戒指 | 承诺、控制、传承、隐藏身份 |
| 项链/吊坠 | 家族、回忆、护身符 |
| 手表 | 时间焦虑、阶级、控制欲 |
| 眼镜 | 知识分子、伪装、隐藏 |
| 手套 | 距离感、优雅、暴力预谋 |
| 帽子 | 身份、隐藏、年代、仪式感 |
| 腰带 | 权力中心、束缚、解放 |
| 鞋 | 阶层、旅程、舒适区 |

**配饰提示**：
- 同一配饰反复出现 = 叙事线索
- 取下配饰 = 放下身份/承诺
- 戴上配饰 = 接受新身份/关系
