# 材质与质感参考

用于在 prompt 中精确描述各类材质的视觉和触感特征，提升 AI 生成的画面质感。

---

| 编号 | 材质 | 视觉特征 | 光影表现 | 适用 |
|------|------|---------|---------|------|

## 六、在 Prompt 中使用

追加材质描述到角色或场景描述中：

```
材质细节：[人体材质]+[服装材质]+[道具材质]+[环境材质]。[光影在材质上的表现]。
```

例：
```
材质细节：皮肤[MT1]细腻光滑，战斗后略有擦伤[MT3]血迹湿润反光。白色交领丝绸[MT15]长衫有流动光泽，银色云纹刺绣在光下闪烁。腰间玄铁剑[MT7]有锐利金属反光。背景仙山岩石[MT24]粗糙纹理，云海半透明散射光。
```
