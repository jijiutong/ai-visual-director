## 四、混响空间（8种）

| 编号 | 空间 | 混响特征 | 适用 |
|------|------|---------|------|
| RS1 | 大教堂 | 长混响 3-5s，低频轰鸣，高频衰减 | 宗教/庄严 |
| RS2 | 小房间 | 短混响 0.5-1s，温暖亲密 | 私密对话 |
| RS3 | 山洞/洞穴 | 中长混响 2-3s，低频增强 | 冒险/隐藏 |
| RS4 | 大厅/宫殿 | 长混响 2-4s，庄严回声 | 宫廷/权力 |
| RS5 | 室外开阔 | 几乎无混响，只有风声环境音 | 自然/旷野 |
| RS6 | 走廊/隧道 | 定向回声，声音来回弹 | 紧张/追逐 |
| RS7 | 水下 | 低频闷响，高频被吸收 | 深海/潜水 |
| RS8 | 太空 | 绝对无声，只有船体传导振动 | 太空/科幻 |

## 五、AI 视频平台声音适配

| 平台 | 声音支持 | 备注 |
|------|---------|------|
| Sora | 自然语言描述声音 | "sound of rain falling heavily" |
| Runway Gen-3 | 无内置声音，需后期添加 | 可在 prompt 中描述声音氛围 |
| 可灵 Kling | 无内置声音 | 同上 |
| Luma | 无内置声音 | 同上 |
| Pika | 可选添加音效 | 有限音效库 |

## 六、在 Prompt 中使用

```
声音层：
- 环境音：[编号. 环境音]，[描述]
- 拟音：[编号. 拟音]，[描述]
- 音乐：[编号. 音乐类型]，节奏[快/慢]，[乐器描述]
- 混响空间：[编号. 空间]
```

例：
```
声音层：
- 环境音：SE1. 雨声，暴雨敲击屋顶和地面，雷鸣间歇轰响
- 拟音：FX3. 拔剑出鞘，金属摩擦+清越剑鸣；FX1. 脚步声，皮鞋在积水中沉重踏步
- 音乐：MU2. 管弦史诗，弦乐低音铺底+定音鼓推进，节奏从80 BPM加速到120 BPM
- 混响空间：RS4. 大厅/宫殿，庄严回声2-3s
```
