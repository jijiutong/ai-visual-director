## 五、环境描述模板

在 prompt 中描述环境时使用：

```
世界观场景：[编号. 环境名称]。[核心视觉元素描述]。氛围[氛围词]。光影[光影方案]。色彩[主色+辅色+点缀色]。[环境动态：风起/云涌/雨落/雪飘等]。[材质细节：石材质感/水面反射/金属光泽等]。
```

例：
```
世界观场景：EV1. 高山云海。山峰穿云而出，翻涌云海覆盖山谷，悬崖峭壁如刀削，远处几座孤峰隐约可见。氛围壮阔、仙气、高远。阳光穿透云层形成金色光柱照射云海。色彩青白+金色。云海翻涌流动，山风劲吹。岩石质感粗粝，云雾半透明层次。
```
