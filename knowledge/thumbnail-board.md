# 缩略故事板速出

先生成极简版（只用关键词+线框描述）快速验证方向，确认后再出完整版，省时间省 token。

## 工作流程

```
用户输入 → 生成缩略版（关键词+线框） → 用户确认方向 → 生成完整版
```

## 缩略版格式

```
【缩略故事板 - [片名]】

风格：[风格编号. 名称]
格式：[格式编号. 名称]
画幅：[画幅]

角色：
  [角色A]：[关键词] [关键词] [关键词]
  [角色B]：[关键词] [关键词] [关键词]

场景：[地点] [时间] [天气] [氛围关键词]

镜1 [景别] [运镜]：[关键词式画面描述]
镜2 [景别] [运镜]：[关键词式画面描述]
...

色彩：[色1] [色2] [色3]
节奏：[节奏类型]
关系：[关系类型]

—— 回复"确认"生成完整版，或指出需要修改的方向
```

## 缩略版 vs 完整版对比

| 维度 | 缩略版 | 完整版 |
|------|--------|--------|
| 长度 | 100-200 字 | 500-1000 字 |
| 描述粒度 | 关键词 | 完整句子 |
| 技术参数 | 简略 | 详尽 |
| 生成速度 | 快 | 慢 |
| 用途 | 方向验证 | 最终出图 |

## 确认后再出完整版

用户回复"确认"或"OK"后，基于缩略版生成完整 prompt，追加：

- 完整角色 DNA
- 详细灯光方案
- 情绪曲线分段描述
- 色彩叙事演进
- 质量约束和负面提示词
- 平台适配参数

用户指出修改方向时，在缩略版基础上调整，再生成完整版。

## 在 Prompt 中使用

缩略版输出：

```
【缩略故事板 - [片名]】

风格：[编号. 名称]
角色：[A关键词] + [B关键词]
场景：[地点] [时间] [氛围]
镜1-[镜N]：[关键词序列]
色彩：[色板]
节奏：[类型]

回复"确认"生成完整版，或说"改XX"调整方向。
```
