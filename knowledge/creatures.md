## 六、生物描述模板

在 prompt 中描述生物时使用：

```
生物外观：[编号. 生物名称]，[体型大小]，[主色+点缀色]，[特征1]+[特征2]+[特征3]，[气势/压迫感描述]，[动态描述]，[光线照射下的材质效果]。
```

例：
```
生物外观：CR1. 中国龙，巨物（身长数百米），青金色鳞片，蛇形长身+鹿角+鹰爪+龙须飘动，巨物压迫感云海翻腾，龙睛如电金光闪烁，鳞片在日光下反射青金色光芒，腾云驾雾穿梭云间。
```
