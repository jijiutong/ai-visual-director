# 音效设计表模板

> **治理**：生成前读 `state/format-contract-state.md`。产出标记 `draft`，不写回主状态。文本输出，无 asset_purpose 限制。详见 `rules/format-contract.md` §1.4。

按镜号列出完整音效方案。含环境音（SE）、拟音（FX）、音乐（MU）、混响空间（RS）。用于后期配音、音效制作、混音参考。

---

## Prompt 模板

```text
专业音效设计表，片名《[片名]》，类型 [类型]，总镜头 [N] 镜，情绪曲线 [EC编号]。

按镜头列出完整音效方案：

┌──────┬───────────────────────────────┬────────┬──────────────────────┐
│ 镜号 │ 画面内容（简述）              │ 音效层 │ 音效方案             │
├──────┼───────────────────────────────┼────────┼──────────────────────┤
│ SH0X │ [阶段名]：[画面简述]          │ 环境音 │ SE编号. 描述         │
│      │                               │ 拟音   │ FX编号. 描述         │
│      │                               │ 音乐   │ MU编号. 节奏 描述    │
│      │                               │ 混响   │ RS编号. 描述         │
│      │                               │ 特殊   │ [沉默/冲击/骤停]     │
├──────┼───────────────────────────────┼────────┼──────────────────────┤
│ ...  │ ...                           │ ...    │ ...                  │
└──────┴───────────────────────────────┴────────┴──────────────────────┘

整体音量曲线：[渐强/渐弱/起伏]
跨镜声音连贯性：[连续环境音/各镜独立/音乐贯穿]
特殊音效说明：[如有特殊处理]

音效编码参考：
  环境音 SE：knowledge/sound-design.md §一（16种）
  拟音 FX：knowledge/sound-design.md §二（20种）
  音乐 MU：knowledge/sound-design.md §三（12种）
  混响 RS：knowledge/sound-design.md §四（8种）
  类型推荐：knowledge/audio-reference.md（6类型×4阶段）
```

## 变量说明

| 变量 | 参考文件 | 填充方式 |
|------|---------|---------|
| 片名/类型 | `state/variable-registry.md` | project.title / story.genre |
| 镜号/阶段/动作 | `state/shot-state.md` → shots | 逐镜读取 |
| EC编号 | `engines/emotion-curve.md` | 自动匹配 |
| SE/FX/MU/RS 编号 | `knowledge/sound-design.md` | 按场景/动作/情绪自动匹配 |
| 类型推荐组合 | `knowledge/audio-reference.md` | 按 genre×阶段 查表匹配 |
| 台词时机 | `state/dialogue-map.md` | 音效避开台词、配合台词节奏 |

## 联动

← 读取 `state/sound-map.md`（音效数据）
← 读取 `state/shot-state.md`（镜号 + 动作 + 阶段）
← 读取 `state/dialogue-map.md`（台词时机配合）
← 读取 `knowledge/sound-design.md`（编码表）
← 被 `/sound` 子路由调用
→ 输出：音效设计表（含 SE/FX/MU/RS 编号+时机+描述）
