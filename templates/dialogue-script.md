# 台词脚本模板

> **治理**：生成前读 `state/format-contract-state.md`。产出标记 `draft`，不写回主状态。文本输出，无 asset_purpose 限制。详见 `rules/format-contract.md` §1.4。

独立台词输出。含角色台词表、节奏标注、语气标记、字幕方案。用于导演排练、配音录制、字幕制作。

---

## Prompt 模板

```text
专业台词脚本，片名《[片名]》，总台词 [N] 句，情绪曲线 [EC编号. 曲线名称]。

【角色台词基调】
[角色1]（[N]句）：语气基调 [描述]，语速习惯 [快/中/慢]，音量习惯 [大/中/小]
[角色2]（[N]句）：语气基调 [描述]，语速习惯 [快/中/慢]，音量习惯 [大/中/小]

【台词序列】（按镜号排序）

镜 SH0X — [阶段名]：[时长]
  角色 [说话人]（D[N]）：
    台词：「[台词内容]」
    表演要求：[delivery 语气标注]，[rhythm 节奏标注]
    字幕：[位置] [时长] [口型同步: ✅/❌]
    承接上句：D[N-1]「[前一句台词]」（如有）

镜 SH0X — [阶段名]：[时长]
  角色 [说话人]（D[N]）：
    台词：「[台词内容]」
    表演要求：[delivery 语气标注]，[rhythm 节奏标注]
    字幕：[位置] [时长] [口型同步: ✅/❌]

...（逐镜展开所有台词）

【沉默设计】
  镜N→镜N+1：[SL编号. 沉默类型] [时长] — [沉默原因/效果]

【字幕方案】
  字幕格式：底部中央 / 黑底白字 / 中文黑体
  双人对白：各自底部左右分区
  旁白/内独白：顶部中央 / 斜体 / 灰底
  时间轴：全部对齐 shot-state 时间码

【台词连贯性检查】
  □ 每句台词归属明确（speaker 唯一）
  □ 相邻台词之间有情绪/内容承接（不凭空跳话题）
  □ 沉默设计有动机（不无故冷场）
  □ 台词量不压过画面表现力（动作镜无冗余台词）
  □ 口型同步镜号与正面/近景镜匹配

台词语气标注参考 knowledge/dialogue-rhythm.md（停顿/重音/语速/音量/沉默编码）。
```

## 输出示例

```text
专业台词脚本，片名《剑冢决》，总台词 6 句，情绪曲线 EC7 复仇。

【角色台词基调】
墨渊/师父（4句）：语气基调 低沉压抑+沧桑，语速习惯 慢，音量习惯 中→低
韩霜/徒弟（2句）：语气基调 锋利激动+压抑，语速习惯 中→快，音量习惯 中→高

【台词序列】

镜 SH03 — 对峙：[2s-5s]
  角色 墨渊（D1）：
    台词：「你走吧……我不拦你。」
    表演要求：[VL2 低声] [SP2 慢] [ST8 断续音]
    节奏：你走｜吧……[SL2 犹豫沉默 2s]｜我**不**拦你。
    字幕：底部中央 3-6s 口型同步 ✅

镜 SH04 — 对峙→爆发：[5s-8s]
  角色 韩霜（D2）：
    台词：「为什么！」
    表演要求：[VL4→VL5 提高→大喊] [ST5 颤音] [SP4 快]
    节奏：**为什么**！[SL3 震惊沉默 3s]
    字幕：底部中央 7-8s 口型同步 ✅
    承接上句：D1「你走吧……我不拦你。」

镜 SH05 — 爆发：[8s-14s]
  角色 韩霜（D3）：
    台词：「是你教我这把剑……现在你用同一把剑指着我？」
    表演要求：[VL4 提高] [ST2 情感重音] [SP6 渐快→]
    节奏：[SP3→SP6] 是你教我这把剑｜……｜现在你用**同一把剑**｜指着我？
    字幕：底部中央 8-14s 口型同步 ✅

镜 SH05 — 爆发：[12s-14s]
  角色 墨渊（D4）：
    台词：「……」
    表演要求：[VL2 低声] [SL3 震惊沉默 3s] — 无声，被徒弟的话击中
    节奏：3s 对视沉默 → 缓缓闭眼
    口型同步 ❌

镜 SH06 — 余韵：[14s-15s]
  角色 墨渊（D5）：
    台词：「所以这一剑……我来还。」
    表演要求：[VL2 低声] [ST5 颤音] [SP7 渐慢→] [SP2 慢]
    节奏：[SP3→SP7] 所以这**一**剑｜……[SL2 犹豫 2s]｜我来还。
    字幕：底部中央 14-15s 口型同步 ✅
    承接上句：D3「是你教我这把剑……」

【沉默设计】
  SH03→SH04：SL2 犹豫沉默 2s — 墨渊开口艰难
  SH04→SH05：SL3 震惊沉默 3s — 韩霜被自己的话震住
  SH05 中段：SL4 对峙沉默 5s+ — 两师徒无声对视，剑锋相向

【字幕方案】
  字幕格式：底部中央，黑底白字(80%不透明)，中文黑体 18px
  口型同步镜头：SH03/SH04/SH05(SH05 双人各一句)/SH06
  时间轴：全部对齐 shot-state 时间码
```

## 变量说明

| 变量 | 参考文件 | 填充方式 |
|------|---------|---------|
| 角色名 + 台词归属 | `state/variable-registry.md` → characters | 自动从角色列表匹配 |
| shot_id + 阶段 | `state/shot-state.md` → shots | 自动分配台词到对应镜 |
| delivery + rhythm | `knowledge/dialogue-rhythm.md` | 按情绪自动标注 |
| EC编号 | `engines/emotion-curve.md` | 自动匹配 |
| DR编号 | 本模板自动标注 | 正常→DR1, 紧张→DR2, 悲伤→DR3, 爆发→DR4 |

## 联动

← 读取 `state/dialogue-map.md`（台词数据）
← 读取 `state/variable-registry.md`（角色 + 情绪曲线）
← 读取 `state/shot-state.md`（镜号 + 阶段 + 时长）
← 被 `/dialogue` 子路由调用
→ 输出：台词脚本（含节奏标注+字幕方案）
