# 视觉干净度规则

定义画面"脏"的标准、各输出类型的默认密度、需自动清理的视觉污染。作为 `prompt-declutter` 的规则来源。

---

## 一、什么是画面脏

> ⚠ 脏 ≠ 真实质感。毛孔、film grain、明暗对比、皮肤红润 —— 这些在真人/写实类型里是** quality**，不是**dirt**。判断标准：看 genre。

| 检查项 | 问题表现 | 例外（非脏） | 修复方向 |
|--------|---------|-------------|---------|
| 随机网格 | 背景出现无意义科技线条/网格 | 科幻 HUD 里有意义的线框 | `hud_elements=none` / `minimal` |
| 伪 UI | 画面出现不可读面板、按钮、图标 | 科幻神使角色系统卡（LS13）的交互界面 | 禁止 fake UI clutter |
| 微文字污染 | 大量不可读小字抢画面 | — | `text_level=none` / `minimal` |
| 粒子过多 | 雾、光点、碎屑淹没主体 | 战争/废土的灰尘、玄幻的灵气粒子 | 降低 `particle_level`（genre 感知） |
| 背景过满 | 主体不突出，背景细节抢眼 | — | 降低 `background_complexity` |
| 纹理噪点 | 纸纹、脏污、锐化过度 | **真人皮肤的毛孔/细纹不是噪点** | genre 感知：写实类允许 medium+ |
| film grain | grain 过量覆盖画面 | **战争/悬疑/写实的 grain 是质感** | genre 感知：写实类允许 cinematic+ |
| 光线不均匀 | 平板均匀柔光 | **明暗对比（chiaroscuro）是电影感** | genre 感知：写实/悬疑禁止均匀光 |
| 标注过多 | 角色卡信息/箭头/文字压住人物 | — | 降低 `annotation_level` |
| 层级不清 | 不知道第一眼看哪里 | — | 强化 focal hierarchy，明确主体 |

### Genre 白名单（以下不算脏）

| 质感 | 允许的 genre | 原因 |
|------|------------|------|
| 皮肤毛孔/细纹/自然瑕疵 | 武侠/战争/末日/悬疑/纪录片/写实/宫廷 | 真实感需要，不是噪点 |
| 自然红润/微血管 | 爱情/都市/古风仙气 | 活人感 |
| 汗水/湿润反光 | 武侠/战争/战斗/末日 | 状态真实 |
| film grain | 战争/末日/悬疑/纪录片/写实/科幻 | 电影质感 |
| 灰尘/泥垢 | 战争/末日/废土/武侠(江湖线) | 环境真实 |
| 明暗强对比/硬光 | 武侠/战争/悬疑/末日/科幻 | 戏剧张力 |
| 霓虹/体积光/边缘光 | 科幻/赛博 | 世界观美学 |
| 烛光/窗光/暖色不均匀 | 宫廷/爱情/古风 | 时代/氛围真实 |

## 二、各输出类型默认密度

| 输出类型 | 默认密度 | 推荐控制 |
|---------|---------|---------|
| 角色卡 | 1-2 | 白底/浅灰、主体优先、少文字 |
| 三视图 | 1 | 极简、无噪点、无背景 |
| 面部一致性 | 1 | 统一背景、统一光线 |
| 服饰细节 | 2 | 可有局部标注，但不压细节 |
| 场景卡 | 2-3 | 保留空间层次，不堆装饰 |
| 分镜图 | 2 | 每格清楚，参数克制 |
| 视频首帧 | 2 | 干净、可延展、少文字 |
| 海报 | 3-4 | 可丰富，但主体第一 |
| 世界观板 | 4 | 可复杂，但分区明确 |
| HUD 科技卡 | 3 | HUD 克制，线框有意义 |

---

## 三、默认追加质量词

所有 prompt 末尾自动追加（除非输出类型明确允许更高密度）：

```text
clean composition, controlled detail, clear focal hierarchy, low visual noise,
subject-first layout, restrained background complexity, no random grid artifacts,
no fake UI clutter, no meaningless micro text, no over-detailed background stealing focus
```

## 四、致命 vs 非致命

| 问题 | 级别 | 理由 |
|------|------|------|
| 角色卡背景太复杂 | **致命** | 视频锚点失效，角色一致性崩 |
| 视频首帧大量文字/HUD | **致命** | AI 视频工具无法理解，画面漂移 |
| 三视图有背景/噪点 | **致命** | 三视图必须干净才能锁角色 |
| 海报粒子稍多 | 非致命 | 海报允许丰富 |
| 世界观板信息多但分区清楚 | 放行 | 世界观板设计意图 |
| 场景卡氛围粒子适度 | 非致命 | 场景需要氛围 |
| 分镜图标注挤占画面 | **致命** | 看不清镜头内容 |
