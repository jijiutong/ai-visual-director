# 连续性检查

自动检查镜头间的连贯性（时间线/空间逻辑/角色位置/服装一致性），标出断裂点。

## 检查维度

### 1. 时间线连续性

| 检查项 | 规则 | 断裂示例 |
|--------|------|---------|
| 时间推进 | 镜头时间必须线性推进或明确标注闪回 | 镜3夜晚 → 镜4白天（无过渡说明） |
| 时长累计 | 各镜时长之和 = 总时长 | 7镜每镜3秒 = 21秒 ≠ 标注的15秒 |
| 时序标注 | 闪回/闪前必须明确标注 | 无标注的时间跳跃 |

### 2. 空间逻辑连续性

| 检查项 | 规则 | 断裂示例 |
|--------|------|---------|
| 场景过渡 | 场景变化必须有过渡说明 | 镜3办公室 → 镜4森林（无过渡） |
| 角色位置 | 角色在连续镜头中位置变化必须合理 | 镜3在左边 → 镜4突然在右边（无移动说明） |
| 空间关系 | 两人相对位置变化必须连贯 | 镜3面对面 → 镜4背靠背（无转身说明） |

### 3. 角色一致性

| 检查项 | 规则 | 断裂示例 |
|--------|------|---------|
| 服装 | 同一角色服装在所有镜头中一致 | 镜3白衬衫 → 镜4黑夹克（无换装说明） |
| 发型 | 发型不变 | 镜3长发 → 镜4短发 |
| 配饰 | 配饰位置不变 | 镜3玉佩在腰间 → 镜4消失 |
| DNA 字段 | 所有 13 字段一致 | 任何字段变化无说明 |

### 4. 情绪曲线连续性

| 检查项 | 规则 | 断裂示例 |
|--------|------|---------|
| 情绪递进 | 情绪必须按曲线推进 | 平静 → 爆发（跳过对峙阶段） |
| 色彩演进 | 色彩变化必须与情绪匹配 | 爆发阶段用冷灰蓝（应用高对比色） |
| 节奏匹配 | 节奏必须与情绪阶段匹配 | 平静阶段用快切（应用慢节奏） |

### 5. 光线连续性

| 检查项 | 规则 | 断裂示例 |
|--------|------|---------|
| 光源方向 | 同一场景光源方向一致 | 镜3左侧光 → 镜4右侧光（无光源移动说明） |
| 色温 | 同一时间段色温一致 | 镜3暖光 → 镜4冷光（同一时间） |
| 明暗 | 明暗变化必须有原因 | 突然变暗无说明（应是灯光变化/天气变化） |

## 检查输出格式

```
【连续性检查报告】

✅ 时间线：通过
⚠️ 空间逻辑：镜3→镜4场景过渡缺失，建议追加"切至"说明
✅ 角色一致性：通过
⚠️ 情绪曲线：镜2→镜3跳过对峙阶段，建议调整
✅ 光线连续性：通过

修复建议：
1. 镜3→镜4追加场景过渡描述："切至[新场景]"
2. 镜2→镜3之间插入对峙过渡镜，或调整情绪标注
```

## 自动修复

| 断裂类型 | 自动修复 |
|---------|---------|
| 场景过渡缺失 | 追加"切至[场景]"/"淡入[场景]" |
| 服装变化无说明 | 追加"角色已换装为[新服装]" |
| 情绪跳跃 | 追加过渡描述或调整阶段标注 |
| 光源方向变化 | 追加光源移动说明 |

## 用户指令

| 用户指令 | 行为 |
|---------|------|
| "检查连续性" | 仅输出检查报告 |
| "修复后生成" | 检查 + 自动修复 + 生成 |
| 默认 | 自动检查，发现断裂标注并建议修复 |
