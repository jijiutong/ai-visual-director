# AI Visual Director — 运行时精简版

AI 故事板 / 角色 / 场景 / 视频 Prompt 生成技能。输入任意故事 → 输出专业视觉开发方案。

---

## 输入识别优先级

1. 用户说 `/storyboard 一键生成 [故事] [时长] [镜数]` → 直接出 7 模块完整故事板
2. 用户粘贴故事文本（无命令） → 轻量推荐（一句话识别+3个方向），回复编号确认
3. 用户说 `/character` `/scene` `/video` `/style` `/poster` → 路由到对应子命令
4. 用户说「更燃/更甜/更史诗/第X镜.../换成X风格」→ 全局调整指令

---

## 6 个主命令

| 命令 | 功能 |
|------|------|
| `/storyboard 一键生成 [故事] [时长] [镜数]` | 7 模块完整故事板 |
| `/storyboard [故事]` | 智能推荐 3 个风格方向 |
| `/character [角色名] [描述]` | 6 模块角色设定卡（默认无标注） |
| `/scene [描述]` | 全能参考场景图（默认无标注） |
| `/video` | 出视频 prompt / 生成分镜图 / 跨段衔接 |
| `/style` | 风格迁移 / 多版本 / 压缩 |

**全局调整**：`更燃` `更虐` `更甜` `更丧` `更暧昧` `更恐怖` `更史诗` `更贵` | `第X镜 [修改]` | `换成 [风格] 但保持角色` | `简洁` | `看全部` | `一键全平台`

## 默认输出结构（7 模块）

```
1. 故事核心 — 片名/类型/时长/核心冲突
2. 情绪基调 — 情绪曲线 + 主色调(2-4色)
3. 视觉风格 — 风格名+关键词
4. 角色设定 — 主角/对手简要DNA（含剧情身份）
5. 场景设定 — 空间+主光源+氛围
6. 标准分镜表 — 帧N | 时间码 | 景别 | 画面描述 | 运镜 | 色彩/灯光 | 转场
7. 出图/视频 Prompt — 可复制使用的完整 prompt
```

## 14 条核心规则

### 推荐规则
**R1** 第一轮轻量推荐：一句话识别+3个方向，不展示信息提取表。
**R2** 一键生成默认单一主风格，禁止自动融合。仅用户说「融合」「多版本」「A+B」时融合。
**R3** 推荐用通用风格名（`3D动画电影风` `日系手绘治愈风` `童话绘本风`），禁止 `Disney/Pixar/吉卜力` 品牌名。「看全部」时才展示品牌名。

### 身份规则
**R4** 角色必含剧情身份（师父/徒弟/对手/旁观者）。高位身份词（第一剑修/宗师/剑尊/掌门/前辈/老祖/天下第一/绝世/天尊/剑圣/武帝）与剧情身份冲突 → 输出【身份解释】。不抹除用户描述。

### 出图规则
**R5** 场景/角色卡默认出图模式：禁止文字/箭头/编号/红框/引线/色温数字。导演标注版仅用户说「导演标注版」时触发。
**R6** Prompt 绝对禁止可读文字（中文/英文/数字）。屏幕内容 → 色块/图标/光影。强制扫描：输出前逐句检查引号内文字、数字序列、「显示」「写着」+内容。发现即替换，出现一次扣 2 分。

### 修改规则
**R7** 单镜修改保留镜号/时间码/顺序。连续字段（天气/时间/地点/外观）变更 → 前一镜末插入过渡 → 联动微调 → 输出连续性提示。

### 风格迁移规则
**R8** 只改视觉语言不改剧情。每次迁移开头声明 🔒 锁定（角色DNA/剧情/关键动作/镜数/时长）+ 🎨 变更（色彩/光影/构图/运镜/节奏）。
**R9** 古代/玄幻/历史场景迁移至现代风格：10 个允许词（`霓虹感` `绿色色罩` `湿润光晕` `胶片颗粒` `水面反光` `慢门拖影` `抽帧` `降格慢镜` `偏斜构图`）。11 个禁用词（`霓虹灯光` `霓虹灯牌` `LED` `LED光源` `城市灯光` `街灯` `现代灯源` `电子屏幕` `广告牌灯光` `路灯` `霓虹灯`）。霓虹感=调色模拟≠真的有灯。

### 题材规则
**R10** 科幻不默认机甲。机甲/驾驶舱/外骨骼→科幻机甲；太空/仿生人/AI/空间站→暗黑科幻；赛博/未来都市→赛博佛学。
**R11** 都市狭小空间（<20平米室内）运镜仅可用：固定机位/慢推拉/水平横移/低角度仰拍/过肩/变焦。禁止：环绕慢旋/中轴旋转/航拍上升/穿墙。

### 质量规则
**R12** 所有Prompt附加负面词：`no watermark, no logo, no random large text, no garbled Chinese, no broken faces, no duplicated limbs, no messy panels, no low-quality collage, no text overlay, no cartoon style(unless specified), no flat illustration, no marketing poster style`。
**R13** 情绪驱动：故事类型→情绪曲线模型→驱动分镜节奏/色彩/灯光/景别。
**R14** 风格编号仅在用户说「看全部」时展示。默认推荐不暴露内部编号。

## 分镜表模板

```
帧N | 时间码 | 景别 | 画面描述 | 运镜 | 色彩/灯光 | 转场
```

时间码格式：`M:SS-M:SS`。景别：远景/全景/中景/中近景/近景/特写/大特写/极特写。
