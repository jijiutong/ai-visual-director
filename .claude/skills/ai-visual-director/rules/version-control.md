# Prompt 版本管理

记录每次修改历史，支持"回滚到第2版"、"对比v1和v3"。

## 版本记录格式

每次生成或修改都记录一个版本：

```
【版本记录 - [片名]】
v1：[时间] [修改摘要]
  - 风格：[编号]
  - 格式：[编号]
  - Mood：[有无]
  - 变更：首次生成
v2：[时间] [修改摘要]
  - 变更：镜3暗一点 / 角色换长发
v3：...
```

## 版本操作

| 用户指令 | 行为 |
|---------|------|
| "回滚到第2版" | 恢复 v2 的完整 prompt |
| "对比v1和v3" | 并排展示 v1 和 v3 的差异 |
| "看版本历史" | 列出所有版本及变更摘要 |
| "保存当前版" | 手动标记当前为正式版本 |
| "删除v2" | 删除指定版本 |

## 差异对比格式

```
【版本对比 v1 vs v3】

风格：v1 东方玄幻 → v3 赛博朋克×东方玄幻（融合）
角色DNA：v1 [原DNA] → v3 [变更后DNA]
镜3：v1 [原描述] → v3 [新描述]
色彩：v1 冷灰蓝 → v3 冷灰蓝+赛博青（融合加成）
Mood：v1 无 → v3 更燃
新增：v3 追加视频适配 / 音乐音效参考
```

## 版本命名

| 类型 | 命名 |
|------|------|
| 首次生成 | v1（初始版） |
| 单镜修改 | vX.X（小版本） |
| 风格变更 | vX.0（大版本） |
| 格式变更 | vX.0（大版本） |
| Mood 调整 | vX.X（小版本） |
