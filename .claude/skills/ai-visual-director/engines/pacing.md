# 分镜节奏引擎

根据类型自动匹配分镜节奏：动作片快切、文艺片长镜头、悬疑片渐进。

## 节奏类型

### 快切节奏（动作/战争/格斗）

| 参数 | 值 |
|------|-----|
| 每镜时长 | 0.5-1.5 秒 |
| 7镜总时长 | 15 秒 |
| 13镜总时长 | 15-20 秒 |
| 剪辑频率 | 高（频繁切换） |
| 运镜特点 | Crash Zoom / Handheld / Dutch Angle |
| 景别变化 | 大范围跳跃（ELS → ECU → Wide） |
| 声音设计 | 快速音效 + 打击声 + 爆炸 |

**Prompt 追加**：
```
分镜节奏：快切模式，每镜 0.5-1.5 秒，7镜共 15 秒。高剪辑频率，景别大范围跳跃（大远景→特写→全景），运镜以 Crash Zoom/Handheld/Dutch Angle 为主。声音：快速音效 + 打击声 + 爆炸。
```

### 标准节奏（都市/悬疑/剧情）

| 参数 | 值 |
|------|-----|
| 每镜时长 | 1.5-3 秒 |
| 7镜总时长 | 15-20 秒 |
| 13镜总时长 | 20-30 秒 |
| 剪辑频率 | 中（自然过渡） |
| 运镜特点 | Push In / Pull Out / Tracking / Pan |
| 景别变化 | 渐进式（Wide → MS → CU → Wide） |
| 声音设计 | 环境音 + 对话 + 渐强配乐 |

**Prompt 追加**：
```
分镜节奏：标准模式，每镜 1.5-3 秒，7镜共 15-20 秒。自然剪辑过渡，景别渐进式变化（远景→中景→近景→拉远），运镜以 Push In/Pull Out/Tracking/Pan 为主。声音：环境音 + 对话 + 渐强配乐。
```

### 慢节奏（文艺/治愈/情感）

| 参数 | 值 |
|------|-----|
| 每镜时长 | 3-5 秒 |
| 7镜总时长 | 20-30 秒 |
| 13镜总时长 | 30-45 秒 |
| 剪辑频率 | 低（长镜头） |
| 运镜特点 | Slow Push / Static / Slow Pull Out |
| 景别变化 | 缓慢过渡（Wide → MS → CU 缓慢） |
| 声音设计 | 轻柔配乐 + 自然音 + 沉默间隙 |

**Prompt 追加**：
```
分镜节奏：慢节奏模式，每镜 3-5 秒，7镜共 20-30 秒。低剪辑频率长镜头风格，景别缓慢过渡，运镜以 Slow Push/Static/Slow Pull Out 为主。声音：轻柔配乐 + 自然音 + 沉默间隙。
```

### 渐进节奏（悬疑/惊悚/恐怖）

| 参数 | 值 |
|------|-----|
| 每镜时长 | 递变：2秒 → 1秒 → 0.5秒 → 3秒 |
| 7镜总时长 | 15-20 秒 |
| 13镜总时长 | 20-30 秒 |
| 剪辑频率 | 从低到高再骤停 |
| 运镜特点 | Static → Slow Push → Handheld → Static |
| 景别变化 | Wide → MS → CU → ECU → 突然拉远 |
| 声音设计 | 安静 → 渐强心跳 → 寂静 → 冲击音效 |

**Prompt 追加**：
```
分镜节奏：渐进模式，节奏从慢到快再到骤停。镜1-2每镜2秒（平静铺垫），镜3-4每镜1.5秒（不安升级），镜5-6每镜0.5秒（快速逼近），镜7停3秒（反转/冲击）。运镜从 Static → Slow Push → Handheld → Static。声音：安静 → 渐强心跳 → 寂静 → 冲击音效。
```

### 喜剧节奏（搞笑/轻松/四格）

| 参数 | 值 |
|------|-----|
| 每镜时长 | 2-3 秒（含 pause 留白） |
| 剪辑频率 | 中（有节奏感） |
| 运镜特点 | Static → Quick Zoom → Static（喜剧 timing） |
| 景别变化 | MS → CU（表情夸张）→ MS（反应） |
| 声音设计 | 轻快配乐 + 喜剧音效 + 沉默留白 |

**Prompt 追加**：
```
分镜节奏：喜剧模式，每镜 2-3 秒含 pause 留白。喜剧 timing 运镜：Static → Quick Zoom（反应）→ Static。景别：中景对话 → 特写夸张表情 → 中景反应。声音：轻快配乐 + 喜剧音效 + 沉默留白（punchline 前）。
```

| 惊悚/恐怖/密室 | 渐进 |
| 爱情/治愈/情感 | 慢节奏 |
