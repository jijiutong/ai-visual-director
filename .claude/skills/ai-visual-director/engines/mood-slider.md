## Mood 叠加规则

用户可以说多个 mood，按权重叠加：

- "更燃更虐" → MD1 + MD2：爆发加重 + 痛苦特写
- "更甜更暧昧" → MD3 + MD5：余韵加重 + 手指/眼神特写
- "更丧更恐怖" → MD4 + MD6：余韵拉长 + 暗部极端化
- "更神圣更空灵" → MD18 + MD16：金色顶光 + 半透明飘渺
- "更疯狂更暴力美学" → MD13 + MD23：扭曲高饱和 + 慢动作血腥细节

## 在 Prompt 中使用

```
Mood 调整：[编号. Mood 名称]。[对应的色彩/镜头/节奏/构图调整描述]。
```

例：
```
Mood 调整：MD18. 更神圣 + MD16. 更空灵。色彩：金色顶光 + 白色半透明 + 浅蓝飘渺。镜头：Low Angle + Crane Up + Slow Pull + Soft Focus。节奏：慢→庄严→飘逸。构图：顶光笼罩 + 大面积留白 + 主体悬浮感。
```
