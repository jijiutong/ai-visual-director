## 八、在 Prompt 中使用动物

### 描述模板

```
动物外观：[编号. 动物名称]，[体型描述]，[毛色/鳞片颜色]，[特征1]+[特征2]+[特征3]，[姿态/行为]，[运动状态]，[光线/环境效果]。
```

### 互动描述模板

```
互动场景：角色[角色名]与[编号. 动物名称][互动类型编号. 互动名称]。角色[角色动作]，动物[动物动作]。[两者相对位置]。[情绪氛围]。
```

### 示例

```
动物外观：AN1. 战马，体型高大健壮（肩高170cm），青黑色皮毛，佩戴青铜战甲+红色缰绳，昂首嘶鸣前蹄刨地，四蹄蓄力待冲锋，夕阳下战甲反射暗金色光芒。

互动场景：角色[将军]与AN14. 虎 AN-I7. 猎狼伴侧。将军挥剑前冲，虎从左侧扑向敌人。将军与虎同向进攻，虎距将军两步。战场肃杀，虎啸与喊杀声交织。
```

### 一致性检查清单

- [ ] 四肢数量正确（马/虎/狼/鹿=4条，鸟=2腿2翼，蛇=0条，金蟾=3条）
- [ ] 尾巴位置和形态符合物种特征
- [ ] 耳朵形状与物种匹配（非通用"动物耳"）
- [ ] 嘴/喙的结构准确（不开合超出解剖限制）
- [ ] 神话动物区分中西方版本（龙/凤凰最易混淆）
- [ ] 骑乘时鞍具/缰绳物理合理性（不穿透动物身体）
- [ ] 角色与动物比例正确（骑乘马vs矮种马差异明显）
- [ ] 动态场景中运动方向一致（头发/鬃毛/尾巴/衣物同向）
