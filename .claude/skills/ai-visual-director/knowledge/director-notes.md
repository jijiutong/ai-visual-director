# 导演阐述

追加导演注释段落：为什么选这个镜头/为什么这个色彩/角色弧光解读。

## 导演阐述内容

### 镜头选择理由

| 镜头 | 阐述角度 | 示例 |
|------|---------|------|
| 开场 ELS | 为什么用大远景开场 | "以大远景开场建立世界观的孤独感，让观众先感受环境再进入角色" |
| Crash Zoom | 为什么用疾推 | "疾推模拟角色内心冲击，让观众在生理层面感受到震惊" |
| Dutch Angle | 为什么用倾斜镜头 | "倾斜角度暗示心理失衡，世界不再稳定" |
| Pull Out | 为什么用拉远收尾 | "拉远让角色重新融入环境，暗示孤独感的回归/释怀" |

### 色彩选择理由

| 色彩 | 阐述角度 | 示例 |
|------|---------|------|
| 冷暖分割 | 为什么用冷暖对比 | "冷暖分割光象征两人内心的对立与吸引——各站自己的世界，却被对方照亮" |
| 色彩演进 | 为什么颜色变化 | "从冷灰到暖金的色彩演进，是角色内心从孤独到温暖的视觉化" |
| 高饱和爆发 | 为什么高潮用高对比 | "高潮时刻色彩饱和度达到峰值，情感张力在视觉上同步爆发" |

### 角色弧光解读

| 角色 | 弧光 | 示例 |
|------|------|------|
| 主角 | 从 X 到 Y | "主角从冷漠到柔软，镜头从远景（疏离）逐渐推进到特写（亲密）" |
| 对手 | 从 X 到 Y | "对手从压迫到脆弱，仰拍逐渐变为平视，权力的消长" |
| 关系 | 从 X 到 Y | "两人从对角线对峙到并肩站立，构图的接近是关系的外化" |

### 节奏选择理由

| 节奏 | 阐述角度 | 示例 |
|------|---------|------|
| 快切 | 为什么用快切 | "快切模拟心跳加速的生理节奏，让观众在剪辑频率中感受到紧张" |
| 长镜头 | 为什么用长镜头 | "长镜头强迫观众与角色共处同一时空，不做剪辑逃避" |
| 渐进 | 为什么节奏变化 | "从慢到快到骤停，是悬疑的呼吸——让观众在节奏中感受不安" |

## 导演阐述模板

```
【导演阐述】

关于镜头选择：
[镜头选择理由]

关于色彩叙事：
[色彩选择理由]

关于角色弧光：
[角色弧光解读]

关于节奏设计：
[节奏选择理由]

核心视觉主题：
[一句话总结整个故事板的视觉主题和意图]
```

## 使用场景

- 用户说"加导演阐述" → 追加导演注释
- 用户说"为什么要这样拍" → 单独输出导演阐述
- 全案板模式中可选择是否包含导演阐述段
- 汇报/提案场景中作为补充说明

## 在 Prompt 中使用

追加导演阐述段（作为 prompt 的补充说明，不进入图像生成 prompt）：

```
【导演阐述】
关于镜头选择：[理由]
关于色彩叙事：[理由]
关于角色弧光：[解读]
关于节奏设计：[理由]
核心视觉主题：[一句话]
```
