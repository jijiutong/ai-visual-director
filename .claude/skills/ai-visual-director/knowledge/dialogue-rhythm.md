# 台词节奏标注

台词的停顿、语气、重音决定了情感传达的准确度。在 prompt 中描述台词节奏，可以让 AI 更好地理解角色情绪状态。

---

## 一、停顿标记

| 符号 | 停顿时长 | 说明 | 情绪效果 | 示例 |
|------|---------|------|---------|------|
| ，逗号 | 0.5s | 短停顿 | 日常呼吸、轻微犹豫 | "我……不知道" |
| 、顿号 | 0.2s | 极短停顿 | 列举、急促 | "刀、剑、枪，都有" |
| ——破折号 | 1-2s | 拉长/情绪转折 | 思绪延伸、情绪转变 | "我本来以为——算了" |
| ……省略号 | 2s+ | 长沉默 | 无言、哽咽、思绪万千 | "你走吧……" |
| 。句号 | 0.8s | 标准句末停顿 | 确定、结束 | "就这样吧。" |
| ！叹号 | 0.3s（短促） | 急促停顿 | 激动、愤怒、震惊 | "你疯了！" |
| ？问号 | 0.5-1s | 上扬后停顿 | 疑问、不确定 | "真的吗？" |
| ｜斜杠 | 1.5s | 刻意停顿 | 思考、选择词语 | "我觉得｜你不该去" |
| （空行） | 3s+ | 场景级沉默 | 重大沉默、无言以对 | 两人对视，无人开口 |

| ST2 | 情感重音 | **加粗+叹号** | 情感爆发式强调 | 愤怒、激动 | "我**不**走！" |

| ST5 | 颤音 | ~~波浪重复~~ | 声音颤抖 | 恐惧、激动、哭泣 | "~~求你了~~" |
| ST6 | 破音 | [破音] | 声音破裂 | 情绪失控、崩溃 | "[破音]为什么" |
| ST7 | 拖长音 | 拉——长—— | 元音延长 | 强调、讽刺、痛苦 | "你——好——狠——" |
| ST8 | 断续音 | 断…续… | 说话断断续续 | 虚弱、濒死、哭泣 | "我…可能…不行了…" |

## 三、沉默的力量（留白标注）

沉默是台词中最有力的部分。在 prompt 中明确标注沉默时长和沉默类型：

| 编号 | 沉默类型 | 时长 | 适用场景 | 情绪效果 |
|------|---------|------|---------|---------|
| SL1 | 呼吸沉默 | 1s | 对话间隙 | 自然呼吸、思考 |
| SL2 | 犹豫沉默 | 2s | 难以开口前 | 不确定、害怕 |
| SL3 | 震惊沉默 | 3s | 听到意外消息后 | 无法消化、呆滞 |
| SL4 | 对峙沉默 | 5s+ | 两人对视不说话 | 张力拉满、无声胜有声 |
| SL5 | 悲痛沉默 | 10s+ | 重大损失后 | 无言的痛 |
| SL6 | 释怀沉默 | 5s | 和解后 | 无需多言 |
| SL7 | 濒死沉默 | 渐长至无限 | 角色生命最后 | 最后的安静 |

## 四、在 Prompt 中使用

```
台词节奏：[角色名] 说："[台词内容，含停顿/重音/语气标记]"。[沉默类型]。[语速]。[音量]。
```

例：
```
台词节奏：
男主说："我**不**走｜……[对峙沉默 5s]｜~~求你别赶我走~~"。[VL4 提高→VL2 低声→ST5 颤音]。[SP6 渐快→SP2 慢]。
女主（沉默 3s [震惊沉默]）："……为什么。" [VL2 低声]。[SP2 慢]。
```

### 简写格式（Prompt 内嵌）

```
台词："我**不**走｜……~~求你~~" [对峙沉默→颤音/低声/渐慢]
```
