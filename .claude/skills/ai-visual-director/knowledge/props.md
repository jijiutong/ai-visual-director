## 六、道具描述模板

在 prompt 中描述道具时使用：

```
道具：[编号. 道具名称]。[材质]，[颜色]，[尺寸描述]，[纹饰/雕刻描述]，[光效/能量效果]，[持有方式/状态]。
```

例：
```
道具：PR13. 玉佩。和田玉质地，温润青白色，巴掌大小，雕刻云纹龙形图案，系深绿色丝绦。情绪波动时发出微弱青光光芒。悬挂于腰间。
```
