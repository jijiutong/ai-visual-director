# 灯光与色彩参考（40+ 方案）

## 一、灯光类型

### 主光（Key Light）
- 场景中最强的光源，决定整体光影方向
- 描述方式：[光源类型] + [方向] + [色温]
- 例：右侧窗户外射入的暖黄色阳光、头顶冷白色荧光灯

### 补光（Fill Light）
- 弱化主光产生的阴影，降低对比
- 描述方式：[光源类型] + [强度描述]
- 例：白色墙壁反射的柔和补光、地面水洼反射的微弱补光

### 轮廓光/逆光（Rim Light / Backlight）
- 从主体后方照射，勾勒边缘轮廓
- 效果：分离主体与背景、增加层次、营造氛围
- 例：强烈的金色逆光勾勒人物剪影边缘

### 环境光（Ambient Light）
- 场景中弥漫的整体光，没有明确方向
- 例：阴天散射光、雾气中的漫射光

### 特效光（Effect Light）
- 特殊效果光源
- 类型：霓虹、全息投影、火焰、电弧、生物发光、魔法光效

## 四、配色方案

| 方案 | 说明 | 示例 |
|------|------|------|
| 互补色 | 色环相对两色 | 蓝+橙（电影最常用） |
| 冷暖对比 | 冷色调背景+暖色调主体 | 冷灰街道+暖色人物 |
| 单色系 | 同一色相不同明度 | 不同层次的蓝 |
| 三分色 | 色环上120度间隔三色 | 红+黄+蓝 |
| 相邻色 | 色环上相邻色 | 蓝+绿+青 |
| 黑金 | 黑色底+金色点缀 | 豪华、权力、史诗 |
| 分裂互补 | 主色+互补色两侧相邻色 | 蓝+橙红+橙黄 |
| 四分配色 | 矩形四色 | 红+绿+蓝+橙 |

## 在 Prompt 中使用

```
色彩与光影：主色调 [色1] + [色2] + [点缀色]，[配色方案名称]，高级电影调色，暗部有层次，高光不过曝，皮肤/金属/布料/雨水/火焰/能量特效都有真实材质细节。

灯光方案：[编号. 灯光方案名称] — [具体描述]，色温 [色温描述]。
```

例：
```
色彩与光影：主色调 深蓝灰 #2c3e50 + 暖米 #f5e6d3 + 珊瑚红 #c17767，冷暖对比，高级电影调色。

灯光方案：LT38. 霓虹+雨反射 — 霓虹侧光（冷蓝）+ 台灯暖橙补光 + 雨面反射光斑 + 大面积暗部，色温 3000K 暖 + 6500K 冷混合。
```
