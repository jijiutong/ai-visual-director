# 多画幅自适应

自动为不同平台生成画幅变体，一套故事板适配所有发布渠道。

## 画幅规格库

| 画幅 | 比例 | 适用平台 | 版式调整 |
|------|------|---------|---------|
| 超宽银幕 | 21:9 | IMAX/电影宽屏 | 横向扩展两侧环境，核心居中 |
| 电影宽银幕 | 2.39:1 | 电影/预告片 | 横向展开，角色并排布局 |
| 横版高清 | 16:9 | YouTube/B站/电视 | 标准全案板布局 |
| 横版标清 | 4:3 | 老电视/复古 | 压缩两侧，聚焦中心 |
| 方形 | 1:1 | Instagram/朋友圈 | 对称构图，核心居中 |
| 竖版海报 | 2:3 | 电影海报/豆瓣 | 纵向排列，标题在顶部 |
| 竖版短视频 | 9:16 | 小红书/抖音/TikTok | 纵向重构，模块上下排列 |
| 竖版故事 | 9:20 | Instagram Stories | 超纵向，单列流式布局 |
| 超宽超短 | 32:9 | 双屏展示/曲面屏 | 超宽展开，信息密度加倍 |

## 画幅适配规则

### 横版→竖版转换

```
横版 16:9 布局：
┌────────────────────────┐
│ 顶部栏 │ 角色栏 │ 场景 │
├────────────────────────┤
│         分镜表          │
├──────────┬─────────────┤
│ 镜头轨迹  │ 技术参数    │
└──────────┴─────────────┘

转换为竖版 9:16：
┌────────┐
│ 顶部栏  │
├────────┤
│ 核心场景│
├────────┤
│ 角色栏  │
────────┤
│ 分镜表  │
│ (纵向)  │
├────────
│ 技术参数│
└────────┘
```

### 横版→方形转换

```
横版 16:9 → 方形 1:1：
- 裁切两侧多余信息
- 核心场景保持完整
- 角色栏和分镜表精简
- 技术参数移至底部小栏
```

## 用户指令

| 用户指令 | 行为 |
|---------|------|
| "适配小红书" | 自动转 9:16 竖版 |
| "适配抖音" | 自动转 9:16 竖版 + 漫画分镜模式 |
| "适配朋友圈" | 自动转 1:1 方形 |
| "适配所有平台" | 生成 16:9 + 9:16 + 1:1 + 2:3 四个版本 |
| "21:9" | 超宽银幕版 |

## 在 Prompt 中使用

追加画幅适配标注：

```
画幅适配：[目标画幅]。版式调整：[调整说明]。核心元素位置：[位置描述]。
```
