# 输出格式库

10 种输出格式，每种包含：格式说明、适用场景、核心模块、画幅比例、模板文件。

---

## 1. 全案板（完整）

- **模板**：`templates/full-board.md`
- **适用**：完整项目提案、前期视觉开发、导演参考板
- **核心模块**：项目总览 + 角色三视图 + 核心场景图 + 7/13镜分镜表 + 镜头轨迹 + 技术参数 + 灯光色卡 + 材质特效
- **画幅**：16:9 横版（默认），可改 9:16
- **信息密度**：最高

## 2. 快速故事板

- **模板**：`templates/quick-board.md`
- **适用**：快速验证想法、单场景分镜、临时参考
- **核心模块**：片名 + 核心场景 + 7镜分镜 + 简技术参数
- **画幅**：16:9 横版
- **信息密度**：中等

## 3. 角色设定卡

- **模板**：`templates/character-sheet.md`
- **适用**：角色设计确认、服装道具参考、演员casting参考
- **核心模块**：正面/侧面/背面三视图 + 面部特写 + 道具/能力特写 + 服装细节
- **画幅**：4:3 横版 或 9:16 竖版
- **信息密度**：中低

## 4. 场景概念卡

- **模板**：`templates/scene-card.md`
- **适用**：场景美术方向确认、环境氛围参考、制片选址参考
- **核心模块**：主场景概念图 + 环境细节 + 灯光方向 + 材质参考 + 天气/时间变化
- **画幅**：16:9 横版
- **信息密度**：中低

## 5. 海报

- **模板**：`templates/poster.md`
- **适用**：宣传海报、预告片帧、封面图
- **核心模块**：主视觉 + 片名排版 + 角色剪影/特写 + 标语 + 底部信息
- **画幅**：2:3 竖版（电影海报）或 16:9 横版（横幅）
- **信息密度**：低

## 6. 漫画分镜页

- **模板**：`templates/manga-page.md`
- **适用**：漫画改编、条漫/页漫分镜、叙事节奏验证
- **核心模块**：5-9格漫画布局 + 对话气泡 + 旁白框 + 效果线 + 翻页节奏
- **画幅**：按平台定（条漫 9:16，页漫 3:4）
- **信息密度**：高

## 7. 情绪板

- **模板**：`templates/mood-board.md`
- **适用**：美术方向确认、色彩基调参考、氛围验证
- **核心模块**：色彩色卡 + 材质贴图参考 + 灯光氛围小图 + 字体参考 + 纹理参考
- **画幅**：16:9 横版
- **信息密度**：中

## 8. 四格漫画

- **模板**：`templates/manga-page.md`（四格模式）
- **适用**：搞笑片段、关键情节提炼、社交媒体传播
- **核心模块**：4格（起承转合）+ 对话气泡 + 表情夸张 + 结尾 punchline
- **画幅**：4:5 竖版（社交媒体）
- **信息密度**：低

## 9. 三视图设定卡

- **模板**：`templates/character-sheet.md`（三视图模式）
- **适用**：3D建模参考、游戏角色设计、手办原型
- **核心模块**：正面/侧面/背面严格对齐三视图 + 尺寸标注 + 材质标注 + 颜色编号
- **画幅**：16:9 横版
- **信息密度**：低（专注几何信息）

## 10. 关键帧序列

- **模板**：`templates/quick-board.md`（关键帧模式）
- **适用**：动作场面设计、特效镜头参考、动画关键帧
- **核心模块**：5-9个关键动作帧 + 动作轨迹线 + 特效标注 + 时间标注
- **画幅**：16:9 横版或 2.39:1 超宽银幕
- **信息密度**：中
