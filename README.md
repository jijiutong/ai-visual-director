# AI Story Board Skill

AI 电视剧 / 漫剧 / 短剧故事板生成技能。将任意故事、小说片段、漫画设定转化为 GPT Image 2 可直接使用的电影级分镜提示词。

## 功能

- 从故事中自动提取：片名、类型、角色设定、核心冲突、情绪曲线
- 5 种视觉风格可选：黑金动作 / 科幻机甲 / 东方玄幻 / 悬疑惊悚 / 都市情绪
- 生成包含：项目总览、角色三视图、核心场景图、7/13 镜分镜表、镜头轨迹、摄影参数、灯光色卡
- 支持竖版（9:16 小红书）和横版（16:9）两种画幅

## 使用方式

```
/storyboard 《给一段故事内容》
```

或直接描述需求：

```
帮我把这个小说片段生成故事板：[故事内容]
```

## 输出类型

| 风格 | 适用题材 |
|------|----------|
| 黑金动作分镜板 | 武侠、复仇、谍战、犯罪、战争、格斗 |
| 科幻机甲全案板 | 机甲、赛博、外星文明、未来战争、AI 对抗 |
| 东方玄幻开发板 | 修仙、古风、神魔、异世界、史诗冒险 |
| 悬疑惊悚导演板 | 密室、犯罪、恐怖、心理惊悚 |
| 都市情绪电影板 | 现实题材、爱情、职场、家庭、人生转折 |

## 自定义指令

- `更燃` — 强化动作分镜、光效、爆点、镜头推进
- `更贵` — 强化专业制片板、摄影参数、材质、色卡、版式秩序
- `小红书竖版` — 改为 9:16 画幅，保持电影项目板结构
