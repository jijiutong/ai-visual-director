<p align="center">

> 从故事到视觉，一条完整生产管线

AI 视觉导演 — 将任意故事、小说、剧本转化为电影级视觉开发方案。30 项核心能力，34 个参考文件，覆盖从**输入→分析→生成→评分→检查→修改→导出→视频→行业格式**的完整制作流程。

支持 GPT Image 2 / Midjourney v6 / Stable Diffusion XL / Sora / Runway / 可灵。

## 快速开始

### 默认模式（先推荐，你确认后生成）

```
你：[粘贴你的故事片段]
AI：【智能推荐】根据你的故事，推荐以下组合...
     推荐 1（最匹配）：风格 X + 格式 Y — 理由
     推荐 2（备选）：  风格 A + 格式 B — 理由
     推荐 3（不同视角）：风格 C + 格式 D — 理由
你：1          ← 回复编号确认，或说"全来"一次出4个
AI：```[完整 prompt]```
```

### 快捷模式（跳过推荐，直接输出）

| 指令 | 效果 |
|------|------|
| `一键生成` | 全流程自动完成，不展示选项 |
| `一键全平台` | 全流程 + GPT/MJ/SD 三平台输出 |
| `一键多版本` | 全流程 + A/B/C 三版本对比 |
| `一键全来` | 3风格×4格式×3平台=36个prompt |
| `全来` | 4个最相关格式 prompt |

## 完整能力清单

### 视觉能力（5项）

| 能力 | 说明 |
|------|------|
| **32 种视觉风格** | 黑金动作→中国水墨，含 Disney/Pixar/吉卜力/王家卫/张艺谋/Wes Anderson/Tim Burton 等导演风格 |
| **风格融合引擎** | "王家卫+吉卜力" 自动融合，60%主+40%副，23种高频组合预设 |
| **Mood 滑块** | 更燃/更虐/更甜/更丧/更暧昧/更恐怖/更史诗/更治愈/更紧张/更梦幻 |
| **色彩叙事引擎** | 颜色随情绪推进变化，5种叙事模型，色彩心理映射 |
| **风格迁移** | 跨风格换肤但保持角色DNA不变 |

### 角色能力（4项）

| 能力 | 说明 |
|------|------|
| **15 种角色关系** | 宿敌变恋人/双向暗恋/师徒/年上/先婚后爱/救赎/强制/战友等，每种×4种风格加成 |
| **角色一致性 DNA** | 13 字段 DNA 锚定，跨镜头服装/发型/面部完全一致 |
| **续集/系列支持** | "第2集/继续" 自动延续DNA/色彩/场景，状态变化追踪 |
| **连续性检查** | 自动检查时间线/空间/角色/情绪连贯性，标出断裂点 |

### 叙事能力（4项）

| 能力 | 说明 |
|------|------|
| **三幕结构分析** | 自动拆解建制(25%)/对抗(50%)/解决(25%)，每幕匹配镜头/色彩/节奏 |
| **情绪曲线驱动** | 平静→对峙→爆发→余韵，动态调整景别/运镜/灯光/色彩 |
| **分镜节奏引擎** | 快切/标准/慢/渐进/喜剧，5种节奏模式自动匹配 |
| **音乐音效参考** | 每幕/每镜自动推荐音乐 mood + 音效，6种类型音乐映射 |

### 输出能力（5项）

| 能力 | 说明 |
|------|------|
| **10 种输出格式** | 全案板/快速故事板/角色卡/场景卡/海报/漫画分镜/情绪板/四格/三视图/关键帧 |
| **多版本 A/B/C** | 同一故事 3 种不同视觉诠释对比 |
| **多画幅自适应** | 16:9/9:16/1:1/2:3/21:9/32:9，一键切换 |
| **行业格式导出** | Storyboard Pro / FrameForge / Celtx 标准格式 |
| **批量故事处理** | 整章小说自动拆场景，批量生成对应故事板 |

### 工作流能力（6项）

| 能力 | 说明 |
|------|------|
| **一键工作流** | "一键生成/全平台/多版本/全来" 全流程自动 |

## 安装

### 方式一：npx skills 安装（推荐）

```bash
# 全局安装 — 所有项目可用
npx skills add jijiutong/ai-visual-director -g -y

# 项目级安装 — 仅当前项目可用
npx skills add jijiutong/ai-visual-director
```

安装后 Claude Code 自动发现，在对话中粘贴故事即可使用。

### 方式二：手动安装

```bash
git clone git@github.com:jijiutong/ai-visual-director.git
```

将 `SKILL.md` 及 `references/`、`templates/` 放入 `~/.claude/skills/ai-story-board/`（全局）或 `<project>/.claude/skills/ai-story-board/`（项目级）。

### 方式三：直接用

不做任何安装，在 Claude Code 中直接说：
```
帮我做这个故事板：[粘贴你的故事片段]
```

AI 会根据 SKILL.md 的指令自动生成 prompt。
