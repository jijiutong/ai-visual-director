# 连续性状态快照

每段视频生成完毕后，保存完整状态快照。下一段必须从此快照继承起始状态。

---

## 快照结构

```markdown
【段 N 连续性状态快照】

段编号：N
视频时长：10s
分镜数：5镜
尾帧：第5镜结束

── 角色状态 ──

主角：
  姓名：[角色名]
  位置：[空间坐标描述]
  动作末态：[镜头结束时的动作]
  视线方向：[看哪里]
  情绪状态：[从X过渡到Y]
  服装：[穿着描述]
  道具：[手持/佩戴道具及状态]
  身体状态：[受伤/疲惫/完好]

对手（如有）：
  姓名：[角色名]
  位置：[相对主角位置]
  动作末态：[动作]
  情绪状态：[情绪]
  道具：[道具状态]

── 空间状态 ──

场景：[场景名]
光线方向：[主光源从哪来]

色调：[画面主色调]

── 运动状态 ──

── 跨段衔接标记 ──

建议衔接技法：[编号+名称]

```

## 跨段衔接技法选择

从 36 种技法中，根据段间关系自动匹配：

| 段间关系 | 推荐技法 |
|---------|---------|
| 连续动作（挥刀→砍下） | 动作匹配剪 |
| 场景切换（室内→室外） | J-Cut 声音先行 |
| 时间跳跃（→三小时后） | 时间跳跃 + 色彩洗入 |
| 视角切换（看→被看） | 视线匹配剪 |
| 情绪反差（静→闹） | 反差撞击 |
| 回忆闪回 | 记忆闪回 / 焦点过渡 |
| 梦境进入 | 梦境过渡 + 色彩抽离 |
| 平行叙事 | 平行交叉剪 |
| 高潮结束→尾声 | 情绪持留 |

详细技法见 SKILL.md 跨段衔接章节。

## 快照使用流程

```
段1完成 → 保存快照 → 
  段2开始时:
    1. 读取段1快照
    2. character-dna 继承：服装/发型/道具状态不变
    3. scene 继承：光线方向/色温/色调不变（除非声明场景切换）
    4. 位置继承：角色从段1尾帧位置开始
    5. 情绪继承：从段1末态情绪开始，本段内发展
    6. 衔接技法应用：尾帧→首帧锚定
    7. video-director 收到快照后生成段2分镜
```

## 状态冲突检测

段2开始前检查：

| 检查项 | 冲突 | 处理 |
|--------|------|------|
| 场景是否切换？ | 是 → 光线/空间可以不同 | 声明新场景，更新空间快照 |
| 是否时间跳跃？ | 是 → 允许情绪/位置大幅变化 | 声明跳跃量（几分钟/几小时/几天） |
| 角色是否换装？ | 是 → 必须有换装画面或已声明 | 在段2首镜描述服装变化 |
| 是否有新角色加入？ | 是 → 补充新角色DNA | 追加到角色状态 |
| 道具是否消失？ | 检测道具在段1存在但段2缺失 | 补道具状态说明或删除 |
