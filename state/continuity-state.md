# 连续性状态规则

每个镜头结束时必须记录状态。下一镜必须继承。杜绝"瞬移"。

---

## 状态记录字段（每镜必须输出）

| 字段 | 说明 | 示例 |
|------|------|------|
| 角色位置 | 主角在空间中的位置 | 剑冢中央、石碑前方3米 |
| 动作末态 | 镜头结束时主角的动作 | 剑已拔出、正对镜头站立 |
| 视线方向 | 主角在看向哪里 | 看向右上方的阳光裂口 |
| 光线方向 | 主光源从哪个方向来 | 顶光从裂缝射入、左暖右冷 |
| 画面调性 | 此镜结束时的色彩氛围 | 冷暖对峙、暗部深压 |
| 道具状态 | 关键道具的状态 | 剑身完整、发微弱蓝光 |
| 情绪状态 | 角色此刻的情绪 | 从愤怒过渡到决绝 |
| 空间关系 | 角色之间的物理位置关系 | A在左距B3米，B在右背对A |

---

## 继承规则

```
镜N 结束状态 = 镜N+1 起始状态

除非：
  - 明确的场景切换（新空间需声明）
  - 明确的时间跳跃（"三小时后"）
  - 明确的转场（硬切到全新画面）
```

---

## 继承检查表

| 检查项 | 如果断裂 | 修复 |
|--------|---------|------|
| 角色位置跳跃 | 上镜在室内→下镜在室外，中间无出门 | 补过渡态 |
| 服装变化 | 上镜穿黑→下镜穿白，无换装画面 | 修正为一致 |
| 道具消失/出现 | 上镜拿剑→下镜空手 | 补道具状态说明 |
| 光线突变 | 上镜黄昏→下镜正午 | 补时间过渡 |
| 情绪跳跃 | 上镜大哭→下镜狂笑（非喜剧） | 补情绪过渡 |
| 对手位置跳跃 | 上镜A在B对面→下镜A在B背后 | 补走位说明 |

## 跨段衔接状态快照

当视频拆段时：

```
段1尾帧状态快照：
  角色位置：[位置]
  动作末态：[动作]
  视线方向：[方向]
  光线：[光源方向+色温]
  道具：[状态]
  情绪：[状态]
  ───────────────────
  ↓ 衔接技法：[编号]
  ───────────────────
段2首帧：
  继承：角色位置/动作末态/光线/道具 保持不变
  变化：情绪从[段1末态]自然过渡到[段2初态]
```

## 联动

← 每镜状态由 `video-director`（位置/情绪/道具） + `motion-physics`（运动/方向） 共同记录
→ **每镜实例数据写入 `state/shot-state.md`** 的 end_state 8 字段（与本文件字段定义对齐）
→ 连续性断裂由 `prompt-scorer` 调用 `rules/continuity-check.md` 检测
→ 修复由 `auto-repair` 执行
