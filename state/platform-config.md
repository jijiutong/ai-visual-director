# Platform Config — 平台统一配置

所有引擎读取此文件获取平台参数。一处修改，全链生效。**禁止在引擎中硬编码平台限制。**

---

## 一、默认平台

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `platform.image.default` | GPT Image 2 | 角色卡/场景图/分镜图/海报等静态图默认出图平台 |
| `platform.video.default` | Seedance | 视频 prompt 默认目标平台 |

---

---

## 三、自动拆分规则

```
输入时长 > 平台最大单段时长 → 自动拆分

拆分算法：
  段数 = ceil(输入时长 / 平台最大单段时长)
  每段镜数 = ceil(总镜数 / 段数)

示例（60s / 15镜 → Seedance 15s上限）：
  段数 = ceil(60/15) = 4段
  每段 = 4镜（第4段3镜）
  段1 0-15s  | 段2 15-30s | 段3 30-45s | 段4 45-60s
```

## 四、资产按段分配

| 资产类型 | 分配策略 |
|---------|---------|
| 场景图 | 跨段复用（同一场景）或按段独立（不同场景） |
| 角色卡 | 跨段复用（角色 DNA 不变） |
| **全案分镜图** | **每段 1 张**（7镜以内一张图清晰可读，>7镜拆多张） |
| 高潮关键帧 | 每段 1 张（每段有自己的高潮镜） |
| 首帧 | 段1首帧 |
| 尾帧 | 最后一段尾帧 |
| 段间接力 | 段N尾帧 = 段N+1首帧（自动锚定） |

> **禁止**：60s 视频只出 1 张全案图塞 15 镜——看不清、Seedance 也用不了。

## 五、出图平台参数

### 图像平台语言支持

| 平台 | 中文 | 英文 | 备注 |
|------|------|------|------|
| GPT Image 2 | ✅ 默认 | ✅ | 原生中文，中文 prompt 更精准 |
| Midjourney v6/v7 | ❌ | ✅ 必须 | 只认英文，中文会报错 |
| SDXL / SD3 | ❌ | ✅ 推荐 | 英文效果最好，社区模型英文训练 |
| DALL-E 3 | ✅ | ✅ | 原生中文 |
| Nano Banana (Gemini) | ✅ | ✅ | 中文原生，14张参考图 |
| Flux (Replicate) | ⚠️ 部分 | ✅ 推荐 | 英文为主 |
| Ideogram | ❌ | ✅ | 英文 |
| 通义万相 | ✅ 默认 | ✅ | 中文原生 |
| Recraft | ❌ | ✅ | 英文 |
| ComfyUI | ⚠️ 取决于模型 | ✅ | 取决于加载的 checkpoint |

> **规则**：引擎生成 prompt 时，语言必须匹配平台支持。中文平台优先用中文，英文平台必须用英文。不可跨语言生成。

### 默认平台参数（GPT Image 2）

| 配置项 | 默认值 |
|--------|--------|
| 画幅 | 16:9 |
| 风格 | 跟随 video-director 的 VS 决策 |
| 角色卡版式 | LS13（科幻）/ LS11（通用） |
| 场景图版式 | LS8 场景全能参考板 |
| 分镜图版式 | LS6 横幅长镜头时间轴 |
| 海报版式 | 跟随 video-director 的 LS 决策 |
| 输出语言 | 中文（可切换英文） |
