| 操作 | 说明 |
|------|------|
| `lock` | 用户确认锁定 |
| `commit` | 用户确认写回主状态 |
| `unlock` | 用户解锁 |
| `auto-repair` | 自动修复（仅格式，不碰 locked 字段） |
| `rollback` | 回滚到上一版本（需确认） |
