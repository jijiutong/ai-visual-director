# Video Workflow — AI 视频全流程

> 从故事板到 AI 视频成片的标准工作流文档。
> 视频 Prompt 工作流已测试。视频 API 出片未验证。

## 概览

AI Visual Director 的视频工作流将故事板转化为 AI 视频工具可直接使用的 Prompt 和参考图方案。

### 支持平台

| 平台 | 输入方式 | 特点 | API 状态 |
|------|---------|------|---------|
| Seedance | 多图 + Prompt | 中文原生，音频输出 | 未验证 |
| Runway Gen-4 | 图片 + Prompt | SDK 完善 | 未验证 |
| 可灵 (Kling) | 图片 + Prompt | 4K 输出，角色 ID | 未验证 |
| Luma Dream Machine | 首尾帧 + Prompt | 丝滑过渡 | 未验证 |
| Pika | Prompt + 图片 | 短视频快速验证 | 未验证 |

## 5 步推荐流程

```
前置准备 → 全景锁定 → 分镜图 → Prompt → 视频
```

### Step 1: 前置准备（2-3 张参考图）

```
角色卡 → 场景参考图 → 道具细节卡

出图量：2-3 张  |  耗时：~1min
```

定义世界观基础，奠定角色+场景+道具的视觉锚点。

### Step 2: 全景图锁定（1 张）

```
720° 全景图 → 6 面立方体拼接

出图量：6 张 → 拼接为 1 张  |  耗时：~90s
```

覆盖所有拍摄角度，后续每镜从全景空间"取景"。

### Step 3: 生成分镜图（3-4 张）

```
合并帧 → 按动作阶段分组

出图量：3-4 张  |  耗时：~2min
```

将 N 帧故事板合并为 3-4 张画面锚点，每张覆盖 2-3 帧。

### Step 4: 生成视频 Prompt

```
压缩模式 ≤1500 字 → 直接喂 AI 视频工具

耗时：~30s
```

### Step 5: 输入视频工具

```
Seedance: prompt + 角色卡 + 全景图 + 3-4 张分镜图

耗时：1-3min
```

## 命令体系

### `/video 转视频`

故事板 → 推荐流程 → 视频方案。

输出：逐镜运动方向 + 转场 + 视频负面提示词。

### `出视频 prompt`（压缩模式）

输出一条 ≤1500 字的完整视频 prompt，结构：

```
【视频基础信息】时长/画幅/风格
【分镜序列】每帧单行：画面描述(≤15字) | 景别|运镜|色彩|灯光|转场
【约束层】场景@图0 / 角色@图1 / 分镜图@图2-5
【风格收尾】色彩弧线 + 8K + 视频负面词
```

### `详细模式` / `展开 prompt`

每帧展开为完整段落（3000+ 字）：

- 景别 + 画面内容（叙事描述）
- 运镜方式 + 画面构图
- 镜头切换（转场）
- 人物台词
- 音效设计（环境音+拟音+音乐 mood）

适合 Runway / Sora / 可灵等文生视频工具。

### `切分各帧` / `逐帧接力`

N 帧 → N 条独立 prompt，每帧 16:9 单独出图。

帧间参考链：帧 N 作参考图出帧 N+1，链式传递。前一帧的关键状态（角色位置/动作末态/光线方向）在下一帧开头延续。

### `生成视频分镜图` / `合并帧`

N 帧按动作阶段合并为 3-4 张分镜图：

- 建置阶段（帧1-2）
- 冲突阶段（帧3-4）
- 高潮阶段（帧5-6）
- 收尾阶段（帧7，如有）

每张覆盖 2-3 帧，作为画面锚点防止 AI 视频工具偏离故事板。

### `继续下一段` / `续写下一段`

AI 自动续写下一段剧情 + 匹配衔接技法。

**硬锁定**（不可变）：角色 DNA 不可变字段、核心冲突、世界观规则、时间线方向、角色关系。

**软锁定**（可演进但须说明）：服装、情绪状态、场景切换、道具武器、光线天气。

**叙事逻辑检查**：因果链完整、动机连贯、信息不跳级、数字一致、称呼一致。

**衔接技法**：自动匹配 36 种（视觉匹配 10 + 叙事节奏 10 + 声音驱动 6 + 特殊技法 10）。尾帧→首帧锚定。

**手动指定**：
- `用 [技法名] 衔接，继续下一段`
- `下一段往 [方向] 发展，用 [技法] 衔接`

### `检查视频` / `视频检查`

10 项生成后检查清单：

| # | 检查项 | 问题归属 |
|---|--------|---------|
| 1 | 场景一致性 | 缺场景参考图 |
| 2 | 角色一致性 | 缺角色卡 |
| 3 | 光照方向 | 缺光照材质图 |
| 4 | 道具形制 | 缺道具细节卡 |
| 5 | 动作连贯 | 缺首尾帧 |
| 6 | 画面锚点 | 缺视频分镜图 |
| 7 | 转场流畅 | Prompt 缺过渡方式 |
| 8 | 字幕正确 | Prompt 字幕约束不完整 |
| 9 | 材质质感 | 缺光照材质图 |
| 10 | 总体打分 | 低于 60 → 补全锚点重新生成 |

## 约束规则

### 时长与镜数

- 15s → 7 镜（短篇标准）
- 12s → 6 镜（微短篇）
- 30s → 10 镜（中篇）
- 视频 prompt 中总时长必须与故事板一致，不可漂移

### 视频 Prompt 字数限制

| 模式 | 限制 | 适用平台 |
|------|------|---------|
| 压缩模式 | ≤1500 字 | Seedance |
| 详细模式 | 3000+ 字 | Runway / Sora / 可灵 |

### 视频专属负面提示词

```
no flickering, no frame flashing, no sudden brightness jumps, no color shifts,
no morphing, no body distortion, no face melting, no limb warping, no extra fingers,
no floating, no sliding feet, no hovering, no broken physics,
no background shifting, no environment pop-in, no sky color change,
no disappearing props, no weapon shape change, no accessory deformation,
no clothing texture change, no fabric stiffness, no costume color drift,
no broken continuity between frames, no jump cuts, no ghosting artifacts,
no text garbling, no watermark, no logo, no subtitles mismatch
```

### 运动描述

每镜必须包含至少一种运动类型：

| 类型 | 示例 |
|------|------|
| 相机运动 | 慢推/拉远/环绕/升降/手持 |
| 主体运动 | 行走/转身/挥剑/指向/流泪/微笑 |
| 环境运动 | 雨落/风起/雾散/火焰/光变 |

## 跨段衔接（长视频 >30s）

```
段1 → [截尾帧1-3] → 段2首帧锚定 → 段2 → [截尾帧] → 段3 → ...
```

### 36 种衔接技法

| 类别 | 技法 |
|------|------|
| 视觉匹配 10 | 图形匹配剪 / 运动矢量匹配 / 视线匹配剪 / 动作匹配剪 / 物件桥接剪 / 光影衔接 / 色彩洗入 / 焦点过渡 / 遮挡擦除 / 晃动过渡 |
| 叙事节奏 10 | 动作高潮断 / 情绪持留 / 悬念切断 / 反差撞击 / 喜剧节拍 / 时间跳跃 / 梦境过渡 / 记忆闪回 / 平行交叉剪 / 倒计时加速 |
| 声音驱动 6 | J-Cut / L-Cut / 声音撞击 / 音乐短语完成 / 环境音渐变 / 静默过渡 |
| 特殊技法 10 | 冷冻帧过渡 / 速度渐变 / 倒放过渡 / 裂像转场 / 画幅变化 / 分屏汇合 / 色彩抽离 / 负片过渡 / 遮罩转场 / 匹配溶解 |

### AI 自动续写规则

1. 读取当前段叙事状态
2. 基于叙事结构自动推进（建置→对抗→解决）
3. 延续全部硬锁定 + 软锁定
4. 保持输出格式一致
5. 用户确认后生成参考图/prompt → 进入衔接流程
