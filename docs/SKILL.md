# AI Visual Director Skill 正式文档

> 版本：v0.5.0 · 14 条核心规则 · 6 个主命令
> AI 故事板/角色/场景/视频 Prompt 生成技能

---

## 1. Skill 定位

AI Visual Director 将任意故事内容（小说/剧本/场景描述/一句话创意）转化为专业级视觉开发方案。

**核心能力**：故事板生成 · 角色设计 · 场景概念 · 视频 Prompt · 风格迁移 · 多平台适配

**适用场景**：漫剧/短剧/电视剧/电影前期视觉开发、AI 视频制作、概念设计

---

## 2. 输入识别逻辑

### 信息来源
1. 用户直接粘贴故事文本 → 自动提取片名/类型/角色/场景/冲突/情绪
2. 用户说「一键生成」→ 跳过推荐，直接输出 7+1 模块完整故事板（含技术参数总表）
3. 用户指定命令 → 路由到对应 `/command`

### 缺字段补全
- 无角色名 → 起电影感名字
- 无明确类型 → 按冲突强度推断
- 无明确关系 → 按互动张力推断
- 无明确结构 → 按类型匹配三幕

## 5. 角色一致性规则

### 5.1 角色 DNA（20 字段）
姓名 · 性别年龄 · 身高体型 · 脸型 · 眼型瞳色 · 眉型 · 鼻型 · 嘴型唇色 · 发型发色 · 肤色 · 服装 · 配饰 · 武器/道具 · 声音 · 体态站姿 · 习惯动作 · 气味 · 情绪底色 · 可变字段 · 不变量标注

### 5.2 剧情身份
- 必含身份标注（师父/徒弟/对手/旁观者）
- 高位身份词冲突 → 输出【身份解释】

### 5.3 角色卡出图模式
- 默认：无文字标注/红色标注 → 纯视觉传达
- 导演标注版：允许标注（用户明确触发）

### 5.4 8 种一致性锁定方法
角色设定卡（6模块）/ 三视图 / 面部5角度 / 12表情 / 服装武器细节 / 14图参考 / IP-Adapter / DNA文字锚定

## 6. 场景一致性规则

### 6.1 7 种锁场景方法
俯拍图 / 4宫格 / 9宫格 / 全能参考 / 720°全景 / 环绕截图 / 文字场景锁定

### 6.2 场景卡出图模式
- 默认：无箭头/编号/红框/文字 → 通过光影构图传达
- 导演标注版：允许标注元素（用户明确触发）

### 6.3 场景说明书字段
空间尺寸 · 主光源（方向+类型+色温） · 辅助光源 · 地面材质 · 天花板 · 色调 · 氛围

## 7. 视频工作流规则

### 推荐流程
```
角色卡+场景参考图+道具卡 → 全景图 → 视频分镜图(3-4张) → 视频Prompt → AI视频工具
```

### 跨段衔接
段1尾帧(1-3帧) = 段2首帧锚定 → 平滑接力链
36 种衔接技法：视觉匹配 10 + 叙事节奏 10 + 声音驱动 6 + 特殊技法 10

### 视频 Prompt 限制
- 压缩模式 ≤1500 字（Seedance）
- 详细模式 3000+ 字（Runway/Sora/可灵）
- 视频专属负面词自动附加

## 8. 风格迁移规则

### 锁定不变
角色 DNA · 剧情事实 · 关键动作 · 镜头数量 · 总时长

### 可变更
色彩/光影/构图/运镜/节奏/表演方式

### 古代场景限制
使用「霓虹感调色/胶片颗粒/水面反光」模拟现代感
禁止「霓虹灯光/LED/街灯/城市灯光」等现代物理光源词

### 迁移声明
每次迁移开头输出：
```
🔒 锁定不变：...
🎨 变更维度：...
```

## 10. 禁止规则和出图安全

### Prompt 禁可读文字
绝对禁止要求模型生成中文/英文/数字文字
发现立即替换为纯视觉描述
强制扫描：输出前逐句检查

### 运镜可执行性
都市狭小空间（<20平米）禁止环绕/旋转/穿墙

### 出图模式禁标注
默认禁止文字/箭头/编号/红框/引线/色温数字

### 负面清单
```
no watermark, no logo, no random large text, no garbled Chinese,
no broken faces, no duplicated limbs, no messy panels,
no low-quality collage, no text overlay, no speech bubbles (unless manga format),
no cartoon style (unless specified), no flat illustration, no marketing poster style
```

## 11. 默认输出模板

### 7+1 模块输出
1. 故事核心（片名/类型/冲突）
2. 情绪基调（曲线+主色调）
3. 视觉风格（名称+关键词）
4. 角色锚点（主角+对手DNA）
5. 场景锚点（空间+光源+氛围）
6. 分镜表（N帧完整）
7. 出图/视频 Prompt
8. 技术参数总表 — 画幅/摄影机/焦段覆盖/景深策略/色彩系统（当用户指定电影级CG/超写实等高级需求时输出）

### 分镜表格式
```
帧N | 时间码 | 景别 | 画面描述 | 运镜 | 焦段 | 色彩/灯光 | 转场
```
