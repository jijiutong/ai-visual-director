# 14 条核心规则

> 版本：v0.5.0

---

## 规则列表

### 规则 1：默认少选项，直接给推荐
第一轮只输出「一句话识别 + 3 个方向」，不展示完整信息提取表。完整表放入一键生成输出。

### 规则 2：一键生成默认单一主风格，不主动融合
禁止自动风格融合。仅当用户明确说「融合」「多版本」「A+B」时融合。

### 规则 3：所有角色必须有剧情身份；冲突时输出身份解释
必含剧情身份字段（师父/徒弟/对手/旁观者）。高位身份词冲突时输出【身份解释】，不抹除用户描述。

### 规则 4：场景/角色卡默认无文字标注；导演标注版单独触发
出图模式禁止文字/箭头/编号/红框/引线。导演标注版仅当用户明确说「导演标注版」「标注」时启用。

### 规则 5：精准修改必须保留镜号/时间码/镜头顺序
单镜修改不改动时间码和顺序。除非用户明确要求调整时长。

### 规则 6：连续性强字段变更必须自动补桥
修改天气/时间/地点/角色外观 → 前一镜末插入过渡 → 联动微调前后镜 → 输出连续性提示。

### 规则 7：风格迁移只改视觉语言，不改剧情事实
迁移开头声明 🔒 锁定 + 🎨 变更。锁定：角色DNA/剧情/关键动作/镜数/时长。变更：色彩/光影/构图/运镜/节奏。

### 规则 8：古代场景禁止现代光源词
古代/玄幻/历史场景迁移至现代风格时：10 个允许词（霓虹感/绿色色罩/湿润光晕/胶片颗粒...），11 个禁用词（霓虹灯光/LED/街灯/城市灯光...）。

### 规则 9：Prompt 出图模式禁止文字/箭头/编号/红框
场景参考图和角色设定卡默认无任何标注元素。信息通过光影/构图/色彩传达。

### 规则 10：标注/编号/红框只在导演标注版出现
仅 `/scene 导演标注版` 和 `/character 导演标注版` 允许技术标注。

### 规则 11：科幻不默认机甲
机甲/驾驶舱/外骨骼/机器人战斗 → 科幻机甲。
太空/空间站/仿生人/AI/外星 → 暗黑科幻。
赛博/未来都市/黑客/虚拟现实 → 赛博佛学。

### 规则 12：推荐用通用风格名，禁用品牌名
默认推荐：3D动画电影风 / 日系手绘治愈风 / 童话绘本风。
禁止：Disney / Pixar / 吉卜力 / Studio Ghibli / 迪士尼。
品牌名仅「看全部」时展示。

### 规则 13：Prompt 禁止需模型生成的可读文字
绝对禁止中文/英文/数字可读文字。屏幕/标识/书本内容 → 纯视觉描述。
强制扫描：输出前逐句检查引号内文字/数字序列/「显示」「写着」+内容。
扫描不通过不输出。出现一次扣 2 分。

### 规则 14：都市狭小空间运镜保持物理可执行性
场景<20平米 → 仅可用：固定机位/慢推拉/水平横移/低角度仰拍/过肩/变焦。
禁止：环绕慢旋/中轴旋转/航拍上升/穿墙。
