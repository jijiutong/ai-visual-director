# AI Visual Director 命令参考

> 命令体系分为：6 个核心入口、自然语言动作、输入源命令、质检修复、兼容入口和 API 直出。

---

## 6 个核心入口

### `/create` — 一键总编排

总编排入口。用户给一个故事，系统自动完成故事摄入、镜头预算、视频导演、资产规划、平台适配、质检修复和最终打包。

`/create` 不替代三大核心资产入口，而是自动调用它们：

```text
/character 角色卡（谁）
/scene     场景图（在哪）
/storyboard 分镜图（怎么拍）
```

| 子指令 | 效果 |
|--------|------|
| `/create [故事]` | 一键生成完整视频执行包 |
| `/create 一键生成 [故事]` | 零确认最快路径 |
| `/create 10s/15s/30s [故事]` | 指定时长，自动决定镜数和是否拆段 |
| `/create 竖屏 [故事]` | 9:16 短视频优先 |
| `/create 多版本 [故事]` | A/B/C 三个视觉方向 |
| `/create 一键全平台 [故事]` | GPT Image / MJ / SD / 视频平台适配版本 |

默认链路：

```text
task-router
→ sources
→ story-intake
→ shot-budget
→ video-director
→ asset-plan
→ reference-anchor
→ motion-physics
→ video-prompt-assembly
→ prompt-scorer
→ final-video-qc
→ auto-repair
→ render-package
```

### `/source` — 输入源 / 项目读取

负责 Obsidian、Markdown、frontmatter、章节目录和批量处理。只做输入处理，不直接替代导演链路。

| 子指令 | 效果 |
|--------|------|
| `/source 从 Obsidian 读取 [路径/项目名]` | 扫描 Vault 或项目目录 |
| `/source Obsidian 读取 [项目名] 全部 15s 7镜` | 全章批量处理 |
| `/source 读取 [Markdown 文件]` | 解析单个 Markdown 故事 |
| `/source 解析 frontmatter` | 读取 YAML 元数据并预填字段 |
| `/source 批处理 [章节范围]` | 多章节循环走 `/create` 链路 |

### `/storyboard` — 核心资产：故事板 / 分镜图

视觉导演的中枢资产。需要全案图、分镜表、镜头设计、分镜图 Prompt 时使用。

| 子指令 | 效果 |
|--------|------|
| `/storyboard [故事]` | 生成分镜方案和智能风格推荐 |
| `/storyboard 一键生成 [故事] [时长] [镜数]` | 直接出完整故事板 |
| `/storyboard 简洁` | 精简输出：角色锚点 + 场景锚点 + 分镜表 + Prompt |
| `/storyboard 多版本` | A/B/C 三版本对比 |
| `/storyboard 看全部` | 展示视觉风格和版式样式 |

### `/character` — 核心资产：角色卡

生成角色卡、三视图、面部多角度、表情范围、服装武器细节和角色 DNA。

| 子指令 | 效果 |
|--------|------|
| `/character [角色名] [描述]` | 默认 6 模块角色设定卡 |
| `/character 导演标注版` | 红色标注/编号版，用于团队协作 |
| `/character 三视图` | 正/侧/背三视图 |
| `/character 面部多角度` | 5 角度面部特写 |
| `/character 12表情` | 3×4 情绪范围图 |
| `/character 服装武器细节卡` | 服装、武器、饰品细节 |
| `/character 角色DNA` | 20 字段文字锚定 |

### `/scene` — 核心资产：场景图

生成场景参考、空间锁定、全景图、光照材质和机位动线。

| 子指令 | 效果 |
|--------|------|
| `/scene [描述]` | 默认全能参考图 |
| `/scene 全能参考图` | 一张图锁场景、角色、光照、道具、氛围 |
| `/scene 九宫格锁定` | 多角度锁定复杂场景 |
| `/scene 720全景` | 360°×180° 全景环境 |
| `/scene 场景蓝图` | 俯视布局、站位、机位动线 |
| `/scene camera remix` | 同场景多角度镜头变化 |

### `/video` — 视频执行包

已有故事板或视觉资产后，用它生成平台视频 Prompt、首尾帧、合并帧、跨段衔接和最终执行清单。

| 子指令 | 效果 |
|--------|------|
| `/video 出视频 prompt` | 压缩模式，直喂 Seedance/可灵 |
| `/video 详细模式` / `展开 prompt` | 详细段落版，适合 Runway/Sora |
| `/video 生成视频分镜图` / `合并帧` | 3-4 张画面锚点 |
| `/video 生成首尾帧` | 帧1 + 帧N 对照图 |
| `/video 切分各帧` / `逐帧接力` | 逐帧出图和锁帧链 |
| `/video 继续下一段` / `续写下一段` | 跨段衔接和剧情推进 |
| `/video 检查视频` | 生成后 10 项检查 |

## 自然语言动作命令

这些不一定带 `/`，由 `engines/task-router.md` 自动识别。

| 指令 | 路由 |
|------|------|
| `一键生成` / `直接出` / `auto` | `/create` 最快路径 |
| `多版本` / `A/B/C` | 多版本视觉方向 |
| `一键全平台` | 多平台 Prompt 输出 |
| `生成角色卡` | `/character` |
| `生成场景参考图` / `生成场景图` | `/scene` |
| `生成视频分镜图` / `合并帧` | `/video` |
| `出视频 prompt` | `/video` 压缩输出 |
| `详细模式` / `展开 prompt` | `/video` 详细输出 |
| `继续下一段` / `续写下一段` | 跨段衔接 |
| `检查视频` / `视频检查` | 视频质检 |
| `评分` | Prompt 质量评分 |
| `优化后生成` | 评分 + 自动修复 + 输出 |
| `压缩到 MJ` / `压缩到 SD` / `压缩到可灵` | 平台长度压缩 |
| `换成 [风格] 但保持角色` | 风格迁移 |
| `第X镜 [修改内容]` | 单镜精准修改 |

### Mood 滑块

| 指令 | 色彩 | 运镜 | 镜头 |
|------|------|------|------|
| `更燃` | 暖色 | 快切 | 低角度 + 疾推 |
| `更虐` | 冷色 | 慢镜 | 高角度俯拍 + 拉远 |
| `更甜` | 柔光 + 粉 | 环绕 | 浅景深 |
| `更丧` | 去饱和 | 固定 | 空旷构图 |

| `更恐怖` | 暗调 | 手持 | 窥视视角 |
| `更史诗` | 金色逆光 | 慢推 | 广角 + 航拍 |

## 兼容入口

旧命令继续可用，但不作为主流程推荐入口。

| 命令 | 新定位 |
|------|--------|
| `/style` | 风格浏览、风格融合、风格迁移 |
| `/poster` | 海报/封面输出格式 |
