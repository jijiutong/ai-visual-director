# AI Visual Director 用户版说明书

> 5 分钟上手，把故事变成角色卡、场景图、分镜图和视频执行包。

---

## 最简单的用法

### 1. 粘贴故事

```
雨夜古寺，两名剑客在佛像前对峙，师父发现徒弟已经入魔，二人拔剑相向。
```

系统自动识别题材 → 摄入故事字段 → 规划镜头与资产 → 输出视频执行包。

### 2. 一键生成视频包

```
一键生成
```

或直接从故事开始：

```
/create 一键生成 深夜便利店，女主下班后独自吃关东煮，前男友推门进来买烟。15s 6镜
```

输出：故事摄入 + 镜头预算 + 角色卡 Prompt + 场景图 Prompt + 分镜图 Prompt + 视频 Prompt + 执行清单。

## 快速调整

### 改情绪

| 指令 | 效果 |
|------|------|
| `更燃` | 暖色+快切+低角度 |
| `更虐` | 冷色+慢镜+俯拍 |
| `更甜` | 柔光+粉色+环绕 |
| `更丧` | 去饱和+固定机位 |
| `更暧昧` | 暖暗调+负空间 |
| `更史诗` | 广角+航拍+金色逆光 |

### 改单镜

```
第3镜更暗一点
第5镜雨改雪
第7镜更史诗
```

镜号不变、时间码不变、只改指定帧。天气变更自动补过渡。

### 换风格

```
换成王家卫风格但保持角色
换成水墨风但保持角色
```

角色不变、剧情不变、镜数时长不变。只换色彩/运镜/构图/质感。

## 出视频

```
出视频 prompt
```

```
一键生成 → 生成角色卡 → 生成场景参考图 → 出视频 prompt
```
