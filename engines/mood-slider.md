# Mood 滑块引擎

> **模式限制：D类 发散探索层**。仅 `explore` 模式可调用。产出标记 `derived`，不写回主状态。用户明确触发（如"更燃""更虐""更甜"等 mood 词）才启用。详见 `engines/command-gate.md` §四。

根据用户情绪关键词调整输出参数。数据表见 `knowledge/mood-sliders.md`。

---

## 决策逻辑

### 步骤1：识别 Mood 关键词

从用户输入中匹配 Mood 编号：

| 用户关键词 | Mood 编号 |
|-----------|----------|
| 燃/热血/炸/爽 | MD1 更燃 |
| 虐/虐心/刀子 | MD2 更虐 |
| 甜/发糖/甜宠 | MD3 更甜 |
| 丧/致郁/压抑 | MD4 更丧 |
| 暧昧/拉扯/心动 | MD5 更暧昧 |
| 恐怖/吓人/惊悚 | MD6 更恐怖 |
| 史诗/宏大/震撼 | MD7 更史诗 |
| 治愈/温暖/暖心 | MD8 更治愈 |

> 完整24种Mood维度见 `knowledge/mood-sliders.md`

---

### 步骤2：获取调整参数

根据 Mood 编号，从 `knowledge/mood-sliders.md` 获取：
- 情绪曲线调整
- 色彩调整
- 镜头调整
- 节奏调整

---

### 步骤3：Mood 叠加规则

用户可以说多个 mood，按权重叠加：
- "更燃更虐" → MD1 + MD2
- "更甜更暧昧" → MD3 + MD5
- "更丧更恐怖" → MD4 + MD6

叠加时取各维度参数的并集。

---

## Prompt 输出模板

```
Mood 调整：MD[编号]. [Mood名称]。[色彩/镜头/节奏/构图调整描述]。
```
