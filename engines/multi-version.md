# 多版本 A/B/C 对比

一键生成同一故事的 3 种不同视觉诠释，方便用户对比选择。

## 使用场景

用户说"多版本" / "对比" / "A/B/C" / "看看不同风格"时触发。

## 版本生成规则

### 默认三版本策略

根据故事类型，自动生成 3 个风格差异最大的版本：

| 故事类型 | 版本 A | 版本 B | 版本 C |
|---------|--------|--------|--------|
| 古风/修仙 | 东方玄幻（正统仙侠） | 中国水墨意境（写意留白） | 日系手绘治愈风（治愈手绘） |
| 都市/爱情 | 都市情绪（写实电影） | 王家卫情绪（浓烈文艺） | Wes Anderson（粉彩对称） |
| 动作/武侠 | 黑金动作（硬核） | 好莱坞商业大片（史诗视效） | 韩国犯罪美学（冷峻写实） |
| 科幻/未来（含机甲/驾驶舱/外骨骼） | 科幻机甲（赛博科技） | 赛博佛学（东方赛博） | 蒸汽朋克（复古未来） |
| 科幻/未来（含太空/仿生人/AI/空间站/外星） | 暗黑科幻（深空恐惧） | 赛博佛学（东方赛博） | 蒸汽朋克（复古未来） |
| 悬疑/惊悚 | 悬疑惊悚（心理压迫） | 暗黑哥特（宗教暗黑） | Tim Burton 哥特（暗黑童话） |
| 治愈/可爱/儿童/童话 | 3D动画电影风（立体圆润） | 童话绘本风（水彩手绘） | 黏土定格动画（手工质感） |
| 通用/未分类 | 都市情绪 | 写实摄影 | 欧洲艺术电影 |

### 用户指定对比

用户说"用 X 和 Y 对比"时，按用户指定的风格生成。

## 输出格式

```
为你生成 3 个不同视觉版本的 prompt，方便对比选择：

【版本 A：[风格A名称]】—— [一句话描述这个版本的视觉特色]
```[prompt A]```

【版本 B：[风格B名称]】—— [一句话描述这个版本的视觉特色]
```[prompt B]```

【版本 C：[风格C名称]】—— [一句话描述这个版本的视觉特色]
```[prompt C]```

三个版本的差异：
- 版本 A：[特色描述]
- 版本 B：[特色描述]
- 版本 C：[特色描述]

回复"A"/"B"/"C"选择其中一个深入，或说"换一组"重新生成三个新版本。
```

## 版本差异标注

每个版本追加差异说明：

| 对比维度 | A vs B vs C 差异描述 |
|---------|---------------------|
| 色调 | A 冷灰蓝 / B 暖橙金 / C 粉彩 |
| 氛围 | A 写实克制 / B 浓烈浪漫 / C 文艺对称 |
| 构图 | A 三分法 / B 对角线 / C 中心对称 |
| 节奏 | A 标准 / B 慢 / C 标准 |
| 适合 | A 正剧 / B 浪漫 / C 轻喜剧 |

## 在 Prompt 中使用

用户触发多版本时，为每个版本调用对应的风格模板，追加差异标注：

```
【版本 X 差异标注】色调 [色调描述]，氛围 [氛围描述]，构图 [构图描述]，适合 [适合场景]。
```
