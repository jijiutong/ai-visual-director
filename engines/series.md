# 续集/系列支持

用户说"这是第2集"或"继续上一集"，自动保持角色DNA/色彩基调/场景延续性，同时体现剧情推进。

## 系列信息追踪

### 需要保持的元素

| 元素 | 保持方式 |
|------|---------|
| 角色 DNA | 完全一致（服装/发型/面部/配饰） |
| 色彩基调 | 主色调保持一致，允许微调（如从冷到暖的情绪变化） |
| 场景基调 | 同一场景的材质/灯光/天气保持一致 |
| 视觉风格 | 风格编号不变 |
| 镜头语言 | 摄影机/镜头类型保持一致 |

### 需要变化的元素

| 元素 | 变化方式 |
|------|---------|
| 情绪曲线 | 根据新剧情重新分配 |
| 分镜节奏 | 根据新场景类型调整 |
| 角色状态 | 追加临时状态（受伤/换装/情绪变化） |
| 场景变化 | 新场景或同一场景的不同时段/天气 |
| 色彩叙事 | 根据新情绪曲线演进 |

## 系列指令

| 用户指令 | AI 行为 |
|---------|---------|
| "这是第 X 集" | 新建系列，记录当前角色 DNA/色彩/场景 |
| "继续上一集" / "下一集" | 延续上一集的 DNA/色彩/场景，生成新内容 |
| "回到第 X 集" | 调取指定集数的设定，继续创作 |
| "总结一下系列设定" | 输出当前系列的所有角色 DNA/色彩基调/场景记录 |

## 系列状态存储格式

```
【系列：[片名]】
集数：当前第 X 集
风格：[风格编号. 风格名称]
色彩基调：[主色1] + [主色2] + [点缀色]
角色 DNA：
  - [角色1名]DNA: [完整DNA]
  - [角色2名]DNA: [完整DNA]
场景记录：
  - 场景1：[地点] [时间] [天气] [材质]
  - 场景2：...
情绪曲线：[类型]
```

## 角色状态变化

续集中角色状态可能变化，用追加方式：

```
[角色名] 当前状态：[基础DNA] + [变化描述]
```

例：
```
碧瑶 当前状态：碧瑶DNA（基础） + 表情疲惫/衣衫微乱/额间朱砂痣光芒暗淡（力量消耗后）
```

## 色彩基调变化

系列中色彩基调可以缓慢演变：

```
第1集：冷灰蓝（孤独/初遇）
第2集：冷灰蓝 + 暖橙（关系升温）
第3集：暖橙 + 粉红（甜蜜/暧昧）
第4集：暖金（确认/圆满）
```

## 在 Prompt 中使用

续集 prompt 开头追加系列信息：

```
【系列延续】《[片名]》第 [X] 集。延续第 [X-1] 集设定：风格 [风格编号]，色彩基调 [色调]，角色 DNA [引用]。
本集变化：[情绪曲线/节奏/角色状态/场景变化]。
```
