# 版式样式引擎

根据输出类型自动匹配版式样式。数据表见 `knowledge/layout-styles.md`。

---

## 决策逻辑

### 步骤1：识别输出类型

从用户需求中识别输出类型：

| 用户关键词 | 版式编号 |
|-----------|---------|
| 全案图/项目总览 | LS14（主）/LS1/LS2/LS3/LS4 |
| 分镜图/故事板 | LS5/LS6/LS7 |
| 场景图/环境锁定 | LS8/LS9/LS10 |
| 角色卡/人物设定 | LS11/LS12/LS13/LS41/LS42/LS43/LS44 |
| 竖屏短剧 | LS34 |
| 动作拆解 | LS35 |
| 巨物场景 | LS31 |
| 真人脸一致 | LS20 |

> 完整44种版式定义见 `knowledge/layout-styles.md`

---

### 步骤2：获取版式参数

根据版式编号，从 `knowledge/layout-styles.md` 获取：
- 适用场景
- 结构描述
- 视觉风格
- 强调要点
- 避免事项

---

### 步骤3：VS 与 LS 分离

- **VS（视觉风格）**：决定美术风格（像什么类型片）
- **LS（版式样式）**：决定图面组织（如何排版）

示例组合：
- VS7 东方玄幻 + LS6 横幅时间轴
- VS2 科幻机甲 + LS13 角色系统卡
- VS1 黑金动作 + LS12 武侠角色圣经

---

### 步骤4：视频锚点优先少文字

给视频模型看的版式优先：
- LS7 合并帧视频锚点
- LS10 双联宽幅场景锚点
- 模式A 6模块全量版（默认角色卡）
- LS40 首尾帧对照板

规则：减少文字、表格、箭头，保持纯画面。

---

## Prompt 输出模板

```
版式样式：LS[编号]. [版式名称]，[结构描述]，[视觉风格]。
```

| 入口 | 首选组合 |
|------|---------|
