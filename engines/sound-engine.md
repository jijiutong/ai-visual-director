# 音效设计引擎

按情绪阶段、场景类型、动作内容自动匹配 SE/FX/MU/RS 编号。独立于 video-director，可单独调用或通过 `/sound` 子路由使用。

---

## 职责边界

| 引擎 | 职责 |
|------|------|
| `dialogue-engine` | 标注台词节奏和语气——音效需与台词时机配合 |
| **`sound-engine`（本引擎）** | 按镜头阶段自动匹配环境音/拟音/音乐/混响 |
| `video-director` | 调用本引擎，不自己选音效 |
| `video-prompt-assembly` | 读取 sound-map 组装音效层 |

---

## 一、读取

| 读取参数 | 来源 | 用途 |
|---------|------|------|
| 类型/genre | `state/variable-registry.md` → story.genre | 确定音效风格基调 |
| 场景/环境 | `state/variable-registry.md` → scene.primary | 匹配环境音 SE 编号 |
| 各镜阶段 | `state/shot-state.md` → shots[].phase | 匹配音乐 MU 编号 |
| 各镜动作 | `state/shot-state.md` → shots[].action | 匹配拟音 FX 编号 |
| 场景空间 | `state/variable-registry.md` → scene.primary.location_type | 匹配混响 RS 编号 |
| 台词时机 | `state/dialogue-map.md` → 台词所在镜号和时长 | 音效与台词配合 |
| 音效知识库 | `knowledge/sound-design.md` | SE/FX/MU/RS 完整编码 |
| 类型推荐 | `knowledge/audio-reference.md` | 按类型推荐音乐音效组合 |

## 三、输出 → 写入 state/sound-map.md

```yaml
sounds:
  - sound_id: S1
    shot_id: SH01
    type: ambient
    sd_code: SE2
    description: 狂风呼啸，山顶孤绝
    timing: 0-4s
    volume: medium

  - sound_id: S2
    shot_id: SH03
    type: foley
    sd_code: FX3
    description: 拔剑出鞘，金属摩擦+清越剑鸣
    timing: 5s（瞬间）
    volume: high
    sync_with: D1（台词结束后）

  - sound_id: S3
    shot_id: SH01
    type: music
    sd_code: MU4
    description: 古琴独奏+箫铺底，慢 60 BPM
    timing: 0-4s
    volume: low→medium

  - sound_id: S4
    shot_id: SH01
    type: reverb
    sd_code: RS5
    description: 室外旷野，风声为主无混响
```

→ 被 `video-prompt-assembly` 读取（追加音效层）
