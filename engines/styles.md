# 视觉风格引擎

根据故事类型自动匹配视觉风格。数据表见 `knowledge/visual-styles.md`。

---

## 决策逻辑

### 步骤1：识别故事类型

从 `state/variable-registry.md` 读取 `project.genre`（由 story-intake 写入），匹配风格编号：

| 故事关键词 | 推荐风格编号 |
|-----------|-------------|
| 武侠/复仇/谍战/犯罪/战争 | VS1, VS5, VS11 |
| 机甲/赛博/太空歌剧/AI | VS2, VS34, VS35 |
| 修仙/古风/神魔/仙侠 | VS7, VS8, VS10 |
| 悬疑/犯罪/推理/恐怖 | VS4, VS39, VS40 |
| 都市/爱情/职场/家庭 | VS29, VS37 |
| 校园/青春/成长/初恋 | VS24, VS38 |
| 末日/废土/丧尸/求生 | VS36 |
| 宫廷/权谋/古装 | VS8 |
| 童话/治愈/奇幻 | VS22, VS24, VS25 |
| 动画/二次元/异世界 | VS21, VS23 |

> 完整风格定义见 `knowledge/visual-styles.md`

---

### 步骤2：获取风格参数

根据风格编号，从 `knowledge/visual-styles.md` 获取：
- 配色方案
- 氛围关键词
- 环境特效
- 灯光方案
- 镜头语言
- 禁止方向

---

### 步骤3：风格融合

用户说"A+B风格"时，按融合规则处理：

| 融合类型 | 权重分配 |
|---------|---------|
| 主风格+副风格 | 60% 主 + 40% 副 |
| 三风格融合 | 50% 主 + 30% 副 + 20% 第三 |

**融合规则**：
- 配色：取主风格配色 + 副风格点缀色
- 氛围词：主风格氛围词 + 副风格氛围词（去重）
- 灯光：主风格灯光为主，副风格灯光点缀
- 镜头语言：主风格镜头为主，副风格镜头穿插
- 禁止：取两者禁止项并集

> 详细融合规则见 `engines/fusion.md`

---

### 步骤4：输出风格参数

整合风格编号 + 风格参数，输出完整风格描述。

---

## Prompt 输出模板

```
整体风格：[编号]. [风格名称]，[氛围词]，配色 [配色方案描述]。
灯光：[灯光方案]。
镜头语言：[镜头语言]。
禁止：[禁止方向]。
```
