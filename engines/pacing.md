# 分镜节奏引擎

根据类型自动匹配分镜节奏。数据表见 `knowledge/pacing-types.md`。

---

## 决策逻辑

### 步骤1：识别故事类型

从 SKILL.md Step 1 提取的故事类型，匹配节奏编号：

| 故事类型 | 节奏编号 |
|---------|----------|
| 动作/武侠/格斗/战争 | P1 快切 |
| 科幻/机甲/爆炸 | P1 快切 |
| 都市/剧情/职场 | P2 标准 |
| 悬疑/侦探/推理 | P4 渐进 |
| 惊悚/恐怖/密室 | P4 渐进 |
| 爱情/治愈/情感 | P3 慢节奏 |
| 校园/青春/日常 | P2 标准 |
| 喜剧/搞笑/轻松 | P5 喜剧 |
| 文艺/艺术/诗意 | P3 慢节奏 |
| 史诗/宏大/奇幻 | P2→P1（高潮加速） |

> 完整匹配表见 `knowledge/pacing-types.md` 三、类型→节奏匹配表

---

### 步骤2：计算镜数

根据节奏编号 + 时长，查表计算镜数：

| 时长 | P1 快切 | P2 标准 | P3 慢节奏 | P4 渐进 | P5 喜剧 |
|------|---------|---------|-----------|---------|---------|
| 10s  | 7镜 | 5镜 | 3镜 | 5镜 | 4镜 |
| 15s  | 9镜 | 7镜 | 5镜 | 7镜 | 6镜 |
| 30s  | 15镜 | 10镜 | 7镜 | 10镜 | 8镜 |
| 60s  | 25镜 | 15镜 | 10镜 | 13镜 | 12镜 |

> 完整计算表见 `knowledge/pacing-types.md` 二、时长×节奏镜数计算表

---

### 步骤3：获取节奏参数

根据节奏编号，从 `knowledge/pacing-types.md` 获取：
- 每镜时长
- 剪辑频率
- 运镜特点
- 景别变化
- 声音设计
- Prompt 追加模板

---

### 步骤4：联动情绪曲线

将节奏编号 + 镜数传递给 `engines/emotion-curve.md`，确定每镜的阶段/景别/运镜分配。

---

## 联动流程图

```
故事类型 ──→ pacing.md ──→ 节奏编号(P1-P5)
                │
                ▼
           时长 + 节奏 ──→ 镜数
                │
                ▼
        knowledge/pacing-types.md ──→ 节奏参数
                │
                ▼
        engines/emotion-curve.md ──→ 每镜分配
```

---

## Prompt 输出模板

```
分镜节奏：[节奏名称]，每镜 [时长范围]，[镜数]镜共 [总时长]。
[剪辑频率描述]，景别[变化规律]，运镜以 [运镜特点] 为主。
声音：[声音设计]。
```
