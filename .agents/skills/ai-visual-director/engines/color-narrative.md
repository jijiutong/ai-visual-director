# 色彩叙事引擎

颜色随情绪推进而变化。不只是静态配色，而是动态色彩叙事。50+ 种色彩方案，覆盖各类题材和情绪弧线。

---

## 一、标准四阶段色彩演进模型

### 通用四阶段

```
阶段1 平静：冷灰蓝 #2a2a3a + 中性白 #f0f0f0          → 孤独/日常/铺垫
阶段2 对峙：冷灰蓝 + 暖橙 #ff6633（开始混合）         → 冲突萌芽/张力
阶段3 爆发：血红 #cc0033 + 金色 #ffd700（高对比）     → 高潮/释放/决断
阶段4 余韵：暖米 #f5e6d3 + 柔和金 #daa520              → 释怀/温暖/新开始
```

### 各类型色彩叙事

| 类型 | 色彩演进 |
|------|---------|
| 动作型 | 暗部(黑金) → 金属火花(金) → 爆炸火光(红+橙) → 硝烟余灰(灰) |
| 爱情型 | 冷色调(孤独蓝灰) → 冷暖交汇(蓝+粉) → 暖色调(粉红+金) → 柔和统一(暖米) |
| 悬疑型 | 暗蓝(不安) → 暗红渐变(危险接近) → 血红爆发(真相揭示) → 苍白(虚无/反转) |
| 悲剧型 | 暖金(美好) → 冷灰入侵(裂痕) → 深蓝黑(崩溃) → 灰白(余痛/空白) |
| 成长型 | 暗淡(起点) → 微光出现(希望) → 多彩爆发(突破) → 金色成熟(成长完成) |

## 三、色彩对比策略

| 对比类型 | 用法 | 示例 |
|---------|------|------|
| 冷暖对比 | 内心冲突/两人对峙 | 一人站冷光中，一人站暖光中 |
| 明暗对比 | 善恶/希望绝望 | 大面积暗部 + 单一亮光 |
| 饱和度对比 | 现实 vs 回忆 | 现实低饱和 + 回忆高饱和 |
| 色相对比 | 阵营对立 | 蓝方 vs 红方 |
| 纯度渐变 | 情绪推进 | 从低纯度灰 → 高纯度纯色 |

## 在 Prompt 中使用

```
色彩叙事：按情绪曲线 [阶段数] 阶段演进——
方案：[编号. 方案名称]。
阶段1[平静]：[色1]+[色2]，[情绪描述]
阶段2[对峙]：[色1]+[色3]（冷暖开始混合），[情绪描述]
阶段3[爆发]：[色4]+[色5]（高对比爆发），[情绪描述]
阶段4[余韵]：[色6]+[色7]（统一色调），[情绪描述]
```

例：
```
色彩叙事：按情绪曲线四阶段演进。
方案：CN2. 虐心刀子。
阶段1[平静]：暖金 #ffd700 + 柔白，美好初遇
阶段2[对峙]：暖金 + 冷灰 #8a8a8a 入侵，裂痕出现
阶段3[爆发]：深蓝黑 #0a0a2a + 暗红，崩溃绝望
阶段4[余韵]：灰白 #d0d0d0 余痛空白，无言结局
```
