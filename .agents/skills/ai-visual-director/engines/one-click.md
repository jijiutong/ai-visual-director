# 一键输出工作流

从故事→提取→推荐→确认→全平台输出，整个流程一句话完成。

## 一键指令

| 用户指令 | 触发工作流 |
|---------|-----------|
| "一键生成" / "直接出" / "auto" | 全流程自动完成，不展示选项 |
| "一键全平台" | 全流程 + 三个平台输出 |
| "一键多版本" | 全流程 + A/B/C 三版本对比 |
| "一键全来" | 全流程 + 批量 4 格式 + 全平台 |

## 一键生成工作流（全自动）

```
用户输入 → 自动提取 → 自动匹配风格+格式 → 自动追加关系/情绪/DNA/文化 → 输出 prompt
```

**不展示任何选项**，直接输出最终 prompt。

### 自动决策规则

| 决策项 | 自动策略 |
|--------|---------|
| 风格选择 | 按关键词匹配表取第一个推荐 |
| 格式选择 | 按故事类型取最相关格式 |
| 情绪曲线 | 按类型自动匹配（动作→标准，爱情→爱情型，悬疑→悬疑型） |
| 角色关系 | 按互动描述自动识别 |
| 分镜节奏 | 按类型自动匹配（动作→快切，文艺→慢） |
| Mood | 无指定则不追加，有"更X"则追加 |
| 平台 | 默认 GPT Image 2 |

## 一键全平台工作流

```
用户输入 → 自动提取 → 自动匹配 → 生成 GPT Image 2 版本 → 翻译为 MJ 版本 → 翻译为 SD 版本 → 三个版本同时输出
```

## 一键多版本工作流

```
用户输入 → 自动提取 → 生成 3 个不同风格的版本 → 每个版本自动完成全流程 → 三个版本对比输出
```

## 一键全来工作流（终极模式）

```
用户输入 → 自动提取 → 
  ├→ 生成 A/B/C 三个风格版本
  ├→ 每个版本生成 4 个格式（全案板/角色卡/海报/情绪板）
  ├→ 每个格式生成 3 个平台版本（GPT/MJ/SD）
  └→ 输出 3×4×3 = 36 个 prompt
```

**注**：全来模式输出量大，建议分段输出。

```[完整 prompt]```
