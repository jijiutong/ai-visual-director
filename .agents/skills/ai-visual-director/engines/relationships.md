## 关系详细预设

### R1. 宿敌变恋人

- **构图建议**：对角线对峙构图 → 逐渐靠近的正面构图 → 最终并肩
- **镜头语言**：low angle（双方各自仰拍显强大）→ eye level（平视对话）→ close-up（面部微表情）
- **灯光方案**：冷暖分割光（两人各站冷暖光中）→ 混合光（中间地带）→ 统一暖光
- **氛围关键词**：剑拔弩张、势均力敌、眼神交锋、刀尖舔血、相爱相杀、宿命纠缠
- **Prompt 追加**：
  ```
  角色关系：宿敌变恋人。构图从对角线对峙逐渐过渡到并肩站立，两人各占画面一侧形成对称张力。灯光采用冷暖分割方案。镜头从 low angle 各自仰拍推进到 eye level 平视，最终 close-up 面部微表情。氛围：剑拔弩张、势均力敌、眼神交锋、相爱相杀。
  ```

### R2. 双向暗恋

- **构图建议**：三分法构图，两人分别占据两个交点，视线交汇处为视觉中心
- **镜头语言**：over the shoulder → medium close-up → tight close-up（眼神交汇）
- **灯光方案**：柔和散射光 + 暖色补光
- **氛围关键词**：欲言又止、眼神闪躲、心跳加速、指尖触碰、假装不经意
- **Prompt 追加**：
  ```
  角色关系：双向暗恋。三分法构图，视线交汇处为视觉中心。过肩镜头展现试探与闪躲，tight close-up 定格眼神交汇。柔和散射光 + 暖色补光。氛围：欲言又止、眼神闪躲、心跳加速。
  ```

### R3. 师徒

- **构图建议**：俯视/仰视角度对比 → 最终平视
- **镜头语言**：high angle → low angle → eye level
- **灯光方案**：顶光（师傅神圣感）+ 侧光（徒弟仰望）
- **氛围关键词**：禁忌、仰望、越界、守护、克制、传承
- **Prompt 追加**：
  ```
  角色关系：师徒。构图从 high angle 俯视教导过渡到 eye level 平视。顶光营造神圣感，侧光凸显仰望。氛围：禁忌、仰望、克制、守护。
  ```

### R4. 年上/年下

- **构图建议**：体型差/身高差构图，年上者保护位，年下者依赖位
- **镜头语言**：wide shot 展示身高差 → medium → close-up 表情反差
- **灯光方案**：年上者暖光，年下者自然光
- **氛围关键词**：成熟引导、青春冲击、体型差、保护欲、依赖感
- **Prompt 追加**：
  ```
  角色关系：年上/年下。构图强调体型差和身高差。wide shot 展示对比，close-up 展示成熟稳重 vs 青春活力的表情反差。氛围：成熟引导、青春冲击、保护欲。
  ```

### R5. 先婚后爱

- **构图建议**：画面中间留白/隔断 → 隔断消失 → 亲密接触
- **镜头语言**：wide shot（分居两端）→ medium → close-up
- **灯光方案**：冷色调分隔 → 统一暖光
- **氛围关键词**：冷淡、契约、习惯、依赖、深爱
- **Prompt 追加**：
  ```
  角色关系：先婚后爱。构图从画面中间留白隔断过渡到亲密接触。灯光从冷色调分隔到统一暖光。氛围：冷淡→习惯→依赖→深爱。
  ```

### R6. 死对头/冤家

- **构图建议**：对称对峙构图，表情夸张喜剧化
- **镜头语言**：medium shot 互怼 → close-up 表情夸张 → medium 意外靠近
- **灯光方案**：明亮均匀光，偏喜剧调性
- **氛围关键词**：互怼、嘴硬心软、吃醋、口是心非、欢喜冤家
- **Prompt 追加**：
  ```
  角色关系：死对头/冤家。对称对峙构图，表情夸张喜剧化。明亮均匀光。氛围：互怼、嘴硬心软、口是心非。
  ```

### R7. 救赎/治愈

- **构图建议**：暗→明过渡构图，一人从暗部走向另一人的光中
- **镜头语言**：dark wide shot → push in → close-up
- **灯光方案**：大面积暗部 + 单一暖光源
- **氛围关键词**：黑暗、光、依赖、共生、破碎、修复
- **Prompt 追加**：
  ```
  角色关系：救赎/治愈。暗到明过渡构图。大面积暗部 + 单一暖光源。氛围：黑暗中的光、依赖、破碎与修复。
  ```

### R8. 替身/白月光

- **构图建议**：重叠/镜像构图
- **镜头语言**：rack focus → close-up 痛苦表情
- **灯光方案**：冷光（替身）+ 暖光回忆滤镜（白月光）
- **氛围关键词**：混淆、痛苦、选择、真假、执念
- **Prompt 追加**：
  ```
  角色关系：替身/白月光。重叠/镜像构图。rack focus 切换焦点。冷光笼罩替身 + 暖光回忆滤镜。氛围：混淆、痛苦、真假难辨。
  ```

### R9. 强制/囚禁

- **构图建议**：框中框构图
- **镜头语言**：dutch angle → tight close-up → wide
- **灯光方案**：单一方向强光 + 大面积阴影
- **氛围关键词**：控制、反抗、斯德哥尔摩、禁锢、危险吸引力
- **Prompt 追加**：
  ```
  角色关系：强制/囚禁。框中框构图强调禁锢压迫。dutch angle 不安，tight close-up 窒息感。单一强光 + 大面积阴影。氛围：控制、反抗、禁锢。
  ```

### R10. 战友/搭档

- **构图建议**：并肩构图，背对背或肩并肩
- **镜头语言**：wide battle shot → medium 配合 → close-up 信任眼神
- **灯光方案**：战场火光/爆炸光映照
- **氛围关键词**：信任、牺牲、默契、生死与共
- **Prompt 追加**：
  ```
  角色关系：战友/搭档。并肩构图。wide battle shot 展现配合，close-up 定格信任眼神。战场火光映照。氛围：信任、牺牲、默契、生死与共。
  ```

### R11. 重逢/破镜重圆

- **构图建议**：从画面两侧逐渐向中心靠近
- **镜头语言**：medium（远处认出）→ push in → close-up → over shoulder
- **灯光方案**：回忆暖光 + 现实冷光 → 统一暖光
- **氛围关键词**：回忆、回避、释怀、复合、物是人非
- **Prompt 追加**：
  ```
  角色关系：重逢/破镜重圆。构图从两侧向中心靠近。medium 远处认出，push in 走近，close-up 复杂表情。回忆暖光与现实冷光交织到统一暖光。氛围：物是人非、释怀、复合。
  ```

### R12. 身份对立

- **构图建议**：画面中间分界线（阵营分割）
- **镜头语言**：wide 阵营对立 → medium 私下相遇 → close-up 挣扎
- **灯光方案**：两侧不同色温 + 中间暗部
- **氛围关键词**：阵营冲突、私情、抉择、背叛、忠义两难
- **Prompt 追加**：
  ```
  角色关系：身份对立。画面分界线构图，各站阵营一侧。两侧不同色温灯光，中间暗部暗示灰色地带。氛围：阵营冲突、私情、忠义两难。
  ```

### R13. 养成/守护

- **构图建议**：大/小对比构图
- **镜头语言**：wide → medium → close-up → eye level（平等）
- **灯光方案**：守护者暖光 + 被守护者柔光
- **氛围关键词**：照顾、依恋、成长、从保护到平等
- **Prompt 追加**：
  ```
  角色关系：养成/守护。大/小对比构图。wide 守护者视角到最终 eye level 平等。暖光 + 柔光。氛围：照顾、依恋、成长、从保护到平等。
  ```

### R14. 暧昧拉扯

- **构图建议**：安全距离内的微妙靠近——手指将触未触
- **镜头语言**：medium close-up → close-up（手指/嘴唇/眼神特写）→ pull out
- **灯光方案**：暖色暧昧光 + 暗部留白
- **氛围关键词**：试探、靠近、后退、再靠近、心跳、克制
- **Prompt 追加**：
  ```
  角色关系：暧昧拉扯。构图强调安全距离微妙靠近——手指将触未触。close-up 特写手指/嘴唇/眼神。暖色暧昧光 + 暗部留白。氛围：试探、靠近、后退、心跳、克制。
  ```

### R15. 一见钟情

- **构图建议**：人群中的聚焦——周围虚化/暗化，两人清晰明亮
- **镜头语言**：wide crowd shot → crash zoom → slow motion → close-up
- **灯光方案**：聚光效果
- **氛围关键词**：命中注定、时间静止、心动
- **Prompt 追加**：
  ```
  角色关系：一见钟情。人群聚焦构图——周围虚化暗化，两人清晰明亮。crash zoom 目光锁定，slow motion 放慢一刻。聚光效果。氛围：命中注定、时间静止。
  ```

### R16. 血缘禁忌

- **构图建议**：安全距离内的渴望靠近，画面中间留一道不可逾越的间隙（门槛/栏杆/光影分界）
- **镜头语言**：medium（发现血缘冲击）→ close-up 痛苦隐忍 → wide（伦理鸿沟）
- **灯光方案**：冷灰蓝笼罩 + 两人之间有微弱暖光连接但被暗影切割
- **氛围关键词**：血缘冲击、伦理挣扎、克制、隐痛、割舍
- **Prompt 追加**：
  ```
  角色关系：血缘禁忌。构图中间留不可逾越的间隙——门槛/栏杆/光影分界。close-up 捕捉痛苦隐忍表情。冷灰蓝光笼罩，微弱暖光连接但被暗影切割。氛围：血缘冲击、伦理挣扎、克制、隐痛。
  ```

### R17. 双面间谍

- **构图建议**：分裂光构图（split lighting），半脸明亮半脸黑暗，暗示双重身份
- **镜头语言**：close-up（试探性对话）→ dutch angle（身份暴露危机）→ rack focus（信任 vs 背叛）
- **灯光方案**：分割光 + 霓虹冷光混合，明暗界线恰好从鼻梁穿过
- **氛围关键词**：试探、背叛、真心与谎言、身份危机、牺牲
- **Prompt 追加**：
  ```
  角色关系：双面间谍。分割光构图——半脸明亮半脸黑暗，明暗界线从鼻梁穿过。close-up 捕捉试探表情，dutch angle 身份暴露危机。霓虹冷光混合。氛围：试探、背叛、真心与谎言。
  ```

### R18. 失忆重逢

- **构图建议**：过肩镜头中对方有熟悉感却无法确认，视线交汇处有微妙迟疑
- **镜头语言**：medium（陌生对视）→ push in（记忆碎片闪回叠加）→ close-up（困惑→泪光）
- **灯光方案**：低饱和冷光 + 记忆闪回时叠加暖色滤镜层
- **氛围关键词**：陌生、熟悉、记忆碎片、心痛、无法确认
- **Prompt 追加**：
  ```
  角色关系：失忆重逢。过肩镜头中对方有熟悉感却无法确认。medium 陌生对视，push in 记忆闪回叠加，close-up 困惑到泪光。低饱和冷光 + 暖色记忆闪回层。氛围：陌生与熟悉交织、记忆碎片、心痛。
  ```

### R19. 克隆体冲突

- **灯光方案**：冷白光对称照射 + 中央暗部分割，一人有暖光点缀（原体）一人纯冷光（复制体）
- **氛围关键词**：自我 vs 复制体、存在危机、身份争夺、谁是真我
- **Prompt 追加**：

  ```
  角色关系：克隆体冲突。镜像对称构图——相同脸面孔对峙。close-up 捕捉微表情差异，crash zoom 自我认同崩溃。冷白光对称 + 中央暗部分割。氛围：自我 vs 复制体、存在危机、谁是真我。
  ```

### R22. 时间线交错

## 在 Prompt 中使用

在生成的 prompt 中，在"核心场景概念图"段落后追加关系加成：

```
角色关系加成：[关系编号. 关系名称]。[构图建议]。[灯光方案]。[氛围关键词]。
```

例：
```
角色关系加成：R2. 双向暗恋。构图采用三分法，两人分别占据三分线交点，视线交汇处为视觉中心。柔和散射光 + 暖色补光营造暧昧氛围。氛围：欲言又止、眼神闪躲、心跳加速、假装不经意。
```
