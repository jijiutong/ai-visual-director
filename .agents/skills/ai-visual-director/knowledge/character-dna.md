# 角色一致性 DNA 系统

解决 AI 生成最大痛点：角色跨镜头崩坏。建立角色 DNA 描述系统，确保每个镜头中服装/发型/面部/体态/声音完全一致。

## DNA 结构

每个角色生成一条 DNA 字符串，在所有镜头中重复引用：

```
[角色名]DNA: 性别,年龄段,体型,面部特征,发型,瞳色,肤色,声音质感,体态/步态,习惯性动作,服装(上/下/外),配饰,武器/道具,随身物品,气味,标志性特征,情绪底色,关系标记
```

| DNA7 | 肤色 | 色号+质感 | 冷白皮,细腻无瑕疵 |

## 状态变化追踪

角色在不同阶段可能发生变化，DNA 需要标记哪些字段可变、哪些不可变：

### 不可变字段（所有镜头严格一致）

| 字段 | 原因 |
|------|------|
| DNA4 面部特征 | 骨相不变 |
| DNA5 发型颜色 | 除非剧情染发 |
| DNA6 瞳色 | 生理特征 |
| DNA7 肤色 | 生理特征 |
| DNA18 标志性特征 | 角色识别核心 |

### 可变字段（随剧情变化，需记录版本）

| 字段 | 变化类型 | 示例 |
|------|---------|------|
| DNA11-13 服装 | 换装/破损/湿润 | 战斗后衣服破损 |
| DNA5 发型样式 | 散开/束起/剪短 | 战斗中长发散开 |
| DNA15 武器/道具 | 损坏/升级/更换 | 剑断裂后换剑 |
| DNA10 习惯动作 | 情绪极端时改变 | 从克制变为失控 |
| DNA19 情绪底色 | 角色弧光完成 | 从隐忍变为愤怒 |

### 临时状态（单镜头有效，不改变 DNA）

```
[角色名] 当前状态：[基础DNA] + 表情[EX编号 微表情] + 服装状态[凌乱/破损/湿润] + 特殊效果[发光/流汗/流血/受伤位置] + 台词状态[SP编号 语速][VL编号 音量]
```

## 一致性检查清单

在每个镜头生成前检查：

- [ ] 发型完全一致（长度/颜色/样式）
- [ ] 面部特征一致（脸型/眉形/特殊标记）
- [ ] 服装颜色不变（同一套服装在不同镜头中颜色统一）
- [ ] 配饰位置一致（发饰/首饰/腰带）
- [ ] 武器/道具外观一致
- [ ] 标志性特征在所有镜头中出现
- [ ] 瞳色不变
- [ ] 肤色不变
- [ ] 体型/身高比例一致

## 多角色交互时的 DNA 引用

当多个角色同框时，在 prompt 中明确引用双方 DNA：

```
画面中同时出现 [角色A名] 和 [角色B名]：
[角色A名]：[完整DNA]
[角色B名]：[完整DNA]
两人相对位置：[距离/角度/姿态]
```

## 在 Prompt 中使用

在角色设定栏后追加 DNA 锚定：

```
角色 DNA 锚定（所有镜头必须保持一致）：
[角色名]DNA: [完整DNA字符串]

可变字段追踪（随剧情变化）：
- 服装: [初始状态] → [镜X后变为新状态]
- 发型: [初始状态] → [镜Y后散开/改变]
- 情绪底色: [初始状态] → [镜Z弧光完成后转变]
```

在分镜描述中简化引用：

```
镜3：[角色名]（参考DNA，当前状态[临时状态描述]）出现在[场景]，[动作描述]
```
