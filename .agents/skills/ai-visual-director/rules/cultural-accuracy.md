# 文化精准度系统

确保不同文化背景的描述精准无误，不出跨文化穿帮元素。覆盖中国各朝代、日本、韩国、欧洲、现代都市等。

## 五、文化混合场景

当故事涉及跨文化元素时：

1. **明确主次**：哪种文化为主（占 70%），哪种为辅（占 30%）
2. **不穿帮**：主文化的核心元素不被副文化替代
3. **融合点**：在装饰/道具/细节层面融合，不在核心架构层面混合

例："赛博朋克 + 东方玄幻" → 主架构赛博（城市/科技），装饰层东方（符文/阵法/仙衣纹样）

## 在 Prompt 中使用

```
文化精准度：[朝代/文化类型]。使用[正确元素列表]。禁止：[错误元素列表]。
```

例：
```
文化精准度：中国明朝。使用马面裙/袄裙/飞鱼服/发髻/金簪/四合院/园林/青花瓷/锦衣卫。禁止：清朝旗装/两把头/马褂/和服/韩服/西方城堡。
```
