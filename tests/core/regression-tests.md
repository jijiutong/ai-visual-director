# Core Regression Tests

> 4 条固定回归样本，每次修改 Skill 后必须全部通过。

---

## 评分维度

| # | 维度 | 满分标准 |
|---|------|---------|
| 1 | 题材识别 | 关键词匹配正确，推荐风格与题材一致 |
| 2 | 风格推荐 | 使用通用风格名，不暴露内部编号，不自动融合 |
| 3 | 情绪强度 | 情绪曲线与故事张力匹配 |
| 4 | 角色锚点 | 含剧情身份标注，含简要DNA，无红字标注 |
| 5 | 场景锚点 | 含空间+主光源+氛围，出图模式无标注 |
| 6 | 分镜节奏 | 帧数与时长匹配，景别递进合理 |
| 7 | 出图可执行性 | 无文字标注/可读文字，运镜可物理执行 |
| 8 | 连续性 | 修改保留时间码，天气变更自动补桥 |
| 9 | 风格迁移 | 声明显性/锁定不变项，不出现禁用词 |
| 10 | 题材禁忌 | 无串味/无过度史诗化/无古今穿帮 |

**通过线**：每项 ≥ 8，综合 ≥ 8.0

## R01 雨夜古寺师徒对决

**题材**：武侠 → 东方玄幻

### 测试文本
```
雨夜古寺，两名剑客在佛像前对峙，师父发现徒弟已经入魔，二人拔剑相向。
```

### 测试命令

```text
/storyboard 一键生成 雨夜古寺，两名剑客在佛像前对峙，师父发现徒弟已经入魔，二人拔剑相向。15s 7镜
```

```text
/character 墨渊 黑衣剑客，冷峻，长发，玄色长袍，佩青色古剑，曾是正道第一剑修
```

```text
/character 顾长空 中年剑宗宗师，灰白鬓发，蓑衣，青锋长剑，师父
```

```text
/scene 雨夜古寺大殿，破败佛像，烛火摇晃，屋顶漏雨，地面积水反光
```

```text
修改测试：
第3镜更暗一点，第5镜雨改雪，第7镜更史诗
```

```text
风格迁移测试：
换成王家卫风格但保持角色
```

### 通过标准
- [ ] 一键生成风格为东方玄幻（非融合）
- [ ] 角色锚点名含身份（师父/徒弟）
- [ ] 墨渊含身份解释（第一剑修+徒弟→天才型）
- [ ] 场景锚点出图模式无箭头/标注
- [ ] 第5镜雨改雪帧4末有过渡
- [ ] 王家卫迁移声明显性锁定，不出现「霓虹灯光」「街灯」「LED」
