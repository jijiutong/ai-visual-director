## OB5 AI 自动续写

**命令**：`下一段往师父黑化方向发展`（基于章3结尾，两人向魔域裂隙进发）

**通过标准**：
- [ ] 基于当前段叙事状态续写（墨渊+顾长空进入暗道→魔域裂隙）
- [ ] **硬锁定 5 项**：
  - [ ] 墨渊 DNA 不可变字段（脸型/眼型瞳色/眉型/鼻型/嘴型/肤色/声音/气味）
  - [ ] 顾长空 DNA 不可变字段
  - [ ] 世界观规则：武侠/中国古代/魔域/双生魔种 → 不新增超自然规则
  - [ ] 核心冲突：对抗苏长老/解除双生魔种 → 不变
  - [ ] 时间线方向：正向，即刻进入暗道 → 不闪回不乱跳
- [ ] **软锁定 6 项**：
  - [ ] 服装：暗道潮湿泥泞 → 衣袍沾泥可演进，标注原因
  - [ ] 情绪：当前震撼→愤怒，下一段可进展到决意，不能跳级到平静
  - [ ] 场景：古寺暗道→魔域裂隙，有过渡（暗道下行→空气硫磺味→裂隙光出现）
  - [ ] 道具：「破妄」剑 + 顾长空青锋长剑 → 延续，如损坏须说明
  - [ ] 光线：暗道微绿荧光→裂隙暗紫脉动光，渐进变化
- [ ] **叙事逻辑 5 项**：
  - [ ] 因果链：章3墨渊揭示真相 + 两人进暗道 → 续写接续此状态
  - [ ] 动机连贯：墨渊要砍苏长老/顾长空要护徒弟 → 动机不突变
  - [ ] 信息不跳级：墨渊知道苏长老在裂隙 → 续写不突然知道新情报
  - [ ] 称呼一致：墨渊/顾长空/苏长老 跨段不混用
- [ ] **用户方向**：「师父黑化」→ 顾长空可能有被魔种残留影响、或情绪失控的倾向，但不颠覆角色
- [ ] 自动匹配衔接技法 + 尾帧→首帧声明
