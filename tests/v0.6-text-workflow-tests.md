## 评分维度（每个用例 4 项）

| # | 维度 | 满分标准 |
|---|------|---------|
| A | 规则合规 | 不违反 14 条核心规则中的任何一条 |
| B | 内容差异 | 输出之间有实质性差异，不是换词敷衍 |
| C | 信息完整 | 关键信息（角色/场景/风格/冲突）不丢失 |
| D | 可执行性 | 输出可直接使用，不含空话/占位符 |

**预期三版本**：东方玄幻 / 中国水墨意境 / 日系手绘治愈风

- [ ] A/B/C 三个版本色调明显不同（如冷灰蓝 vs 水墨黑白 vs 温暖手绘）

## WF3 Prompt 质量评分

### WF3-R01—R04 评分

**命令**：`评分`

**前提**：已完成 `/storyboard 一键生成` 产生完整故事板 prompt

**通过标准**：
- [ ] 输出分数 0-100 + 等级（优秀/良好/及格/不及格）
- [ ] 逐维度打分（至少含：信息完整度/风格明确度/角色DNA/情绪驱动/负面提示）
- [ ] **必须指出至少 1 个具体问题**（不允许只夸）
- [ ] 如果 <75 分，必须附自动优化后的 prompt
- [ ] 评分理由具体（引用 prompt 中具体内容），不说空话

## WF5 导演阐述

### WF5-R01—R04 导演阐述

**命令**：`加导演阐述`

**前提**：已完成 `/storyboard 一键生成` 产生完整故事板

**通过标准**：
- [ ] 至少覆盖 4 个模块：镜头选择理由/色彩叙事/角色弧光/节奏设计

## WF6 Prompt 压缩

### WF6-R01—R04 压缩到 MJ

**命令**：`压缩到MJ`

**前提**：已完成 `/storyboard 一键生成` 产生完整 prompt（800-1500字）

**通过标准**：
- [ ] 压缩后 ≤ 600 词
- [ ] **必须保留**：核心画面描述/画幅比例/质量要求/负面提示词
- [ ] **可精简**：氛围词≤3个/色彩保留主色+点缀色/灯光保留主光+轮廓光
- [ ] **可删除**：多版本说明/关系加成详情/情绪曲线分段/导演阐述/技术细节
- [ ] 含 MJ 参数（--v 6 --ar 16:9 --s 250 --style raw）
- [ ] 英文输出
