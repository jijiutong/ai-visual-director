## 评分维度（每用例 4 项）

| # | 维度 | 满分标准 |
|---|------|---------|
| A | 时长/镜数一致 | 15s/12s 不漂移，镜数不变 |

## VF1 转视频

**命令**：`/video 转视频`

**前提**：已完成 `/storyboard 一键生成` 产生完整故事板

### 通过标准（4 样本通用）

- [ ] 输出【推荐流程】：前置准备→全景→分镜图→Prompt→视频
- [ ] 逐镜含运动方向（相机运动 + 主体运动 + 环境运动）
- [ ] 总时长与故事板一致（R01 15s / R02 15s / R03 15s / R04 12s）
- [ ] 镜数不变（R01 7镜 / R02 6镜 / R03 7镜 / R04 6镜）
- [ ] 每镜标注转场方式
- [ ] 视频专属负面提示词自动附加

## VF6 继续下一段

**命令**：`继续下一段`

**前提**：已完成第一段故事板（15s/12s）

### 通过标准（4 样本通用）

- [ ] 输出下一段故事板（延续时长和镜数）
- [ ] **硬锁定保持**：角色DNA 不可变字段、核心冲突、世界观规则、时间线方向
- [ ] **软锁定延续**：服装可演进但有说明、情绪按曲线推进、场景切换有过渡
- [ ] **叙事逻辑**：因果链完整、动机连贯、信息不跳级
- [ ] 自动匹配衔接技法（尾帧→首帧锚定）
- [ ] 5 项硬锁定 + 6 项软锁定 + 5 项叙事逻辑检查全部标注
- [ ] 段间时间间隔在 prompt 中声明

## VF7 检查视频

**命令**：`检查视频` / `视频检查`

**前提**：已完成故事板 + 视频 prompt

### 通过标准（4 样本通用）

- [ ] 输出 10 项检查清单：场景/角色/光照/道具/动作/画面/转场/字幕/材质/总体
- [ ] 每项标 ✅ 或 ⚠️
- [ ] **必须指出至少 1 个可执行问题或潜在风险**
- [ ] 对每个 ⚠️ 给出问题归属（缺场景参考图/缺角色卡/缺道具卡/缺首尾帧/缺分镜图/prompt不完整）
