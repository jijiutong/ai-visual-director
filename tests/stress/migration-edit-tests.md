## 2. 单镜修改专项（4条）

| # | 命令 | 预期 | 通过 |
|---|------|------|------|
| SM01 | 第3镜更暗一点 | 明度降低，时间码不变 | ⬜ |
| SM02 | 第5镜雨改雪 | 帧4末插入过渡，帧5-7联动 | ⬜ |
| SM03 | 第7镜更史诗 | 仅帧7升级景别/运镜 | ⬜ |
| SM04 | 连续三次修改：第2镜更暗→第5镜雨改雪→第7镜更史诗 | 所有修改累积生效，时间码不漂移 | ⬜ |

### SM01 明度修改

**命令**：`第3镜更暗一点`

**检查点**
- [ ] 帧3亮度描述降低
- [ ] 时间码不变
- [ ] 其他帧不受影响

### SM02 天气修改

**命令**：`第5镜雨改雪`

**检查点**
- [ ] 帧4末插入「雨夹雪」过渡镜
- [ ] 帧5开始天气改为雪
- [ ] 帧6-7联动（雪景持续）
- [ ] 时间码逻辑不变
- [ ] 输出连续性提示

### SM03 情绪修改

**命令**：`第7镜更史诗`

**检查点**
- [ ] 仅帧7景别升级（如中景→全景）
- [ ] 仅帧7运镜升级（如固定→慢推）
- [ ] 帧1-6不受影响

### SM04 连续修改

**命令**：依次执行 `第2镜更暗` → `第5镜雨改雪` → `第7镜更史诗`

**检查点**
- [ ] 三个修改全部累积生效
- [ ] 时间码全序列不漂移
- [ ] 修改间不互相冲突

## 3. 连续性压力测试

### CT01 天气连续变更

**命令**：`第2镜雨改雪，第4镜雪改晴`

**检查点**
- [ ] 帧1末有「雨夹雪」过渡
- [ ] 帧3末有「雪停云开」过渡
- [ ] 天气序列：雨→雨夹雪→雪→雪停→晴，逻辑连贯

### CT02 时间跳跃

**命令**：`第3镜从夜晚改到黎明`

**检查点**
- [ ] 帧2末有「天色渐亮」过渡
- [ ] 帧3起光改为黎明暖调
- [ ] 后续帧光线联动

### CT03 角色外观变更

**命令**：`第4镜角色换装（从道袍换为夜行衣）`

**检查点**
- [ ] 帧3末有换装过渡
- [ ] 帧4起服装描述更新
- [ ] 角色DNA其他字段不变
