# AI Visual Director · 3 分钟演示脚本

> 目标：展示「输入一句故事 → 完整视觉方案」的核心能力
> 流程：4 个场景 × 45 秒 = 3 分钟

---

## 场景 1：东方玄幻 · 雨夜古寺（0:00-0:45）

### 输入

```
雨夜古寺，两名剑客在佛像前对峙，师父发现徒弟已经入魔，二人拔剑相向。
```

### 展示

1. **智能推荐**（5s）— 系统识别为「武侠/悲壮」，推荐东方玄幻/黑金动作/水墨
2. **一键生成**（15s）— 雨夜佛前 · 7 镜分镜表 + 角色锚点 + 场景锚点
3. **角色卡**（10s）— `/character 墨渊` 身份解释：曾是第一剑修→天才型青年入魔者
4. **精准修改**（10s）— 第3镜更暗 / 第5镜雨改雪（自动补桥）/ 第7镜更史诗
5. **风格迁移**（5s）— 换成王家卫风格：锁定角色+剧情，仅改色彩/运镜

### 旁白要点
> 一句 30 字的故事，系统自动识别题材、构建角色 DNA、设计 7 镜分镜。修改一帧不影响其他帧，改天气自动补过渡。

---

## 场景 2：现代都市 · 深夜便利店（0:45-1:30）

### 输入

```
深夜便利店，女主下班后独自吃关东煮，前男友推门进来买烟。两人多年未见，只隔着货架沉默对视。
```

### 展示

1. **一键生成**（20s）— 深夜货架 · 6 镜分镜 + 角色：林晚（银手链）/ 陆时寒
2. **运镜可执行性**（10s）— 便利店<20平米：全部固定机位/过肩/横移/慢推拉。**无旋转无穿墙**
3. **氛围调整**（10s）— 「更暧昧」→ 暖暗调+负空间+拉焦
4. **出视频 Prompt**（5s）— 一条完整 Prompt 直喂 Seedance

### 旁白要点
> 狭小空间不会出现不合理的环绕运镜。调整一句「更暧昧」色彩/运镜/节奏自动跟上。

---

## 场景 3：科幻 · 废弃太空港（1:30-2:15）

### 输入

```
废弃太空港内，最后一名维修员发现休眠舱里的少女是仿生人。空间站将在十分钟后坠入木星大气层。
```

### 展示

1. **一键生成**（10s）— 木星十分钟 · 7 镜分镜。**关键：推荐暗黑科幻，非机甲**
2. **角色卡**（10s）— `/character 艾拉` 银白短发 + 数据流瞳孔 + 机械纹路皮肤
3. **风格迁移**（10s）— 换成 3D动画电影风但保持角色。**无 Disney 品牌名**
4. **题材检查**（10s）— 无现代霓虹灯 / 无东方玄幻串味 / 无机甲

### 旁白要点
> 太空题材不会错误推荐机甲。风格迁移时 DNA 不变——仿生人还是仿生人，木星还是木星。

---

## 场景 4：治愈 · 蓝色桌面宠物（2:15-2:50）

### 输入

```
一只蓝色小桌面宠物趁主人加班到深夜，偷偷推来迷你热牛奶，又笨拙地把键盘回车键擦干净。
```

### 展示

1. **一键生成**（10s）— 3D动画电影风（非 Disney Pixar）。12s 6 镜
2. **禁文字检查**（10s）— 帧6：「屏幕弹出一个绿色勾选图标，柔和提示光」← 不是「提交成功」
3. **微缩世界**（10s）— 回车键如广场/核桃壳如浴缸/便签纸是它的家门
4. **全流程**（5s）— 输入→推荐→一键生成→角色卡→场景图→视频 Prompt

### 旁白要点
> 全部 Prompt 无一处要求模型生成可读文字。风格名是「3D动画电影风」不是 Disney。零品牌侵权风险。

## 剪辑备注

- 每个场景 45 秒，含：输入文字(3s) → 系统输出关键帧截图(25s) → 讲解(15s) → 切场(2s)
- 旁白语速：中慢，每个场景最后一句放慢
- 背景音乐：环境电子/氛围钢琴
- 字幕：底部居中白色，关键词（一键生成/身份解释/自动补桥）用高亮色
