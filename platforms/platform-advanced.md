# AI 平台专属深度优化

不只是格式转换，而是针对每个平台的特性深度优化。

## Midjourney v6 深度优化

### 角色一致性 `--cref`

```
生成流程：
1. 先单独生成角色设定卡（输出角色图 URL）
2. 用该 URL 作为 --cref 参数生成故事板
3. 所有镜头自动保持角色一致
```

**Prompt 追加**：
```
在角色设定卡生成后：
"保存此角色图 URL，后续所有镜头使用 --cref [URL] 保持角色一致性"

故事板 prompt 结尾追加：
--cref [角色图URL] --v 6 --ar 16:9 --s 250 --style raw --no text, watermark
```

### 风格一致性 `--sref`

```
生成流程：
1. 先单独生成情绪板/参考图（输出风格图 URL）
2. 用该 URL 作为 --sref 参数生成故事板
```

**Prompt 追加**：
```
--sref [风格图URL] --sw 100
```

### MJ 排版优化

MJ 不擅长多面板排版，需要调整策略：

- **全案板** → 建议拆分成多个单图生成，后期拼接
- **角色卡** → 单个角色全身像，三视图分三次生成
- **场景卡** → 单张场景概念图，MJ 最强项
- **海报** → MJ 最擅长，直接生成
- **漫画分镜** → 建议拆成单格生成

**MJ 格式转换规则**：
```
全案板 → 拆分为：1张核心场景 + 2张角色卡 + 1张情绪板
快速故事板 → 拆分为 7 张单镜头图，每张独立 prompt
角色设定卡 → 正面/侧面/背面 3 次独立生成
漫画分镜 → 每格独立生成，后期拼接
```

## Stable Diffusion XL 深度优化

### ControlNet 姿势控制

```
使用 ControlNet OpenPose 控制角色姿势：
1. 用 OpenPose 生成角色骨架图
2. 作为 ControlNet 输入控制姿势
3. 确保不同镜头中角色姿势准确
```

**推荐 ControlNet 模型**：
- OpenPose：控制角色姿势
- Depth：控制场景深度和透视
- Canny：控制线条和构图
- Tile：增强细节

**SD 参数优化**：
```
Sampler: DPM++ 2M Karras
Steps: 30-50
CFG Scale: 7-9
Resolution: 1024x576 (16:9)
Model: Juggernaut XL / RealVisXL / Animagine XL（动漫）
VAE: sdxl_vae.safetensors
```

### IP-Adapter 角色一致性

```
使用 IP-Adapter 保持角色一致：
1. 生成角色参考图
2. 用 IP-Adapter 加载参考图
3. 所有后续生成引用该角色
```

### SD 排版优化

SD 多面板能力弱，建议策略：

- 单镜头独立生成，后期用 PS/Canva 拼接
- 使用 SD 的 MultiDiffusion 插件做多镜头拼接
- 角色设定卡用 ControlNet OpenPose 分别生成三个视图

## GPT Image 2 深度优化

### 排版能力最大化

GPT Image 2 最强项是复杂排版，充分发挥：

- **全案板** → 直接生成完整多模块布局
- **角色卡** → 单次生成完整三视图+面部+道具
- **漫画分镜** → 直接生成多格漫画布局
- **情绪板** → 直接生成色卡+材质+灯光参考

### 文字理解优势

- 直接中文输入，不需要翻译
- 支持复杂指令和自然语言描述
- 可以直接理解"冷暖分割光""对角线对峙构图"等专业术语

### GPT Image 2 最佳实践

```
1. 用自然语言描述，不要堆关键词
2. 明确说明排版结构（顶部/中部/底部/左侧/右侧）
3. 指定每个模块的内容和位置
4. 质量要求放在最后
5. 负面提示词用英文补充
```

## 平台选择建议

| 需求 | 推荐平台 | 原因 |
|------|---------|------|
| 完整故事板/全案板 | GPT Image 2 | 最强排版能力 |
| 角色一致性 | Midjourney (--cref) | 角色参考最稳定 |
| 海报 | Midjourney | 海报效果最好 |
| 场景概念图 | 三者皆可 | MJ 最梦幻，GPT 最准确，SD 最可控 |
| 角色设定卡 | SD + ControlNet | 姿势控制最精准 |
| 漫画分镜 | GPT Image 2 | 多格排版能力 |
| 情绪板 | GPT Image 2 | 多模块排版 |
| 批量生成 | SD | 本地运行速度快 |

## 一键跨平台输出

用户说"全平台"时，为同一故事生成三个平台的优化版本：

```
【GPT Image 2 优化版】
[中文完整描述，最大化排版能力]

【Midjourney v6 优化版】
[英文描述 + --cref + --sref + --v 6 + --ar + --s + --style raw + --no]

【Stable Diffusion XL 优化版】
正面：[英文描述]
负面：[专用负面词]
参数：Sampler DPM++ 2M Karras, Steps 30-50, CFG 7-9, 1024x576
ControlNet：OpenPose（角色姿势）+ Depth（场景深度）
```
