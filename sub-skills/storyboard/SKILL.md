# Storyboard — 快速故事板

粘贴任意故事片段，AI 自动提取信息 → 智能推荐最佳风格+格式 → 输出可直接复制使用的 image prompt。

## 触发方式

- `/storyboard [你的故事]`
- 直接说 "故事板"、"分镜"、"帮我做这个故事板"
- 粘贴故事内容即可自动触发

## 工作流

### Step 1: 提取故事信息

从用户输入中自动提取：片名、类型、主角、对手、核心冲突、世界观场景、情绪倾向、Mood 倾向。

**缺字段自动补全**：没角色名 → 起电影感名字；没明确类型 → 按冲突强度推断。

### Step 2: 智能推荐 2-3 个组合

```
【智能推荐】根据你的故事，推荐以下组合：

  推荐 1（最匹配）：风格 X + 格式 Y    — 理由
  推荐 2（备选）：  风格 A + 格式 B    — 理由
  推荐 3（不同视角）：风格 C + 格式 D  — 理由

回复编号确认，或说"看全部"查看所有风格/格式。
```

**推荐引擎规则**（故事关键词 → 风格/格式）：

| 故事关键词 | 推荐风格 | 推荐格式 |
|-----------|---------|---------|
| 修仙/古风/玄幻 | 3. 东方玄幻 | 1 全案板 / 3 角色卡 |
| 武侠/复仇/谍战/格斗 | 1. 黑金动作 | 1 全案板 / 10 关键帧 |
| 机甲/赛博/未来/AI | 2. 科幻机甲 | 1 全案板 / 5 海报 |
| 悬疑/密室/犯罪/恐怖 | 4. 悬疑惊悚 | 2 快速故事板 / 7 情绪板 |
| 都市/爱情/职场/暧昧 | 5. 都市情绪 | 4 场景概念卡 / 7 情绪板 |
| 校园/青春/初恋/成长 | 6. 青春校园 | 8 四格漫画 / 3 角色卡 |

完整 53 种风格 → 说 "看全部" 展示。

### Step 3: 用户确认 → 生成 Prompt


### 快捷指令

| 指令 | 效果 |
|------|------|
| `一键生成` | 全流程自动，不展示选项 |
| `看全部` | 展示 53 种风格 + 10 种格式 |
| `多版本` | A/B/C 三版本对比 |
| `一键全平台` | GPT Image / Midjourney / SD 三平台输出 |

## 输出格式

### 全案板（格式 1）模板

```
【项目总览】[片名] · [类型] · [时长] · [叙事结构]
【核心冲突】[一句话：谁对抗谁，主角要突破什么]
【情绪曲线】[曲线类型]：[阶段1]→[阶段2]→[阶段3]→[阶段4]
【主色调】[2-4个颜色]

【分镜表】
帧1 | [时间码] | [景别] | [画面描述] | [运镜] | [色彩/灯光] | [转场]
帧2 | ...
帧N | ...

【技术参数】
画幅：[16:9/9:16/2.39:1] | 风格：[VS编号+名称] | 平台：[GPT Image/MJ/SD]
摄影机：[型号] / [焦段] / [景深]
负面词：[自动生成]
```

### 快速故事板（格式 2）模板

```
[故事标题] · [风格编号+名称]

帧1：[时间码] | [景别] | [画面内容 — 1句] | [运镜] | [转场]
帧2：...
...
帧N：...

画幅：[比例] | 平台：[目标平台]
```

## 情绪/Mood 调整

用户说以下关键词时自动调整：
- `更燃` → 暖色+快切+低角度+疾推
- `更虐` → 冷色+慢镜+高角度俯拍+拉远
- `更甜` → 柔光+粉色调+环绕镜头+浅景深
- `更丧` → 去饱和+固定机位+空旷构图+空镜
- `更暧昧` → 暖暗调+负空间+拉焦+过肩镜头
- `更恐怖` → 暗调+手持+窥视视角+极速推进
- `更史诗` → 广角+航拍+慢推+金色逆光

## 负面清单

所有 prompt 自动附加：
```
no watermark, no logo, no random large text, no garbled Chinese,
no broken faces, no duplicated limbs, no messy panels,
no low-quality collage, no text overlay, no cartoon style (unless specified),
no flat illustration, no marketing poster style
```
