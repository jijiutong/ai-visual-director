# 文化精准度系统

确保不同文化背景的描述精准无误，不出跨文化穿帮元素。

## 文化检查规则

**核心原则**：东方玄幻不出西方魔幻元素，宫廷权谋服装符合历史朝代，不同文化背景精准描述。

## 东方文化精准描述

### 中国古风/修仙

| 正确 | 错误 |
|------|------|
| 交领/对襟/裙/曲裾 | 和服/韩服（除非明确日本/韩国背景） |
| 发髻/发簪/步摇/发冠 | 西式皇冠/头纱 |
| 御剑/法术/灵气/阵法 | magic spell/wizard staff/wand |
| 仙山/云海/竹林/古刹 | castle/cathedral/medieval tower |
| 玉佩/香囊/折扇/古琴 | sword and shield/knight armor |
| 水墨/留白/宣纸/墨色 | oil painting/canvas/renaissance |

**专用词汇替换**：
- 不说 "magic" → 说 "法术/灵力/真气"
- 不说 "wizard" → 说 "修士/仙人/道士"
- 不说 "castle" → 说 "仙山/宫殿/宗门"
- 不说 "knight" → 说 "剑修/侠士/修士"

## 文化混合场景

当故事涉及跨文化元素时：

1. **明确主次**：哪种文化为主（占 70%），哪种为辅（占 30%）
2. **不穿帮**：主文化的核心元素不被副文化替代
3. **融合点**：在装饰/道具/细节层面融合，不在核心架构层面混合

例："赛博朋克 + 东方玄幻" → 主架构赛博（城市/科技），装饰层东方（符文/阵法/仙衣纹样）

## 在 Prompt 中使用

在 prompt 中追加文化精准度约束：

```
文化精准度：[文化类型]。[正确元素列表]。禁止：[错误元素列表]。
```

例：
```
文化精准度：中国古风修仙。使用交领/对襟/襦裙/发髻/发簪/御剑/法术/灵气/仙山/竹林/玉佩。禁止：和服/西式皇冠/magic spell/wizard staff/castle/knight armor/油画面。
```
