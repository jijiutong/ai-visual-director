# Prompt 质量评分器

生成前自动评估 prompt 质量（0-100分），给出具体改进建议，低于阈值自动优化。

## 评分维度

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 信息完整度 | 20% | 片名/类型/角色/场景/情绪是否齐全 |
| 风格明确度 | 15% | 风格编号/配色/氛围词/特效是否具体 |
| 镜头精确度 | 15% | 景别/运镜/焦段/画幅是否标注 |
| 角色 DNA | 10% | 角色 DNA 是否完整锚定 |
| 情绪驱动 | 10% | 情绪曲线是否标注阶段和变化 |
| 色彩叙事 | 10% | 色彩是否随情绪演进 |
| 文化精准 | 10% | 文化元素是否精准无穿帮 |
| 负面提示 | 5% | 负面词是否覆盖风格/格式/平台 |
| 结构完整 | 5% | 模板结构是否完整（基础/风格/关系/情绪/DNA/文化/平台） |

## 评分规则

**诚实评分**：每个维度低于满分时，必须给出扣分原因和具体改进建议。不允许无理由扣分，也不允许无理由满分。每份评分报告至少指出 1 个具体改进点（可以是在得分最高但仍可优化的维度）。

## 评分等级

| 分数 | 等级 | 处理 |
|------|------|------|
| 90-100 | 优秀 | 直接输出 |
| 75-89 | 良好 | 标注改进建议后输出 |
| 60-74 | 及格 | 自动优化后输出 |
| <60 | 不及格 | 必须优化，不输出 |

## 自动优化规则

当评分 <75 时，自动执行以下优化：

| 扣分项 | 自动优化动作 |
|--------|------------|
| 信息完整度低 | 补全缺失字段（片名/角色/场景） |
| 风格明确度低 | 追加风格配色/氛围词/特效 |
| 镜头精确度低 | 追加景别/运镜/焦段参数 |
| 角色 DNA 缺失 | 生成角色 DNA 并锚定 |
| 情绪驱动缺失 | 追加情绪曲线阶段标注 |
| 色彩叙事缺失 | 追加色彩演进描述 |
| 文化精准度低 | 追加文化精准度约束 |
| 负面提示缺失 | 追加专属负面提示词 |
| 结构不完整 | 补齐缺失层级 |

## 评分输出格式

```
【Prompt 质量评分】XX/100（等级：[优秀/良好/及格]）

各维度得分：
- 信息完整度：X/20
- 风格明确度：X/15
- ...

[如低于75分] 已自动优化：[列出优化项]
```

## 用户指令

| 用户指令 | 行为 |
|---------|------|
| "评分" | 仅输出评分，不生成 prompt |
| "优化后生成" | 评分 + 自动优化 + 生成 |
| 默认 | 自动评分，<60 自动优化后输出 |
