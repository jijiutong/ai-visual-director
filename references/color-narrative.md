# 色彩叙事引擎

颜色随情绪推进而变化。不只是静态配色，而是动态色彩叙事。

## 色彩叙事模型

### 标准四阶段色彩演进

```
阶段1 平静：冷灰蓝 #2a2a3a + 中性白 #f0f0f0          → 孤独/日常/铺垫
阶段2 对峙：冷灰蓝 #2a2a3a + 暖橙 #ff6633（开始混合）  → 冲突萌芽/张力
阶段3 爆发：血红 #cc0033 + 金色 #ffd700（高对比）     → 高潮/释放/决断
阶段4 余韵：暖米 #f5e6d3 + 柔和金 #daa520              → 释怀/温暖/新开始
```

### 各类型色彩叙事

#### 动作型色彩叙事

```
暗部(黑金) → 金属火花(金) → 爆炸火光(红+橙) → 硝烟余灰(灰)
```

#### 爱情型色彩叙事

```
冷色调(孤独蓝灰) → 冷暖交汇(蓝+粉) → 暖色调(粉红+金) → 柔和统一(暖米)
```

#### 悬疑型色彩叙事

```
暗蓝(不安) → 暗红渐变(危险接近) → 血红爆发(真相揭示) → 苍白(虚无/反转)
```

#### 悲剧型色彩叙事

```
暖金(美好) → 冷灰入侵(裂痕) → 深蓝黑(崩溃) → 灰白(余痛/空白)
```

#### 成长型色彩叙事

```
暗淡(起点) → 微光出现(希望) → 多彩爆发(突破) → 金色成熟(成长完成)
```

## 色彩对比策略

| 对比类型 | 用法 | 示例 |
|---------|------|------|
| 冷暖对比 | 内心冲突/两人对峙 | 一人站冷光中，一人站暖光中 |
| 明暗对比 | 善恶/希望绝望 | 大面积暗部 + 单一亮光 |
| 饱和度对比 | 现实 vs 回忆 | 现实低饱和 + 回忆高饱和 |
| 色相对比 | 阵营对立 | 蓝方 vs 红方 |
| 纯度渐变 | 情绪推进 | 从低纯度灰 → 高纯度纯色 |

## 色彩心理映射

| 颜色 | 情绪 | 适用阶段 |
|------|------|---------|
| 深蓝/冷灰 | 孤独、冷静、不安 | 平静/对峙 |
| 暖橙/金黄 | 希望、温暖、力量 | 爆发/余韵 |
| 血红 | 愤怒、激情、危险 | 爆发 |
| 紫色 | 神秘、权力、魔法 | 对峙/爆发 |
| 纯白 | 纯真、空虚、新生 | 余韵 |
| 纯黑 | 恐惧、未知、绝望 | 对峙/爆发 |
| 粉色 | 爱情、温柔、甜美 | 平静/余韵 |
| 绿色 | 成长、自然、治愈 | 余韵/平静 |

## 在 Prompt 中使用

```
色彩叙事：按情绪曲线四阶段演进——
阶段1[平静]：[色1]+[色2]，[情绪描述]
阶段2[对峙]：[色1]+[色3]（冷暖开始混合），[情绪描述]
阶段3[爆发]：[色4]+[色5]（高对比爆发），[情绪描述]
阶段4[余韵]：[色6]+[色7]（统一色调），[情绪描述]
```
