# 风格融合引擎

用户说"风格A + 风格B"时，融合两种风格的视觉基因，产出独一无二的混合风格。

## 融合规则

融合不是简单叠加，而是：
- **主风格**（占比 60%）：决定整体结构、版式、大色调
- **副风格**（占比 40%）：决定细节质感、特效元素、氛围微调

## 融合 Prompt 模板

```
整体风格：[主风格编号. 主风格] × [副风格编号. 副风格]（主风格 60% + 副风格 40%）
```

## 自定义融合

用户输入任意两种风格组合时，AI 自动执行：

1. 确定主副关系（先说的为主，或按故事类型判断）
2. 从 `references/styles.md` 提取两种风格的：配色/氛围词/环境特效/灯光方案
3. 主风格决定：整体结构、版式布局、大色调、核心构图
4. 副风格决定：细节质感、特效元素、氛围微调、材质参考
5. 生成融合后的完整 prompt

例："赛博朋克 + 东方玄幻" → 赛博为主（60%）决定结构和科技感，东方玄幻为辅（40%）注入灵气粒子/飞剑/仙衣元素
