# 镜头语言参考（80+ 技术）

## 一、景别（Shot Size）

| 编号 | 景别 | 英文 | 说明 | 用途 |
|------|------|------|------|------|
| CM1 | 大远景 | Extreme Long Shot (ELS) | 环境为主，人物极小 | 建立世界观、展现规模 |
| CM2 | 远景 | Long Shot (LS) | 人物全身 + 大部分环境 | 场景交代、人物位置 |
| CM3 | 全景 | Full Shot (FS) | 人物从头到脚 | 动作展示、服装全貌 |
| CM4 | 中景 | Medium Shot (MS) | 腰部以上 | 对话、日常互动 |
| CM5 | 中近景 | Medium Close-Up (MCU) | 胸部以上 | 情绪交流、表情变化 |
| CM6 | 近景 | Close-Up (CU) | 肩部以上/面部 | 情绪高潮、关键反应 |
| CM7 | 特写 | Extreme Close-Up (ECU) | 眼睛/嘴/道具局部 | 细节强调、紧张感 |
| CM8 | 大特写 | Macro Shot | 睫毛/汗水/纹理 | 极度紧张、微观信息 |
| CM9 | 过肩 | Over the Shoulder (OTS) | 越过一人肩膀拍另一人 | 对话、对峙、关系 |
| CM10 | 双人中景 | Two-Shot | 两人同框腰部以上 | 关系展示、互动 |
| CM11 | 三人中景 | Three-Shot | 三人同框 | 群像关系 |
| CM12 | 半身像 | Half Shot | 大腿中部以上 | 肢体语言 + 表情 |
| CM13 | 牛仔镜头 | Cowboy Shot | 膝盖以上 | 西部片经典、武器展示 |
| CM14 | 意大利式特写 | Italian Close-Up | 眼睛极度放大 | 赛尔乔·莱翁内式张力 |
| CM15 | 插入镜头 | Insert Shot | 物体/道具特写 | 线索展示、细节强调 |

## 在 Prompt 中使用

```
镜头语言：[摄影机型号] / [焦段] / [画幅] cinematic crop / [景别] / [运镜] / [角度] / [景深描述] / film grain / high contrast lighting / rim light / backlight / volumetric light / motion blur / dynamic composition / professional cinematography
```

```

```
