# 身体语言与姿态库

## 五、在 Prompt 中使用

```
角色姿态：[站姿编号. 站姿名称]。手势[手势编号. 手势名称]。[面部表情/微表情]。[与其他角色的相对位置/距离]。
```

例：
```
角色姿态：BL3. 防御交叉站姿。手势BL14. 握拳（指节发白）。面部EX23. 隐忍（眉头压低+咬牙+嘴唇紧抿）。与对手距离3步，对峙状态。
```
