# 角色一致性 DNA 系统

解决 AI 生成最大痛点：角色跨镜头崩坏。建立角色 DNA 描述系统，确保每个镜头中服装/发型/面部完全一致。

## DNA 结构

每个角色生成一条 DNA 字符串，在所有镜头中重复引用：

```
[角色名]DNA: 性别,年龄段,体型,面部特征,发型,瞳色,肤色,服装(上/下/外),配饰,武器/道具,标志性特征
```

## DNA 字段详解

| 字段 | 描述粒度 | 示例 |
|------|---------|------|
| 性别 | 男/女/非二元 | 男 |
| 年龄段 | 少年(14-17)/青年(18-28)/壮年(29-45)/中年(46-60)/老年(60+) | 青年 |
| 体型 | 瘦削/匀称/健壮/微胖/纤细 | 瘦削挺拔 |
| 面部特征 | 脸型+眉形+鼻型+唇形+特殊标记 | 瓜子脸,剑眉,高鼻梁,薄唇,左眉骨小疤 |
| 发型 | 长度+颜色+样式+特殊处理 | 黑色长发及腰,半束高马尾,几缕碎发垂额前 |
| 瞳色 | 颜色+形状特征 | 深棕色,凤眼,眼尾微挑 |
| 肤色 | 色号描述 | 冷白皮,细腻 |
| 服装-上 | 款式+颜色+材质+细节 | 白色交领宽袖长衫,银色云纹刺绣 |
| 服装-下 | 款式+颜色+材质 | 深蓝色束脚长裤 |
| 服装-外 | 外披/斗篷/铠甲 | 玄色披风,银边,内衬暗红 |
| 配饰 | 发饰/首饰/腰带/鞋 | 银色发冠,腰间玉佩,黑色长靴 |
| 武器/道具 | 名称+外观 | 青云剑,剑身泛青蓝色光芒,剑柄缠白绸 |
| 标志性特征 | 独一无二的识别标志 | 腰间玉佩发出微光时代表情绪波动 |

## DNA 示例

```
碧瑶DNA: 女,青年,纤细,瓜子脸柳叶眉杏眼薄唇额间一点朱砂痣,绿色长发及腰两侧鬓角垂落,翠绿色眼眸猫眼,冷白皮,绿色短裙配白色交领上衣裙摆绣花,银色腰链配绿色流苏,合欢铃(金色铃铛系红绳),额间朱砂痣发光时代表使用合欢铃力量
```

```
张小凡DNA: 男,青年,瘦削挺拔,方脸浓眉大眼厚唇神情木讷,黑色短发略显凌乱,深棕色眼眸,健康小麦色,灰色粗布短打上衣裤腿扎紧,腰间草绳系普通木剑,普通铁剑剑柄缠旧布条,紧张时会不自觉地握紧剑柄
```

## 一致性检查清单

在每个镜头生成前检查：

- [ ] 发型完全一致（长度/颜色/样式）
- [ ] 面部特征一致（脸型/眉形/特殊标记）
- [ ] 服装颜色不变（同一套服装在不同镜头中颜色统一）
- [ ] 配饰位置一致（发饰/首饰/腰带）
- [ ] 武器/道具外观一致
- [ ] 标志性特征在所有镜头中出现

## 多角色交互时的 DNA 引用

当多个角色同框时，在 prompt 中明确引用双方 DNA：

```
画面中同时出现 [角色A名] 和 [角色B名]：
[角色A名]：[完整DNA]
[角色B名]：[完整DNA]
两人相对位置：[距离/角度/姿态]
```

## 在 Prompt 中使用

在角色设定栏后追加 DNA 锚定：

```
角色 DNA 锚定（所有镜头必须保持一致）：
[角色名]DNA: [完整DNA字符串]
```

在分镜描述中简化引用：

```
镜3：[角色名]（参考DNA）出现在[场景]，[动作描述]
```

## 情绪变化时的 DNA 扩展

情绪变化不改变 DNA 基础信息，只追加临时状态：

```
[角色名] 当前状态：[基础DNA] + 表情[愤怒/悲伤/喜悦] + 服装状态[凌乱/破损/湿润] + 特殊效果[发光/流汗/流血]
```
