# 叙事结构分析器（12 种模型）

用户丢一段小说/剧本，AI 自动拆解成叙事结构，每幕/阶段自动匹配对应的镜头组合、色彩、节奏。

---

## 一、12 种叙事结构模型

### NS1. 三幕标准（好莱坞经典）

```
第一幕：建制（Setup）      25%  —— 世界观/角色/日常/激励事件
第二幕：对抗（Confrontation） 50%  —— 冲突升级/障碍/中点转折/最低点
第三幕：解决（Resolution）   25%  —— 最终对决/高潮/结局/余韵
```

**适用**：动作/爱情/科幻/喜剧/大多数商业类型
**镜头分配**（7镜）：镜1-2 建制 → 镜3-5 对抗 → 镜6-7 解决

### NS2. 四幕结构（起承转合·韩剧/东方）

```
起（Introduction）   20%  —— 引入世界与角色
承（Development）    30%  —— 关系发展/日常铺垫
转（Turning）        30%  —— 冲突爆发/关系转变
合（Conclusion）     20%  —— 解决/余韵
```

**适用**：爱情/成长/都市/古装/长篇
**特点**：第二幕（承）比三幕更长，给足关系发展时间

### NS3. 倒叙结构（悬疑/犯罪）

```
开头：高潮/结局片段（5%）—— 吸引注意，展示结果
回到起点 → 建制（20%）→ 调查/对抗（50%）→ 回到开头补完（25%）
```

**适用**：悬疑/犯罪/惊悚/推理
**关键**：开头必须展示足够震撼的画面（尸体/爆炸/对峙），但保留核心悬念

### NS4. 双线叙事（身份对立/替身/回忆）

```
A线[主线/现在时] ←→ B线[副线/过去时] 交替进行
A线：建制(15%) → 对抗(35%) → 解决(20%)
B线：建制(10%) → 发展(15%) → 揭示(5%)
最终汇合：A线结局 = B线揭示的真相
```

**适用**：身份对立/替身/破镜重圆/悬疑
**关键**：B线信息必须逐步释放，不能一次性给完

### NS5. 环形叙事（首尾相接）

```
结尾 = 开头（同一场景/同一画面）
但角色状态/观众认知已完全改变
```

**适用**：哲学/文艺/悲剧/宿命主题
**关键**：表面相同，内涵完全不同

### NS6. 多视角叙事（罗生门式）

```
同一事件 × N个角色视角 = N个不同版本
视角1：角色A的版本 → 视角2：角色B的版本 → 真相揭示
```

**适用**：悬疑/人性/群像/复杂关系
**关键**：每个视角必须有独特的情感色彩和视觉风格

### NS7. 碎片化叙事（记忆拼图）

```
非线性的碎片场景 → 逐渐拼出完整画面
碎片1 → 碎片3 → 碎片2 → 碎片4 → …… → 完整拼图
```

**适用**：失忆/创伤/梦境/心理
**关键**：每个碎片必须独立成立，组合后产生新意义

### NS8. 倒计时叙事（限时压迫）

```
T-30min → T-20min → T-10min → T-5min → T-0
每个时间节点是一个叙事单元
```

**适用**：拆弹/救援/逃生/限时任务
**关键**：每个时间节点必须有进度变化和紧迫感升级

### NS9. 英雄之旅（12阶段）

```
1.平凡世界 → 2.冒险召唤 → 3.拒绝召唤 → 4.遇见导师
5.跨越门槛 → 6.试炼/盟友/敌人 → 7.接近深渊
8.严峻考验 → 9.获得奖赏 → 10.归途
11.复活 → 12.满载而归
```

**适用**：史诗/冒险/成长/奇幻/好莱坞大片
**关键**：每个阶段有明确的视觉特征（如"遇见导师"用仰拍+柔光）

### NS10. 救猫咪（15节拍）

```
1.开场画面 2.主题陈述 3.建置 4.催化剂 5.争论
6.第二幕衔接点 7.B故事 8.游戏时间 9.中点
10.坏人逼近 11.失去一切 12.第三幕衔接点
13.结局 14.终场画面 15.尾声
```

**适用**：商业片/喜剧/爱情/动作
**关键**：15个节拍精确对应情感节奏

### NS11. 情绪弧线叙事（非事件驱动）

```
平静 → 微澜 → 波动 → 高潮 → 余韵 → 新的平静
没有外部事件驱动，纯情绪推进
```

**适用**：文艺片/情绪板/音乐MV/散文诗
**关键**：每个情绪阶段有明确的色彩/镜头/节奏参数

### NS12. 双线汇合（过去与现在）

```
过去线：起因/背景/根源（逐步揭示）
现在线：结果/现状/行动（逐步推进）
最终汇合点：过去线的揭示 = 现在线的行动理由
```

**适用**：复仇/救赎/破镜重圆/悬疑
**关键**：两条线最终必须在同一个画面/场景中汇合

## 二、结构 → 镜头参数映射

### 第一幕/起/建制（通用参数）

| 参数 | 值 |
|------|-----|
| 景别 | ELS/LS → MS |
| 运镜 | Static / Slow Push / Wide Pan |
| 色彩 | 低对比 / 自然色 / 根据基调定冷暖 |
| 灯光 | 自然光 / 柔和散射 |
| 节奏 | 慢 / 稳定 / 每镜 3-5 秒 |
| 构图 | 留白 / 三分法 / 环境为主 |

### 第二幕/承/对抗（通用参数）

| 参数 | 值 |
|------|-----|
| 景别 | MS/MCU → CU/ECU（逐渐逼近） |
| 运镜 | Tracking / Orbit / Push In / Handheld |
| 色彩 | 冷暖对比 / 饱和度递增 / 冲突色出现 |
| 灯光 | 分割光 / 侧光 / 局部强光 / 阴影加重 |
| 节奏 | 中→快 / 逐渐加速 |
| 构图 | 对角线 / 过肩 / 对称对峙 / 填充画面 |

### 第三幕/转/高潮（通用参数）

| 参数 | 值 |
|------|-----|
| 景别 | CU/ECU（高潮）→ LS/ELS（拉远） |
| 运镜 | Crash Zoom / Crane Up / Pull Out |
| 色彩 | 极端对比（高潮）→ 统一色调（余韵） |
| 灯光 | 爆炸光/火焰光（高潮）→ 柔和光（余韵） |
| 节奏 | 极快→慢 / 0.5秒→5秒 |
| 构图 | 填充画面（高潮）→ 留白（余韵） |

### 第四幕/合/余韵（通用参数）

| 参数 | 值 |
|------|-----|
| 景别 | LS/ELS（拉远）/ Close-Up（情绪收尾） |
| 运镜 | Slow Pull Out / Static / Crane Up |
| 色彩 | 统一色调 / 低对比 / 柔和 |
| 灯光 | 柔和散射 / 暖光 / 余晖 |
| 节奏 | 极慢 / 5-8秒 / 长镜头 |
| 构图 | 大面积留白 / 背影 / 空镜 |

## 三、类型 → 结构自动匹配

| 故事类型 | 推荐结构 | 说明 |
|---------|---------|------|
| 动作/武侠 | NS1 三幕标准 | 建制→对抗→决战 |
| 爱情/甜宠 | NS2 四幕起承转合 | 给足关系发展时间 |
| 悬疑/犯罪 | NS3 倒叙 / NS6 多视角 | 开头吸引+逐步揭示 |
| 科幻 | NS1 三幕标准 / NS9 英雄之旅 | 发现异常→危机→拯救 |
| 成长 | NS2 四幕 / NS9 英雄之旅 | 尝试→挫折→成长 |
| 喜剧 | NS1 三幕 / NS10 救猫咪 | 日常→意外→和解 |
| 悲剧 | NS1 三幕 / NS5 环形 | 美好→裂痕→崩溃 |
| 史诗/奇幻 | NS9 英雄之旅 + NS4 双线 | 宏大叙事+副线 |
| 文艺/情绪 | NS11 情绪弧线 | 纯情绪推进 |
| 限时/救援 | NS8 倒计时 | 时间节点驱动 |
| 失忆/创伤 | NS7 碎片化 | 记忆拼图 |
| 复仇/救赎 | NS12 双线汇合 | 过去揭示=现在行动 |
| BL/耽美 | NS2 四幕 / NS5 环形 | 关系发展+余韵 |
| 群像 | NS6 多视角 | 多角色叙事 |

## 四、在 Prompt 中使用

```
叙事结构：[编号. 结构名称]。
[各幕/阶段描述，含镜号分配、景别、运镜、色彩、节奏]。
```

例：
```
叙事结构：NS2. 四幕起承转合。
起(镜1-2, 20%)：ELS建立云海神宫世界观，Slow Push引入男主。色彩低对比冷灰。节奏慢 4s/镜。
承(镜3-5, 30%)：MS→MCU关系发展，Tracking/Orbit。色彩冷暖交汇。节奏标准 3s/镜。
转(镜6-8, 30%)：CU/ECU冲突爆发，Handheld/Push In加速。色彩高对比极端。节奏快 1s/镜。
合(镜9-10, 20%)：LS拉远余韵，Slow Pull Out。色彩统一暖金。节奏极慢 5s/镜。
```
