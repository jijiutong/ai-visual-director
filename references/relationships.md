## 关系详细预设

### R1. 宿敌变恋人

- **构图建议**：对角线对峙构图 → 逐渐靠近的正面构图 → 最终并肩
- **镜头语言**：low angle（双方各自仰拍显强大）→ eye level（平视对话）→ close-up（面部微表情）
- **灯光方案**：冷暖分割光（两人各站冷暖光中）→ 混合光（中间地带）→ 统一暖光
- **氛围关键词**：剑拔弩张、势均力敌、眼神交锋、刀尖舔血、相爱相杀、宿命纠缠
- **风格加成**：
  - 黑金动作：金属兵器碰撞火花 + 暴雨中对峙 + 血迹与雨水混合
  - 东方玄幻：剑气对峙 + 灵力粒子碰撞 + 仙衣猎猎 + 飞剑光轨
  - 好莱坞大片：爆炸背景中的慢动作对视 + 废墟烟尘 + 英雄式仰拍
  - 中世纪史诗：城堡城墙上对峙 + 战旗猎猎 + 铠甲反光 + 箭雨背景
- **Prompt 追加**：
  ```
  角色关系：宿敌变恋人。构图从对角线对峙逐渐过渡到并肩站立，两人各占画面一侧形成对称张力。灯光采用冷暖分割方案——主角站在冷光中，对手站在暖光中，中间留暗部过渡。镜头从 low angle 各自仰拍（凸显双方强大）推进到 eye level 平视（平等对话），最终 close-up 面部微表情（心动瞬间）。氛围：剑拔弩张、势均力敌、眼神交锋、相爱相杀。
  ```

### R2. 双向暗恋

- **构图建议**：三分法构图，两人分别占据两个交点，视线交汇处为视觉中心
- **镜头语言**：over the shoulder（过肩对话）→ medium close-up（各自独白）→ tight close-up（眼神交汇）
- **灯光方案**：柔和散射光 + 暖色补光，营造暧昧温暖氛围
- **氛围关键词**：欲言又止、眼神闪躲、心跳加速、指尖触碰、假装不经意
- **风格加成**：
  - 都市情绪：咖啡厅窗边/图书馆书架间/雨后街道，霓虹虚化散景
  - 青春校园：教室午后阳光/操场看台/樱花树下，充足自然光
  - Wes Anderson：对称构图 + 粉彩色调 + 复古道具 + 强迫症式排版
  - 写实摄影：纪实手持感 + 自然光 + 粗粝质感
- **Prompt 追加**：
  ```
  角色关系：双向暗恋。构图采用三分法，两人分别占据三分线交点，视线交汇处为视觉中心。过肩镜头展现对话中的试探与闪躲，medium close-up 捕捉各自独处时的思念表情，tight close-up 定格眼神交汇的心动瞬间。柔和散射光 + 暖色补光营造暧昧氛围。氛围：欲言又止、眼神闪躲、心跳加速、假装不经意。
  ```

### R3. 师徒

- **构图建议**：俯视/仰视角度对比（师傅俯视教导，徒弟仰视敬畏）→ 最终平视
- **镜头语言**：high angle（师傅视角）→ low angle（徒弟视角）→ eye level（关系转变）
- **灯光方案**：顶光（师傅神圣感）+ 侧光（徒弟仰望）
- **氛围关键词**：禁忌、仰望、越界、守护、克制、传承
- **风格加成**：
  - 东方玄幻：云端论道 + 灵气缭绕 + 仙山背景 + 法器光芒
  - 吉卜力：手绘水彩 + 自然场景 + 温暖日常 + 微风飘动
  - 水墨意境：留白构图 + 山水间 + 墨色渐变 + 诗意氛围
  - 童话绘本：魔法学院 + 发光书本 + 温暖奇幻
- **Prompt 追加**：
  ```
  角色关系：师徒。构图从俯视/仰视角度对比开始——师傅 high angle 俯视教导显威严，徒弟 low angle 仰视显敬畏，最终过渡到 eye level 平视（关系转变/越界）。灯光：师傅头顶光营造神圣感，徒弟侧光凸显仰望姿态。氛围：禁忌、仰望、克制、守护、越界的紧张感。
  ```

### R4. 年上/年下

- **构图建议**：体型差/身高差构图，年上者占据画面上方/后方（保护位），年下者占据下方/前方（依赖位）
- **镜头语言**：wide shot 展示身高差 → medium 展示互动 → close-up 展示表情反差
- **灯光方案**：年上者暖光（成熟稳重），年下者冷光或自然光（青春活力）
- **氛围关键词**：成熟引导、青春冲击、体型差、身高差、保护欲、依赖感
- **风格加成**：
  - 都市情绪：办公室/天台/雨夜街头，暖光 vs 冷光的年龄对比
  - 青春校园：走廊尽头/放学后的教室，自然光 + 逆光发丝光
  - 王家卫：霓虹香港 + 慢门拖影 + 浓烈色彩 + 城市孤独
  - 复古胶片：老上海/年代感场景，胶片颗粒 + 褪色质感
- **Prompt 追加**：
  ```
  角色关系：年上/年下。构图强调体型差和身高差——年上者占据画面上方/后方保护位，年下者占据下方/前方依赖位。wide shot 展示身高体型对比，medium 展示互动中的引导与追随，close-up 展示成熟稳重 vs 青春活力的表情反差。灯光：年上者暖光（成熟），年下者自然光（青春）。氛围：成熟引导、青春冲击、保护欲、依赖感。
  ```

### R5. 先婚后爱

- **构图建议**：画面中间留白/隔断（初期距离感）→ 隔断消失/距离缩短 → 亲密接触
- **镜头语言**：wide shot（两人分居画面两端）→ medium（逐渐靠近）→ close-up（亲密）
- **灯光方案**：初期冷色调分隔 → 中期暖光混合 → 后期统一暖光
- **氛围关键词**：冷淡、契约、习惯、依赖、深爱、从陌生到亲密
- **风格加成**：
  - 宫廷权谋：华服洞房 + 红烛摇曳 + 盖头掀起 + 金饰反光
  - 都市情绪：现代公寓 + 同居日常 + 窗外城市夜景
  - 复古胶片：年代婚礼 + 胶片颗粒 + 老式灯光
  - 东方玄幻：仙界联姻 + 红绸飞天 + 灵兽见证
- **Prompt 追加**：
  ```
  角色关系：先婚后爱。构图展现距离变化——初期画面中间留白/隔断（冷淡距离感），中期隔断消失/距离缩短（习惯与依赖），后期亲密接触（深爱）。镜头从 wide shot 两人分居画面两端推进到 close-up 亲密互动。灯光从冷色调分隔过渡到统一暖光。氛围：冷淡→习惯→依赖→深爱的情感递进。
  ```

### R6. 死对头/冤家

- **构图建议**：对称对峙构图，表情夸张/喜剧化
- **镜头语言**：medium shot 互怼 → close-up 表情夸张 → medium 意外靠近
- **灯光方案**：明亮均匀光，偏喜剧调性
- **氛围关键词**：互怼、嘴硬心软、吃醋、口是心非、欢喜冤家
- **风格加成**：
  - 青春校园：教室/走廊/食堂，明亮自然光 + 樱花/银杏
  - 都市情绪：职场/咖啡厅，暖光 + 都市背景虚化
  - Disney 2D 手绘：水彩背景 + 优雅线条 + 歌舞剧气质
  - 童话绘本：魔法森林/动物伙伴，柔和梦幻光
- **Prompt 追加**：
  ```
  角色关系：死对头/冤家。对称对峙构图，表情夸张喜剧化。medium shot 展现互怼日常，close-up 捕捉口是心非的微表情，medium 意外靠近时的慌乱。明亮均匀光，偏喜剧调性。氛围：互怼、嘴硬心软、吃醋、口是心非、欢喜冤家。

### R7. 救赎/治愈

- **构图建议**：暗→明过渡构图，一人从暗部走向另一人的光中
- **镜头语言**：dark wide shot（孤独）→ push in（发现光）→ close-up（被治愈的表情）
- **灯光方案**：大面积暗部 + 单一暖光源（救赎者如光）
- **氛围关键词**：黑暗、光、依赖、共生、破碎、修复
- **风格加成**：
  - 都市情绪：雨夜街头/天台边缘/天台花园，城市霓虹 + 路灯暖光
  - 悬疑惊悚：压抑空间中的微弱希望，单一光源 + 长阴影
  - 废土末日：废墟中的篝火/绿洲，昏黄天光 + 火光
  - 吉卜力：手绘水彩 + 自然光 + 治愈系温暖氛围
- **Prompt 追加**：
  ```
  角色关系：救赎/治愈。构图从暗到明过渡——被救赎者从大面积暗部走向救赎者的暖光中。dark wide shot 展现孤独与破碎，push in 镜头捕捉发现光的瞬间，close-up 定格被治愈的表情变化。灯光：大面积暗部 + 单一暖光源（救赎者如光）。氛围：黑暗中的光、依赖、共生、破碎与修复。

### R9. 强制/囚禁

  角色关系：重逢/破镜重圆。构图从画面两侧逐渐向中心靠近。medium shot 远处认出的瞬间，push in 镜头跟随走近的脚步，close-up 定格重逢时复杂表情（回忆/回避/释怀），over shoulder 展现对话中的试探与和解。灯光：回忆暖光滤镜与现实冷光交织，最终过渡到统一暖光（复合）。氛围：物是人非、回忆涌现、释怀、复合。

### R12. 身份对立

- **构图建议**：画面中间分界线（阵营分割），两人各站一侧
- **镜头语言**：wide 展现阵营对立 → medium 私下相遇 → close-up 挣扎表情
- **灯光方案**：两侧不同色温（阵营标志色）+ 中间暗部（选择的灰色地带）
- **氛围关键词**：阵营冲突、私情、抉择、背叛、忠义两难
- **风格加成**：
  - 东方玄幻：仙界 vs 魔界 + 灵气粒子 vs 暗黑能量 + 云端对峙
  - 科幻机甲：人类联盟 vs AI帝国 + 冷蓝全息 vs 暖红警报
  - 好莱坞大片：超级英雄 vs 反派阵营 + 城市天际线背景
  - 中世纪史诗：王国 vs 叛军 + 城堡对峙 + 战旗分界
- **Prompt 追加**：
  ```
  角色关系：身份对立。画面中间分界线构图——两人各站阵营一侧，形成鲜明对比。wide shot 展现阵营对立，medium 捕捉私下相遇的禁忌感，close-up 定格忠义两难的挣扎表情。灯光：两侧不同色温（阵营标志色），中间暗部暗示选择的灰色地带。氛围：阵营冲突、私情、抉择、背叛、忠义两难。
  ```

### R13. 养成/守护

- **构图建议**：大/小对比构图，守护者居高临下或被守护者仰望
- **镜头语言**：wide（守护者视角）→ medium（照顾日常）→ close-up（依恋表情）→ eye level（最终平等）
- **灯光方案**：守护者暖光（可靠）+ 被守护者柔光（需要保护）
- **氛围关键词**：照顾、依恋、成长、超越、从保护到平等
- **风格加成**：
  - 吉卜力：手绘水彩 + 自然场景 + 温暖日常 + 飞行/魔法
  - 东方玄幻：仙山修行 + 灵气缭绕 + 法器传承
  - 童话绘本：魔法森林 + 发光蘑菇 + 温暖梦幻
  - 都市情绪：都市中的守护，暖光 + 城市背景
- **Prompt 追加**：
  ```
  角色关系：养成/守护。大/小对比构图——守护者居高临下或被守护者仰望。wide shot 展现守护者视角，medium 捕捉照顾日常，close-up 定格依恋表情，最终 eye level 展现从保护到平等的关系转变。灯光：守护者暖光（可靠温暖）+ 被守护者柔光（需要保护）。氛围：照顾、依恋、成长、从保护到平等。
  ```

### R14. 暧昧拉扯

- **构图建议**：安全距离内的微妙靠近——手指将触未触、呼吸交错但不接触
- **镜头语言**：medium close-up（对话中的试探）→ close-up（手指/嘴唇/眼神特写）→ pull out（后退一步）
- **灯光方案**：暖色暧昧光 + 暗部留白（未说出口的部分）
- **氛围关键词**：试探、靠近、后退、再靠近、心跳、克制
- **风格加成**：
  - 都市情绪：办公室/电梯/车内/雨夜，暧昧暖光 + 城市虚化
  - 王家卫：霓虹 + 慢门拖影 + 浓烈色彩 + 城市孤独
  - 复古胶片：胶片颗粒 + 老式灯光 + 怀旧暧昧
  - Wes Anderson：对称构图 + 粉彩色调 + 复古道具
- **Prompt 追加**：
  ```
  角色关系：暧昧拉扯。构图强调安全距离内的微妙靠近——手指将触未触、呼吸交错但不接触。medium close-up 捕捉对话中的试探与闪躲，close-up 特写手指/嘴唇/眼神的微小动作，pull out 镜头展现后退一步的克制。灯光：暖色暧昧光 + 暗部留白（未说出口的部分）。氛围：试探、靠近、后退、再靠近、心跳、克制。
  ```

### R15. 一见钟情

- **构图建议**：人群中的聚焦——周围虚化/暗化，两人清晰明亮
- **镜头语言**：wide crowd shot → crash zoom（目光锁定）→ slow motion → close-up（命中注定的表情）
- **灯光方案**：聚光效果（两人在光中，周围暗）
- **氛围关键词**：命中注定、时间静止、心动、追寻
- **风格加成**：
  - 张艺谋色彩：极致色彩 + 红色中国 + 宏大构图 + 仪式感
  - 童话绘本：发光蝴蝶/星光粒子 + 梦幻温暖
  - Disney Pixar 3D：圆润角色 + 温暖光 + 梦幻色彩
  - 按故事类型匹配主风格 + 聚光效果加成
- **Prompt 追加**：
  ```
  角色关系：一见钟情。构图：人群中的聚焦——周围虚化/暗化，两人清晰明亮如在光中。wide crowd shot 展现茫茫人海，crash zoom 模拟目光锁定的瞬间，slow motion 放慢心动一刻，close-up 定格命中注定的表情。灯光：聚光效果（两人在光中，周围暗）。氛围：命中注定、时间静止、心动。
  ```

## 在 Prompt 中使用

在生成的 prompt 中，在"核心场景概念图"段落后追加关系加成：

```
角色关系加成：[关系编号. 关系名称]。[构图建议]。[灯光方案]。[氛围关键词]。
```

例：
```
角色关系加成：R2. 双向暗恋。构图采用三分法，两人分别占据三分线交点，视线交汇处为视觉中心。柔和散射光 + 暖色补光营造暧昧氛围。氛围：欲言又止、眼神闪躲、心跳加速、假装不经意。
```
