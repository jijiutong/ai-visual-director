# Mood 滑块控制

用户说"更虐/更甜/更燃/更丧/更虐心/更暧昧"，自动调整情绪曲线权重、色彩叙事方向、镜头语言强度。

## Mood 维度

| Mood | 情绪曲线调整 | 色彩调整 | 镜头调整 | 节奏调整 |
|------|------------|---------|---------|---------|
| 更燃 | 爆发阶段权重×2 | 血红+金色高对比 | Crash Zoom/Handheld/Dutch | 快切 0.5 秒 |
| 更虐 | 对峙+爆发加重 | 冷蓝→暗红→灰白 | 特写面部痛苦/眼泪 | 慢→极慢（放大痛苦） |
| 更甜 | 余韵阶段权重×2 | 暖粉+金色+柔光 | Close-Up 微笑/触碰 | 慢节奏 3-5 秒 |
| 更丧 | 余韵阶段拉长 | 冷灰+低饱和+暗部加重 | Wide 孤独/Empty frame | 极慢 5-8 秒 |
| 更暧昧 | 对峙阶段加重 | 暖色暧昧光+暗部留白 | 手指/嘴唇/眼神特写 | 慢，含 pause 留白 |
| 更恐怖 | 渐进节奏极端化 | 暗蓝→血红→苍白 | Dutch Angle/Tight CU | 安静→极快→骤停 |
| 更史诗 | 全阶段放大 | 金色+深色高对比 | ELS 宏大/Crane Up | 标准→高潮加速 |
| 更治愈 | 余韵阶段权重×2 | 暖光+柔焦+低对比 | 自然光/Wide 宁静 | 极慢节奏 |
| 更紧张 | 对峙→爆发缩短 | 冷暖剧烈冲突 | Handheld/Push In 加快 | 加速剪辑 |
| 更梦幻 | 全阶段柔化 | 粉紫+柔光+散景 | Slow Push/Soft Focus | 慢节奏+叠化 |

## Mood 叠加规则

用户可以说多个 mood，按权重叠加：

- "更燃更虐" → 爆发加重 + 痛苦特写
- "更甜更暧昧" → 余韵加重 + 手指/眼神特写
- "更丧更恐怖" → 余韵拉长 + 暗部极端化

## Mood 对 Prompt 的影响

### 更燃

```
追加：情绪曲线调整——爆发阶段权重加倍。色彩：血红 #cc0033 + 金色 #ffd700 高对比。镜头：Crash Zoom / Handheld / Dutch Angle 为主。节奏：快切 0.5-1 秒。音效：爆炸/打击/能量爆发。构图：填充画面/动态对角线。
```

### 更虐

```
追加：情绪曲线调整——对峙和爆发阶段加重，痛苦放大。色彩：冷蓝 #2a2a3a → 暗红 #4a0e0e → 灰白 #888888。镜头：Extreme Close-Up 面部痛苦/眼泪/颤抖的手。节奏：慢到极慢（每镜 3-8 秒），放大痛苦瞬间。构图：特写填充/暗部加重。
```

### 更甜

```
追加：情绪曲线调整——余韵阶段权重加倍。色彩：暖粉 #ffb6c1 + 金色 #daa520 + 柔光效果。镜头：Close-Up 微笑/指尖触碰/眼神温柔交流。节奏：慢节奏 3-5 秒，含 pause 留白。构图：三分法/中心对称/柔和。
```

### 更丧

```
追加：情绪曲线调整——余韵阶段大幅拉长。色彩：冷灰 #4a4a4a + 低饱和 + 暗部加重 #0d0d0d。镜头：Wide/ELS 孤独/Empty frame/背影。节奏：极慢 5-8 秒，长镜头。构图：大面积留白/暗部/孤独中心。
```

### 更暧昧

```
追加：情绪曲线调整——对峙阶段加重，心跳加速段拉长。色彩：暖色暧昧光 #ff6633 + 暗部留白。镜头：Close-Up 手指将触未触/嘴唇/眼神闪躲/呼吸交错。节奏：慢，含 pause 留白（未说出口的部分）。构图：安全距离内的微妙靠近。
```

## 在 Prompt 中使用

在 prompt 中追加 mood 调整：

```
Mood 调整：[Mood 关键词]。[对应的色彩/镜头/节奏/构图调整描述]。
```
