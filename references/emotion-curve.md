# 情绪曲线驱动分镜引擎

根据"平静→对峙→爆发→余韵"情绪曲线，动态调整每个镜头的景别/运镜/灯光/色彩。

## 四阶段标准模型

```
平静(25%) → 对峙(25%) → 爆发(25%) → 余韵(25%)
    │            │           │           │
   铺垫         张力升级     高潮释放     收尾回响
```

## 各阶段镜头参数

### 阶段 1：平静（铺垫）

| 参数 | 值 |
|------|-----|
| 景别 | Wide/Long Shot（大远景/远景） |
| 运镜 | Static（固定）/ Slow Push In（慢推） |
| 灯光 | 柔和散射光 / 自然光 |
| 色彩 | 低对比 / 冷色调或中性色 |
| 节奏 | 慢（每镜 3-5 秒感） |
| 构图 | 留白 / 三分法 / 环境为主 |
| 氛围 | 宁静 / 日常 / 暴风雨前的平静 |

### 阶段 2：对峙（张力升级）

| 参数 | 值 |
|------|-----|
| 景别 | Medium / Medium Close-Up（中景/中近景） |
| 运镜 | Tracking（跟踪）/ Pan（摇摄）/ Orbit（环绕） |
| 灯光 | 冷暖分割光 / 侧光 / 局部强光 |
| 色彩 | 冷暖对比开始 / 饱和度上升 |
| 节奏 | 中速（每镜 2-3 秒感） |
| 构图 | 对角线 / 过肩 / 对称对峙 |
| 氛围 | 紧张 / 压迫 / 一触即发 |

### 阶段 3：爆发（高潮释放）

| 参数 | 值 |
|------|-----|
| 景别 | Close-Up / Extreme Close-Up（近景/特写） |
| 运镜 | Crash Zoom（疾推）/ Handheld（手持）/ Dutch Angle（倾斜） |
| 灯光 | 强对比 / 爆炸光 / 火焰光 / 极端明暗 |
| 色彩 | 高饱和 / 极端对比色 / 红色/金色爆发 |
| 节奏 | 快切（每镜 0.5-1 秒感） |
| 构图 | 填充画面 / L型 / 动态对角线 |
| 氛围 | 激烈 / 释放 / 决定性瞬间 |

### 阶段 4：余韵（收尾回响）

| 参数 | 值 |
|------|-----|
| 景别 | Wide / Long Shot（拉远回到远景） |
| 运镜 | Pull Out（拉远）/ Crane Up（升降）/ Static（固定） |
| 灯光 | 柔和散射 / 月光 / 暖光回归 |
| 色彩 | 低对比 / 色调统一 / 暖色或冷色回归 |
| 节奏 | 慢（每镜 3-5 秒感） |
| 构图 | 留白 / 中心对称 / 孤独感 |
| 氛围 | 平静 / 释怀 / 孤独 / 新的开始 |

## 变体模型

### 悬疑型情绪曲线

```
平静(20%) → 不安(20%) → 紧张(20%) → 爆发(20%) → 反转(20%)
```
- 增加"不安"和"反转"阶段
- 反转阶段用 Rack Focus（变焦）+ Dutch Angle（倾斜）+ 冷光突转

### 喜剧型情绪曲线

```
日常(25%) → 意外(25%) → 混乱(25%) → 解决(25%)
```
- 混乱阶段用快速切换 / 表情夸张 / 明亮均匀光
- 解决阶段回归日常光，warm reunion tone

### 悲剧型情绪曲线

```
美好(25%) → 裂痕(25%) → 崩溃(25%) → 余痛(25%)
```
- 美好阶段用温暖光 / 柔焦 / 暖色调
- 崩溃阶段用极端冷光 / 大面积黑色 / 面部特写
- 余痛阶段用空镜头 / 拉远 / 冷色调

### 爱情型情绪曲线

```
相遇(20%) → 试探(20%) → 心动(20%) → 确认(20%) → 余韵(20%)
```
- 相遇用聚光效果（两人在光中，周围暗）
- 试探用三分法 + 暖色暧昧光
- 心动用 Tight Close-Up + 慢动作
- 确认用 Eye Level + 统一暖光
